Отправка объявления, сохраненного как черновик

Внимание: Для того чтобы совершать все эти действия, вам должна быть назначена роль с соответствующими полномочиями. Для того чтобы определить, какая у вас роль, перейдите на страницу Роли участников. Основные сведения о ролях и правах доступа изложены также в Права доступа и роли: обзор.

Для того чтобы отправить объявление, которые вы прежде сохранили как черновик:

  1. На линейке инструментов вашего курса или проекта нажмите Объявления.

  2. Перед заглавием объявления, сохраненного как черновик, стоит префик "черновки". Для того чтобы открыть черник объявления, нажмите на его заглавие.

  3. Внесите необходимые изменения. Для того чтобы выбрать настройки отправки извещений по электронной почте, выберите один из пунктов в выпадающем меню "Извещение по электронной почте" Всем участникам, Только подписавшимся участникам или Без извещения.

  4. Для того чтобы просмотреть объявление перед отправкой, нажмите на кнопку Просмотр. Для того чтобы сохранить изменения и не отправлять объявление, нажмите на кнопку Сохранить изменения. Для того чтобы сохранить изменения и не просматривать объявление, нажмите на кнопку Добавить объявление. Для того чтобы отменить внесенные изменения, нажмите на кнопку Отмена.
