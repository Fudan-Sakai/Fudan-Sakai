Создание/добавление, редактирование и удаление объявления

Разделы:

  • Создание/добавление объявлений
  • Редактирование объявлений
  • Удаление объявлений

Внимание: Для того чтобы совершать все эти действия, вам должна быть назначена роль с соответствующими полномочиями. Для того чтобы определить, какая у вас роль, перейдите на страницу Роли участников. Основные сведения о ролях и правах доступа изложены также в Права доступа и роли: обзор.

Создание/добавление объявлений

Для того чтобы создать/добавить объявление:

  1. На линейке инструментов вашего сайта, нажмите на Объявления.

  2. На странице инструмента сверху нажмите на ссылкуДобавить.

  3. В тектовом поле "Заглавие объявления" введите тему.

  4. После заголовка "Текст объявления" введите и отформатируйте текст.

  5. Под заголовком "Показывать объявления", выберите параметр из списка:

    • Всем: объявление доступно участникам других сайтов
    • Только участникам сайта: объявление доступно только участникам сайта
    • Выбранным блокам участников: объявление доступно определенным группам и блокам участников. При помощи флажков отметьте, каким группам и блокам будет доступно объявление.

    Внимание: Если вы не видите пунктаВыбранным блокам участников, то на вашем сайте нет ни одной группы или блока участников; об этом см. Управление блоками участников.

  6. Под заголовком "Доступ" выберите либо Показывать, чтобы отправить ваше объявление, либо Скрыть, чтобы скрыть его от обычных пользователей, либо Указать даты, чтобы определить дату начала и дату окончания, когда объявление будет доступно.

    Внимание: Если вы выбрали пункт "Установить даты", извещение по электронной почте будет отправлено тогда, когда объявление станет доступным.

  7. Под заголовком "Приложения" вы можете приложить файл с вашего компьтера или из "Ресурсов", либо указать ссылку в Интернете. Подробнее об этом смотри Настройки добавления приложений.

  8. Вы можете также разослать ваше объявление участникам сайта по электронной почте. Для этого в пункте Извещение по электронной почте выберите из выпадающего списка:

    • Без извещения (по умолчанию): извещение не будет разослано по электронной почте.
    • Только подписавшимся участникам: извещение будет разослано только тем участникам, которые выбрали в "Моих настройках" получение низкоприоритетных извещений.
    • Всем участникам: извещение получат все участники сайта.

    Внимание: Участники могут выбрать, хотят ли они получать все объявления или только высокоприоритетные объявления.

  9. Для того чтобы отправить ваше объявление, нажмите на кнопку Добавить объявление. Для того чтобы просмотреть ваше объявление, нажмите на кнопку Просмотр. Для того чтобы не отправлять объявление, нажмите на кнопку Отмена.

Редактирование объявлений

Для того чтобы отредактировать объявление:

  1. На линейке инструментов вашего курса выберите Объявления.

  2. Под заглавием объявлением нажмите на ссылку Редактировать.

    Внимание: Если вы не видите ссылки "Редактировать", значит, у вас недостаточно прав для этого действия. Кроме того, можно редактировать сразу только одно объявление.

  3. Внесите изменения в текст и/или настройки объявления.

  4. По завершении редактирования нажмите на кнопкуСохранить изменения, чтобы обновить объявление, Просмотр, чтобы просмотреть измененное объявление, или Отмена, чтобы отменить все изменения.

Удаление объявлений

Для того чтобы удалить одно или несколько объявлений:

  1. На линейке инструментов вашего курса выберите Объявления.

  2. В столбце "Удалить?" отметьте те объявления, которые вы хотите удалить.

    Внимание: Если вы не видите флажков для выбора, значит, у вас недостаточно прав для этого действия.

  3. Нажмите на кнопку Обновить.

  4. На странице подтверждения удаления нажмите на кнопку Удалить, чтобы удалить объявления, или Отмена, чтобы отменить удаление.
