新增、編輯或刪除作業

在本頁中，您將瞭解：

  • 新建／新增作業
  • 預覽、儲存為草稿或發佈
  • 編輯作業
  • 刪除作業

註釋：如果要完成本頁中的操作，您必需擁有具有必要權限的角色。如需決定自己的角色，請遵循參與者角色中的說明進行操作。如需瞭解關於角色和權限的基礎知識，請參考權限和角色：概述。

新建／新增作業

如需新建/新增作業：

  1. 在功能表欄中，按一下作業，然後按一下頁面頂部的新增按鈕。

  2. 在新增作業頁面中，填寫下列資訊（帶有紅星標誌的為必填項）：

    • 標題：輸入作業的簡短標題（例如 短文一 ）。
    • 開放日期：開放日期和時間是指學生可以開始做作業的日期和時間。如需向學生傳送通告以通知開放日期，請參考下方的步驟 4。
    • 截止日期：指作業的最後提交期限。如需在課程網站的日程表中列出截止日期，請參考下方的步驟 4。
    • 最後接受提交日期：這一日期後不得繼續提交作業。您可以將這一日期和時間設定為晚於截止日期，以便在截止日期後繼續接受提交的作業（例如，允許修改）。
    • 學生提交：您可以選擇要求學生必需以行內的形式（直接在文字方塊中輸入內容），或必需以附件的形式提交作業，或選擇兩者皆可。還可允許學生以非電子檔的形式提交作業。
    • 評分標準：您可以從五種評分標準中進行選擇：無評分、字母評分、分數值、及格/不及格或打勾。如果選擇分數值，那麼輸入最高分數值（必需為正數或是零）。

  3. 在「作業說明」方塊中，輸入作業描述。

    註釋：本步驟為可選步驟，但是如果您將該欄位留空，那麼發佈作業時頁面頂部將出現警示資訊。再次按一下發佈繼續操作。

  4. 在作業說明文字方塊下方，您將看到下列選項：

    • 將截止日期新增到日程表：如需將截止日期新增到網站的日程表中，選中該核取方塊。
    • 將關於開放日期的通告新增到網站通告中：如需在發佈作業時自動宣佈開放日期，並且在網站首頁上發佈通告，選中該核取方塊。
    • 新增榮譽誓言：如需在學生提交作業時顯示榮譽誓言選項，選中該核取方塊。學生必需選中核取方塊同意誓言後才能提交作業。榮譽誓言內容如下：

      「我在完成這一作業過程中，沒有向他人提供幫助，也沒有獲得他人的幫助。是（您必需回答這一問題才能提交作業。）」

  5. 在「評分」欄位下，您將看到下列選項：

    • 不要新增作業至成績冊
    • 新增作業至成績冊。
    • 與現有成績冊項目相關聯

      註釋：使用作業工具新增的作業只能使用作業工具進行評分、編輯或刪除，即使已經將其新增到成績冊或是與成績冊條目相關聯也是如此。

  6. 在 "Access" （存取）欄位下，使用選項按鈕選擇 Display to site（向網站顯示）或 Display to selected groups（向所選群組顯示）。如果選擇後者，那麼您將看到可選群組或班別；使用核取方塊選擇您發佈作業的群組和/或班別。

  7. 在「提交通知電子郵件選項」欄位下，使用選項按鈕選擇下列選項之一：

    • 不要為學生提交的任何作業傳送通知電子郵件
    • 為每份學生提交的作業向我傳送通知電子郵件
    • 每天向我傳送一封有關學生提交作業的摘要通知

      註釋：此處的通知設定適用於包括講師、助教或兼職講師/講師助手在內的全部使用者。

  8. 在「附件」欄位下，您可將本機電腦或資源中的檔案作為附件，或指定一個網頁檔案的 URL 地址。如需瞭解更多說明，請參考新增附件選項。

預覽、儲存為草稿或發佈

如果您想要在發佈前預覽作業，按一下 Preview（預覽）。如需以學生將要看到的樣式檢視作業，按一下 Student View（學生檢視）旁的黑色三角。

預覽作業時，如果想要在發佈前儲存作業，那麼可按一下儲存草稿。如果需要，可在發佈前返回作業對其進行修改；請參考發佈先前儲存為草稿的作業。只有講師（或擁有適當權限的使用者）才能瀏覽草稿。

如果作業已準備就緒，學生可對其進行存取，按一下發佈。

編輯作業

如需在某項已發佈作業的開放日期或截止日期前後對其進行修改：

  1. 在功能表欄中，按一下作業按鈕。

  2. 在您想要修改的作業下，按一下編輯。

  3. 對作業進行修改。

  4. 修改完成後，可發佈或預覽作業。如果選擇預覽作業，那麼可選擇發佈或將其儲存為草稿。

    註釋：如果作業當前已向學生開放，那麼您只能預覽和發佈作業。

當學生檢視修改後的作業時，將通知他們作業已經被更改了。

刪除作業

註釋： 您可以完全刪除還沒有學生提交過的作業。然而，如果您刪除已有學生提交過的作業，所有學生將繼續看到作業帶有通知，說： 「﹙作業已被刪除﹚」 ，即使您以講師身份亦將不可看見作業在作業清單中。

如需刪除作業：

  1. 在功能表欄中，按一下作業按鈕。您將看到作業清單。

  2. 選中您想要刪除的作業旁的刪除？核取方塊。

  3. 按一下更新。

  4. 您將看到確認螢幕，要求您確認是否要刪除作業。按一下刪除永久刪除作業，如果您改變主意不想刪除作業，那麼按一下取消。
