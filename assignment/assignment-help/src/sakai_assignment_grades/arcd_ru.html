Копирование оценок за задания в электронную таблицу

Внимание: Для того чтобы совершать все эти действия, вам должна быть назначена роль с соответствующими полномочиями. Для того чтобы определить, какая у вас роль, перейдите на страницу Роли участников. Основные сведения о ролях и правах доступа изложены также в Права доступа и роли: обзор.

Чтобы загрузить оценкии из инструмента "Задания" в электронную таблицу:

  1. На линейке инструментов сайта курса или проекта нажмите Задание.

  2. Вверху страницы нажмите Сводка оценок.

    Внимание: Если оценки за задание еще не выставлены, они не будут показаваны в Сводке оценок, и их нельзя будет загрузить в таблицу. Чтобы выставить оценки за нужное задание, нажмите Оценить под нужным заданием. Вверзу страницы, нажмите Выставить оценки. Нажмите Сводка оценок, чтобы вернуться на сраницу Сводки оценок.

  3. Нажмите Скопировать в электронную таблицу.

  4. В зависимости от используемого браузера, вам будет предложено либо открыть, либо сохранить файл с таблицей. Следуйте инструкциям своего браузера; если выбрано сохранение файла, вы сможете открыть его на жестком диске компьютера.

    Внимание: Загруженный файл находится в формате Microsoft Excel (то есть .xls)

Электронная таблица содержит в первых двух столбцах ID студентов и их полные имена. Каждый следующий столбец содержит отдельное задние, а в соответствующих ячейках значатся оценки студентов. Если оценка не выставлена, то показана надпись "Не оценено".
