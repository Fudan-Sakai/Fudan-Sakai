Пересдача заданий

Чтобы пересдать задание:

  1. На линейке инструментов сайта курса или проекта нажмите Задания.

  2. Нажмите на название задания, которое вы хотите пересдать.

    Внимание: Возвращенные задания помечены как "Проверено" в колонке "Статус" списка заданий.

  3. Вы увидите оригинал задания и выполненную работу. Преподаватель мог добавить к тексту работы комментарий, который будет показан красным цветом. Любые дополнительные комментарии преподавателя появятся внизу страницы. В текстовое поле "Нажмите для повторной сдачи" введите новый вариант ответа на задание для пересдачи.

  4. Когда пересдача завершена, нажмите Сдать. В противном случае, нажмите Сохранить как черновик, чтобы сохранить задание и продолжить работу позже.
