重新提交作業

如需重新提交作業，請遵循下列步驟進行操作：

  1. 在相關課程或專案網站的功能表欄中，按一下作業。

  2. 按一下想要重新提交的作業的標題。

    註釋：在作業表的「狀態」欄中，返回的作業將顯示「已返回」。

  3. 您可以看到原始的作業和您最初提交的作業。您的講師可能會在您最初提交的作業中新增一些註解，這些註解將以紅色顯示。講師新增的任何註解都將顯示在螢幕下方。在「在下方輸入重新提交的內容」對話方塊中編輯或貼上作業的修改內容。

  4. 作業修改完成後，按一下 Resubmit（重新提交）。或者也可按一下 Save draft（儲存草稿）儲存您將要提交的內容，以便稍後繼續。
