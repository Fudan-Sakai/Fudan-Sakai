Возврат работы студенту

Внимание: Для того чтобы совершать все эти действия, вам должна быть назначена роль с соответствующими полномочиями. Для того чтобы определить, какая у вас роль, перейдите на страницу Роли участников. Основные сведения о ролях и правах доступа изложены также в Права доступа и роли: обзор.

Нажмите Задания на линейке инструментов, появится список заданий. Колонка "В/Новое" показывает общее количество сданных студентами работ ("В") и количество новых работ ("Новое") по каждому заданию. Ориентируясь по значениям в колонке, вы можете выбрать, с каким заданием необходимо работать.

Подробные инструкции по оценке заданий, включая инструкции по возврату работ, смотри в: Оценка задания и просмотр статуса выполнения работы студентом. Обратите внимание, что после того как вы начали оценку задания студента, вы можете вернуть его студенту и без выставленной оценки. Можно добавлять комментарии к работе студента как в тексте работы, так и с помощью текстового поля Комментарии преподавателя. Чтобы разрешить студенту пересдачу задания, отметьте флажком Разрешиить повторную сдачу. Когда задание готово к возврату, нажмите Отправить студенту внизу страницы. В списке сданных работ в колонке "Статус" будет виден статус "Возвращено студенту".
