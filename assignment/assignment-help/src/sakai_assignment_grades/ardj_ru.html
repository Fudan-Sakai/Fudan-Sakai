Дублирование задания

Внимание: Для того чтобы совершать все эти действия, вам должна быть назначена роль с соответствующими полномочиями. Для того чтобы определить, какая у вас роль, перейдите на страницу Роли участников. Основные сведения о ролях и правах доступа изложены также в Права доступа и роли: обзор.

Чтобы продублировать задание:

  1. На линейке инструментов, нажмите Задания. Появится список заданий.

  2. Чтобы продублировать задание, нажмите Дублировать под названием задания. В списке появится продублированное задание с пометкой "Черновик - название задания - Копия", где вместо "название задания" будет стоять оригинальное название. Статус задания будет переключен на "Черновик".

    Внимание: Дубликат задания сохраняется как черновик (то есть участникам сайта оно недоступно). Если необходимо сделать задание доступным для участников, нажмите на его название, а на появившейся странице нажмите кнопку Отправить.
