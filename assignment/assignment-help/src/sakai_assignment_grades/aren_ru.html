Чтение или загрузка студенческих работ

Внимание: Для того чтобы совершать все эти действия, вам должна быть назначена роль с соответствующими полномочиями. Для того чтобы определить, какая у вас роль, перейдите на страницу Роли участников. Основные сведения о ролях и правах доступа изложены также в Права доступа и роли: обзор.

Чтобы прочесть задание, на сайте курса или проекта на линейке инструментов нажмите Задания. Появится список заданий с указанием их статуса. Чтобы просмотреть задание, нажмите на его название.

Преподаватель может просмотреть список заданий по отдельным студентам. Для этого в выпадающем списке "Просмотр" нажмите Список заданий по фамилии студента. Можно просмотреть задания так, как они выглядят для студента. Для этого вверху страницы нажмите Просмотр студентом.

Инструкции по оценке задания смотри в Оценка задания и просмотр статуса выполнения работы студентом.

Загрузка заданий для проверки оффлайн

Чтобы загрузить работы студентов для проверки оффлайн:

  1. На сайте курса или проекта, на линейке инструментов нажмите Задания. Появится список заданий с указанием статуса.

  2. На странице со списком заданий, в столбце "Название приложения" под названием нужного задания нажмите Оценить.

  3. На странице оценки нажмите Загрузить все.

  4. Сохраните zip-архив на жестком диске, указав путь сохранения.

  5. Откройте сохраненный zip-архив и извлеките файлы.

  6. После того как файлы извлечены, вы увидите отдельные папки со сданными работами каждого студента. Откройте папку и нажмите на файл внутри для просмотра работы студента.
