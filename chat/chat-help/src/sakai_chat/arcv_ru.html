Чтение, отправка и удаление сообщений в чате

  • Удаление сообщений в чате

Чтение сообщений в чате

Чтобы читать сообщения в чате:

  • На линейке инструментов нажмите Чат. Загрузится чат по умолчанию (указанный владельцем сайта). Чтобы сменить комнату чата, нажмите Сменить комнату чата вверху экрана и выберите нужную комнату чата.

  • Чтобы узнать, кто в данный момент находится в чате, взгляните на область "Участники в чате" в правой части экрана.

  • Сообщения появляются в хронологическом порядке, новые сообщения появляются наверху. Всегда указывается отправитель сообщений.

  • Чтобы скрыть или показать дату или время отправленных сообщений, выберите нужную настройку из выпадающего списка "Показывать": Только время, Дата и время, Только Дата, Ничего, или Только ID сообщения

  • Чтобы показывать только сообщения, отправленные в течение последних трех дней, нажмите За последние 3 дней. Чтобы вернуться к полному списку сообщений, нажмите Все сообщения.

Отправка сообщений в чате

Чтобы отправлять сообщения:

  1. На линейке инструментов нажмите Чат.

  2. В текстовом поле вверху экрана наберите свое сообщение, затем нажмите Добавить сообщение.

    Внимание: Никто не сможет прочитать сообщение, пока вы не нажмете Добавить сообщение, но если оно было добавлено, только преподаватель или участник со специальными правами доступа может его удалить.

  3. Чтобы очистить текстовое поле с неотправленным сообщением, нажмите Очистить.

Удаление сообщений из чата

Чтобы удалить сообщение из чата, необходимы специальные права доступа. Нажмите на иконку "Корзина" рядом с окном чата, затем нажмите Удалить.

Внимание: Если вы не видите иконку "Корзина", значит, у вас нет прав доступа для удаления сообщений.

Чтобы удалить все сообщения из чата, нажмите Настройки, а затем Очистить историю чата. Нажмите Удалить для подтверждения удаления.
