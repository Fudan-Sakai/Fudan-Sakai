Инструмент "Чат": Обзор

Зачем нужен инструмент "Чат"?

Чат используется для общения в реальном времени пользователей, которые находятся на сайте одновременно.

Преподаватели могут создать чат "Онлайн-консультация" для общения со студентами. Распределеные проектные команды используют чат для обсуждения текущего положения дел, или просматривают более ранние обсуждения, которые они пропустили.

Основные возможности

Несколько комнат чата: Администратор сайта может использовать Настройки, чтобы добавлять комнаты чата, что позволяет одновременно поддерживать более одного обсуждения. Участникам нужно нажать Сменить комнату чата для переключения к другому обсуждению.

Комната по умолчанию: Администратор сайта может использовать Настройки, чтобы указать, какая комната будет видимой для участников по умолчанию.

Настройки комнаты чата: Изменения в Настройках влияют на стиль отображения чата.

Очистить историю: Окончательное удаление всех сообщений из комнаты чата.

Важные замечения

  • Инструмент "Чат" оповещает пользователей о других участниках, которые находятся в той же комнате чата. Таким образом, пользователи узнают, кто в данный момент доступен для общения.

  • Администратор сайта может установить, сколько сообщений остается доступно для чтения и на какой срок. Кроме того, он может разрешить пользователям настраивать свой собственный архив.

  • Инструмент "Чат" не позволяет общаться в приватном формате. По умолчанию, все сообщения являются видимыми для всех пользователей. Тем не менее, можно отказать в доступе отдельным ролям (и, соответственно, всем участникам с такими ролями), изменив права доступа к чату.
