新增、編輯或刪除聊天室

註釋：如果要完成本頁中的操作，您必需擁有具有必要權限的角色。如需決定自己的角色，請遵循參與者角色中的說明進行操作。如需瞭解關於角色和權限的基礎知識，請參考權限和角色：概述。

如果您擁有充分的權限，那麼即可使用聊天室工具在網站內新增、編輯或刪除聊天室。同樣，您還可以選擇參與者按一下聊天室工具時將預設載入的聊天室。

新增聊天室

如需新增聊天室，請遵循下列步驟進行操作：

  1. 在功能表欄中，按一下聊天室。

  2. 按一下選項。

  3. 按一下新增聊天室。

  4. 在文字方塊中輸入聊天室標題和描述。

  5. 使用選項按鈕選擇顯示先前訊息的數量。

  6. 按一下更新選項。新建聊天室將出現在可用聊天室清單的「管理聊天室」欄位下方。如需使新建聊天室在參與者按一下聊天室工具時預設進入，那麼按一下設定為預設值 。

編輯聊天室

如需編輯聊天室的名稱、描述或顯示的訊息數量，請遵循下列步驟進行操作：

  1. 在功能表欄中，按一下聊天室。

  2. 按一下選項。

  3. 在您想要編輯的聊天室名下方，按一下編輯。

  4. 轉換聊天室名或描述，或使用選項按鈕選擇顯示先前訊息的數量。

  5. 編輯完成後，按一下更新選項。

註釋：如需更改參與者按一下聊天室工具時預設進入的聊天室，在選項中的「管理聊天室」下方找到相應的聊天室，然後按一下設定為預設值。

刪除聊天室

如需刪除聊天室，請遵循下列步驟進行操作：

  1. 在功能表欄中，按一下聊天室。

  2. 按一下選項。

  3. 在您想要刪除的聊天室名下方，按一下刪除。

  4. 在「刪除聊天室」欄位下方，如需確認刪除操作，按一下刪除。如需保留聊天室，按一下取消。
