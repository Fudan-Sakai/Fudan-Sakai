Добавление объекта

Вы можете добавить объект (создать новую папку или загрузить файл со своего компьютера, в т.ч. URL, текстовый документ или документ в формате HTML) в ящик или одну из папок системы обмена файлами. Для этого нажмите Добавить рядом с заголовком Обмен файлами или существующей папкой. Затем выполните следующие шаги:

Новая папка

  1. На странице "Добавить объект(ы)" откройте выпадающее меню «Тип объекта» и выберите тип Пустая папка. Выберите количество папок, которые Вы хотите создать.

  2. В поле «Заголовок» напечатайте название новой папки, а в поле «Описание» введите краткое описание папки. Если Вы создаете несколько папок, повторите этот этап для каждой из них.

  3. Нажмите Добавить.

Примечание: Участники сайта не могут видеть папки, принадлежащие другим участникам. Участники видят и имеют доступ только к своим собственным папкам и вложенным в них папкам.

Загрузка файла

  1. На странице "Добавить объект(ы)" откройте выпадающее меню «Тип объекта» и выберите тип Загрузка файла.

  2. Используйте меню, чтобы выбрать количество файлов, которые Вы хотите загрузить. Если Вы хотите добавить несколько файлов, повторите описанные ниже шаги для каждого из них.

  3. Под заголовком «Загрузка файла» нажмите Найти..., чтобы найти и выбрать файл, который Вы хотите загрузить.

  4. Под заголовком «Свойства» введите название и описание файла в соответствующих полях.

  5. Выберите в выпадающем меню «Авторские права» один из статусов:"

    • материал доступен для публичного использования
    • Я являюсь правообладателем
    • Материал предназначен для ограниченного использования
    • Я имею исключительные права на использование материала
    • Статус авторских прав не определен.
    • Используйте статус авторского права, указанный ниже.

  6. Отметьте окошко галочкой, для того чтобы включить оповещение о статусе авторских прав и правообладателе для пользователей, открывающих файл.

  7. Нажмите Добавить.

URL

  1. На странице "Добавить объект(ы)" откройте выпадающее меню «Тип объекта» и выберите тип URL (ссылка на web-сайт).. Выберите количество ссылок, которые Вы хотите включить.

  2. В поле "URL" введите URL адрес сайта, ссылку на который Вы хотите дать.

  3. Под заголовком «Свойства» введите название и описание ссылки в соответствующих полях.

  4. Нажмите Добавить.

Документ HTML

  1. На странице "Добавить объект(ы)" откройте выпадающее меню «Тип объекта» и выберите тип Страница HTML .

  2. Под заголовком «Страница HTML», в предложенном поле для текста введите HTML код Вашей страницы. Вы можете использовать редактор WYSIWYG для форматирования своей страницы.

  3. Под заголовком «Свойства» введите название и описание страницы, которую Вы добавляете, в соответствующих полях.

  4. Выберите в выпадающем меню «Авторские права» один из статусов:

    • Материал доступен для публичного использования.
    • Я являюсь правообладателем.
    • Материал предназначен для ограниченного использования.
    • Я имею исключительные права на использование материала.
    • Статус авторских прав не определен.
    • Используйте статус авторского права, указанный ниже.

  5. Отметьте окошко галочкой, для того чтобы включить оповещение о статусе авторских прав и правообладателе для пользователей, открывающих файл.

  6. Нажмите Добавить.

Текстовый документ

  1. На странице "Добавить объект(ы)" откройте выпадающее меню «Тип объекта» и выберите тип Текстовый документ.

  2. В предложенном текстовом окне введите текст, который Вы хотите разместить.

  3. Под заголовком «Свойства» введите название и описание документа в соответствующих полях.

  4. Выберите в выпадающем меню «Авторские права» один из статусов:

  6. Нажмите Добавить.

Редактирование объекта

Выполните следующие шаги, чтобы отредактировать информацию о ящике, папке или отдельном объекте системы обмена файлами:

  1. На линейке инструментов выберите Обмен файлами.

  2. Нажмите Редактировать рядом с ящиком системы Обмен файлами, папкой или объектом,содержимое которого Вы хотите изменить. Если Вам не видно объект, в который Вы хотите внести изменения, нажмите на черные треугольники слева от надписи «Заголовки», чтобы открыть содержимое всех папок. Вы также можете нажать на значок папки рядом с той папкой, которую Вы хотите открыть. (Если в папке содержатся объекты, не отображенные в дереве, на значке папке будет стоять значок "плюс").

  3. Если Вы редактируете сам ящик обмена файлами, Вы можете вносить изменения только в поле «Описание» (т.е., Вы не можете изменить имя ящика). Если Вы редактируете папку, Вы можете вносить изменения в полях «Заголовок» и «Описание». Если Вы изменяете информацию или объект, поля, которые Вы можете редактировать, зависят от типа объекта:

    • Загруженные файлы: Вы можете нажать Найти... чтобы найти и выбрать файл для замены. Кроме того, Вы можете вносить изменения в поля «Заголовок» и «Описание», изменять статус авторских прав, указывать, отображать или нет предупреждение о статусе авторского права, изменять тип файла.

    • URL: Вы можете редактировать поля «Заголовок» и «Описание», а также заменять ссылки на новые.

    • HTML или текстовый документ: В выпадающем меню «Содержание» выберите Редактировать содержание документа, а затем внесите необходимые изменения в соответствующем текстовом окне, либо выберите Заменить на документ из моего компьютера ,чтобы найти файл для замены (см. раздел «Загруженные файлы» выше). Кроме того, Вы можете вносить изменения в поля «Заголовок» и «Описание», изменять статус авторских прав, указывать, отображать или нет предупреждение о статусе авторского права, изменять тип файла.

  4. Нажмите Обновить чтобы сохранить внесенные изменения.

Удаление объекта

Чтобы удалить объект или папку из системы обмена файлами, отметьте галочкой окошко рядом с удаляемым объектом, а затем нажмитеУдалить отмеченное , а затем подтвердите удаление, нажав Удалить.

Если руководитель сайта удаляет папку участника, папка на имя этого участника снова появится (в поле зрения руководителя), когда участник загрузит новый файл.
