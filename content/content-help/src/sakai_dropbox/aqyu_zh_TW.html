新增項目

您可以將項目（新建資料夾、電腦上的檔案、URL、文字文件或 HTML文件）新增到投遞箱或投遞箱資料夾中。如需進行上述操作，按一下投遞箱或當前資料夾標題旁的新增。然後遵循適當的步驟進行操作：

新建資料夾

  1. 從「新增」下拉式清單中選擇建立資料夾。

  2. 在「資料夾名稱」欄位中，輸入新建資料夾的名稱。按一下「新增此項目詳細資料」連結，「描述」欄位會現出，然後在「描述」欄位中，輸入資料夾的簡短說明。如果您需要建立多個資料夾，那麼重複本步驟建立各資料夾。

  3. 按一下立即建立資料夾。

註釋：網站參與者無法看到屬於其他參與者的投遞箱資料夾。參與者只能看到和存取自己的投遞箱資料夾和子資料夾。

檔案上載

  1. 從「新增」下拉式清單中選擇檔案上載。

  2. 使用「新增另一個檔案」來新增另一個檔案。如果選擇新增多個檔，為各檔重複以下步驟。

  3. 在「上載檔案」欄位，按一下瀏覽...(Browse...)尋找並選擇需要上載的檔案。

  4. 在「顯示名」欄位中輸入檔案的標題和描述。

  5. 按一下「新增此項目詳細資料」來顯示詳細資料，從「版權狀態」下拉式清單選擇下列選項中的一項：

    • 資料已經進入公共領域。
    • 我持有版權。
    • 資料必需遵守公平使用例外規定。
    • 我已經獲得了使用本資料的許可。
    • 版權狀態未知。
    • 使用下方的版權。

  6. 使用核取方塊選擇是否顯示版權警示，以及是否需要在他人存取檔案的時候確認版權狀態。

  7. 按一下立即上載檔案。

URL

  1. 從「新增」下拉式清單中選擇 URL（新增網站連結）。

  2. 在「URL」欄位中，輸入需要連結的目標網站的 URL。

  3. 在「網站名稱」與「描述」的相應欄位中輸入連結的標題和描述。使用「新增此項目詳細資料」來顯示「描述」欄位。

  4. 按一下立即新增網站連結。

HTML 文件

  1. 從「新增」下拉式清單中選擇 建立HTML頁面。

  2. 在「建立HTML頁面」下的文字欄位中輸入頁面的 HTML代碼。可使用 WYSIWYG 編輯器編排頁面的格式。

  3. 在「名稱」與「描述」的相應欄位中輸入您正在新增的頁面的標題和描述。

  4. 從「版權狀態」下拉式清單選擇下列選項中的一項：

    • 資料已經進入公共領域。
    • 我持有版權。
    • 資料必需遵守公平使用例外規定。
    • 我已經獲得了使用本資料的許可。
    • 版權狀態未知。
    • 使用下方的版權。

  5. 使用核取方塊選擇是否顯示版權警告，以及是否需要在他人存取檔的時候確認版權狀態。

  6. 按一下完成。

文字文件

  1. 從「新增」下拉式清單中選擇建立文字文件。

  2. 在文字欄位中輸入您想要顯示的文字，然後按一下「繼續」按妞。

  3. 在「名稱」與「描述」的相應欄位中輸入文件的標題和描述。

  4. 從「版權狀態」下拉式清單選擇下列選項中的一項：

    • 資料已經進入公共領域。
    • 我持有版權。
    • 資料必需遵守公平使用例外規定。
    • 我已經獲得了使用本資料的許可。
    • 版權狀態未知。
    • 使用下方的版權。

  5. 使用核取方塊選擇是否顯示版權警告，以及是否需要在他人存取檔的時候確認版權狀態。

  6. 按一下完成。

編輯項目

遵循下列步驟修改關於投遞箱或投遞箱中的資料夾或項目的資訊：

  1. 在功能表欄中，按一下 投遞箱。

  2. 按一下您想要更改其相關資訊的投遞箱、資料夾或項目旁的編輯詳細資料。如果您無法看到您想要修改的項目，按一下「標題」左邊的黑色三角以展開資料夾的全部內容。或者按一下您想要展開的資料夾旁的資料夾圖示。（如果資料夾中的項目當前沒有完全顯示，那麼資料夾圖示上會出現一個加號。）

  3. 如果您正在修改投遞箱，那麼您只能更改「描述」欄位（即，您無法更改投遞箱的名稱）。如果您正在修改資料夾，那麼您可以更改「名稱」和「描述」欄位。如果您正在修改項目的資訊，那麼您可以更改的欄位取決於項目的類型：

    • 上載的檔案：可按一下瀏覽...尋找和選擇替換檔案。也可編輯「名稱」和「描述」欄位、修改版權狀態、選擇是否顯示版權警告，以及更改檔案類型。

    • URL： 您可以編輯「名稱」和「描述」欄位，以及輸入替換 URL。

    • HTML 或文字文件：從「內容」下拉式清單中選擇編輯內容，然後在文字方塊中輸入修改的內容，或選擇上載新版本選擇替換檔案（請參考上文中的「上載的檔案」部分）。也可編輯「名稱」和「描述」欄位、修改版權狀態、選擇是否顯示版權警告，以及更改檔類型。

  4. 按一下更新儲存更改。

刪除項目

如需刪除投遞箱或投遞箱資料夾中的項目，選中該項目標題旁的核取方塊，然後按一下刪除。在確認頁面中，按一下刪除。

如果網站負責人刪除了一個參與者的資料夾，那麼當該參與者再次上載檔案時，他的資料夾將重新出現（在網站負責人檢視中）。
