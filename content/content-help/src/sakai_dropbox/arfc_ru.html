Обмен файлами: обзор

Для чего нужен обмен файлами

Система обмена файлами дает преподавателям и студентам возможность создавать документы с открытым доступом в рамках личной папки для каждого студента. Обмен файлами выступает в качестве универсального информационного Ресурса, позволяя загружать многочисленные файлы различных типов. В системе обмена файлами возможно создание вложенных папок (папок внутри основных папок).

Основные возможности:

Отчеты о развитии: Отчеты о развитии: Преподаватели могут использовать Обмен файлами, чтобы подготавливать индивидуальные отчеты о развитии каждого студента в группе.

Добавление участников: Когда Вы добавляете участников или студентов на рабочий сайт (в том числе вручную), им автоматически присваиваются папки для обмена файлами (если сервис Обмен файлами включен).

Важные замечания:

  • Новый объект автоматически размещается в той папке, которая является открытой на момент создания объекта.

  • Для обычных текстовых документов, страниц в формате HTML и web ссылок, которые Вы создаете в среде обмена файлами, помимо редактирования заголовка и описания, доступно редактирование содержания. Для загруженных файлов доступно только редактирование информации о файле. Для того, чтобы изменить содержание, Вам необходимо загрузить новую версию файла.

  • Когда бы Вы ни добавляли объект в систему обмена файлами, у Вас есть возможность отправить автоматическое уведомление по email.

  • Возможность для преподавателей: все недавно обновленные папки системы обмена файлами отмечаются оранжевыми звездочками.

  • Убедитесь в том, что Вы знакомы с законами и правилами относительно использования материалов, защищенных авторским правом.
