Копирование элемента инструмента "Ресурсы"

Внимание: Для того чтобы совершать все эти действия, вам должна быть назначена роль с соответствующими полномочиями. Для того чтобы определить, какая у вас роль, перейдите на страницу Роли участников. Основные сведения о ролях и правах доступа изложены также в Права доступа и роли: обзор.

Чтобы копировать элемент инструмента "Ресурсы":

  1. На линейке инструментов нажмите Ресурсы.

  2. Рядом с название элемента, нажмите Действия, затем нажмите Копировать. Чтобы скопировать несколько элементов за один раз, отметьте их флажком, затем над списком элементов нажмите Копировать.

    Внимание: Чтобы скопировать элемент в ту же самую папку, где находится оригинал, не нужно нажимать Копировать. Вместо этого, нажмите Действия, затем нажмите Дублировать.

  3. Укажите папку, в которую нужно поместить копию, затем нажмите иконку "Вставить копируемые элементы"
    или нажмите Действия и выберите Вставить копируемые элементы. Чтобы скопировать элемент на другой сайт, нажмите Показать остальные сайты; откроются папки других сайтов (включая раздел "Мой сайт"), к которым у вас есть доступ.

  4. Чтобы отменить действие копирования, нажмите кнопку "обновить".
