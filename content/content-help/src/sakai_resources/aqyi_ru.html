Загрузка файлов в инструмент "Ресурсы"в помощью WebDAV в операционной системе Mac OS X

WebDAV расшивровывается как "Web-based Distributed Authoring and Versioning (Распределенное создание и правка материалов в сети)". Он представляет собой набор расширений к протоколу HTTP, которые позволяют участникам совместно изменять и управлять файлами на удаленных веб-серверах.

Нужно загрузить хотя бы один объект (то есть папку или файл) в инстурмент "Ресурсы", чтобы использовать WebDAV. Более подробную информацию смотри в Ресурсы.

Сохранение файлов с помощью WebDAV (непример, напрямую из других программ, таких как Word, Excel, или Internet Explorer) ненадежно. Для лучших результатов, сначала сохраните файл на компьютере, а затем загрузите на сайт.

Внимание: Имена файлов, которые включают некоторые знаки, не могут быть сохранены. Список таких знаков смотри в Неподдерживаемые символы.

Mac OS X 10.4 (Tiger) и 10.5 (Leopard)

Чтобы использоватьWebDAV для загрузки файлов в n Mac OS X 10.4 (Tiger) и более поздних версиях:

  1. В Finder нажмите Cmd-kВ . (Также в меню Пуск можно выбрать Соеденить в сервером.)

  2. В поле "Server adress" введине URL, указывающий расположение инструмента "Ресурсы".

    Свяжитесь с администрацией сайта, чтобы узнать формат URL; вероятнее всего, он включает ID сайта.

Внимание: Инструкции по определениюID вашего сайта смотри в Определение ID сайта.

Теперь можно перетаскивать файлы и папки между вашим компьютером и сайтом, к которому выполнено подключение.

Опционально, вы можете нажат знак + справа от адреса сервера, когда заходите на него, и сервер будет добавлен к вашему списку избранных серверов.

Когда вы загружаетесь на сервер, можно установить пароль, который будет записан инструментом Keychain. Если это будет сделано, вы не будете получать запрос о пароле каждый раз, когда загружаетесь в инстурмент "Ресурсы" раздела "Мой сайт" или другого сайта, к которому имеете доступ. Не делайте этого, если ваш компьютер не защизщен от постороннего доступа.

Внимание: Когда вы используете WebDAV tдля доступа к ресурсом, используя, вы заметите, что были созданы файлы .DS_store. База знаний Apple предоставляет инструкции по предотвращению этого свойства.

Mac OS X 10.3 (Panther) и более ранние версии

Использование WebDAV для загрузки и доступа к файлам в операционной системе Mac OS X 10.3 (Panther) и более ранних версиях требуетспециальную программу под названием Goliath.

Внимание : Программа Goliath обладает некоторыми особенностями установки, и поэтому может работать не постоянно. По этой причине, возможен только ограниченный доступ, если вы испытываете проблеммы с импользованием программы.

Можно загрусить Goliath c:

http://www.webdav.org/goliath/installs/Goliath1_0_1.dmg.gz

Когда вы загрузили файл, нажмите на него дважды, чтобы разархивировать его. Затем нажмите дважды на файл .dmg. Должен возникнуть образ диска под названием Goliath 1.0.1 . Следуйте этим инструкциям:

  1. Перетащите папку, содержащую Goliath на свой жесткий диск.

  2. Откройте приложение Goliath.

  3. В поле "Адрес сервера" введине URL, указывающий расположение инструмента "Ресурсы".

    Свяжитесь с администрацией сайта, чтобы узнать формат URL; вероятнее всего, он включает ID сайта.

Внимание: Инструкции по определениюID вашего сайта смотри в Определение ID сайта.

Когда вы впервые загружаетесь на сайт, вы можете получить запрос о сертификате этого сайта. Нажмите OK, если требуется.

Теперь можно перетаскивать файлы и папки между вашим компьютером и сайтом, к которому выполнено подключение.
