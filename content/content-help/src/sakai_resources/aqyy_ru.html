Инструмент WebDAV

Зачем нужен инструмент "WebDAV"

WebDAV - это инструмент, который позволяет передачу файлов и папок с вашего компьютера в папки инстурмента "Ресурсы" вашего раздела Мой сайт или сайта курса или сайта проекта. WebDAV расшивровывается как "Web-based Distributed Authoring and Versioning (Распределенное создание и правка материалов в сети)". Он представляет собой набор расширений к протоколу HTTP, которые позволяют участникам совместно изменять и управлять файлами на удаленных веб-серверах.

Во время использование инструмента WebDAV, компьютер будет использовать папки инструмента "Ресурсы" также, как и локальные папки. Можно перетаскивать файлы и папки из инструмента "Ресурсы", также как и проводнике WindowsXP или Mac.

Более подробную информацию о WebDAV, включая технические спецификации и основные функции, смотри на странице "Часто задаваемые вопросы" WebDAV на сайте:

http://www.webdav.org/other/faq.html

Основные возможности

Настройки блокировки: Если вы используете WebDAV для работы с файлами и папками в инстурменте "Ресурсы", к которому имеют доступ несколько пользователей, можно использовать настройки блокировки, чтобы предотвратить перезапись одного и того же файла разными пользователями без объединения изменений.

Windows против Mac: На компьютере с Windows XP, WebDAV работает через папку "Сетевое окружение". Операционная система Mac OS X 10.4 (Tiger) поддерживает безопасное соединение WebDAV. Если вы используете Mac OS X 10.3 (Panther) или более раннюю версию, вам придется установить программу Goliath (Голиаф), если вы хотите использовать WebDAV.

Важные замечания

  • Нужно загрузить хотя бы один объект (то есть папку или файл) в инстурмент "Ресурсы", чтобы использовать WebDAV.

  • Если вы ипользуете WebDAV в Windows через "Сетевое окружение", нельзя удалить файл, перетащив его в "Корзину", выберите файл и нажмите Удалить.

  • WebDAV запросит ввод ID каждого сайта, доступ к которому вы хотите получить. ID сайта уникален для саждого сайта. Чтобы посмотреть ID сайта, на линейке инстурментов нажмите Ресурсы. Нажмите Загрузить-Скачать ресурсы. ID сайта появится вверху страницы.
