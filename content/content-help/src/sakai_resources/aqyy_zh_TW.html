WebDAV

功能

WebDAV 是一種可用於輕鬆地將本機電腦上的檔案和資料夾傳輸至我的工作區或您所屬的課程網站或專案網站中的資源資料夾中的技術。WebDAV 是指「網路分散式授權與版本控制」。它是一種 HTTP 的擴展協議，透過該協議使用者可協同編輯和管理遠端網路伺服器上的檔案。

一旦您設定 WebDAV 後，您的電腦將會將資源與本機系統上的任何其他資料夾同樣對待處理。您將能夠從資源拖放檔案和資料夾，正如 Windows XP 的我的電腦或 Mac 的 Finder 中的檔案和資料夾操作一樣。

如需瞭解關於 WebDAV 的更多資訊，包括技術規格和主要功能，請存取下列網站檢視 WebDAV 常見問題解答：

http://www.webdav.org/other/faq.html

主要概念

鎖定屬性：如果您正使用 WebDAV 操作共用資源中的檔案和資料夾，您可以使用該鎖定功能避免兩個或更多使用者在未經事先合併更改操作前，覆蓋相同的資源內容。

Windows vs. Mac：在 Windows XP 電腦上，WebDAV 透過網上鄰居工作。Mac OS X 10.4 (Tiger)支持安全 WebDAV 連接。如果您使用 Mac OS X 10.3 (Panther)或更低版本，那麼您必需下載 Goliath 軟體後才能使用 WebDAV。

注意事項

  • 開始設定 WebDAV 前，您的資源中必需至少含有一個物件（即檔案或資料夾）。

  • 如果您正在 Windows 中使用 WebDAV，並且已經透過網上鄰居將磁碟映射至應用程式，那麼您將無法透過將檔案從網上鄰居視窗拖放至回收站來刪除檔案。如需刪除檔案，選擇該檔案，並按一下刪除。

  • WebDAV 將要求您輸入您想要存取的每個網站的網站 ID。網站 ID 是每個工作網站的唯一識別符。如需檢視網站 ID，在工作網站的功能表欄中，按一下資源。按一下上載 – 下載多個資源。捲動至「您的網站 ID」部分檢視網站 ID。
