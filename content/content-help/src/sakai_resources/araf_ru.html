Введение

WebDAV расшивровывается как "Web-based Distributed Authoring and Versioning (Распределенное создание и правка материалов в сети)". Он представляет собой набор расширений к протоколу HTTP, которые позволяют участникам совместно изменять и управлять файлами на удаленных веб-серверах.

Нужно загрузить хотя бы один объект (то есть папку или файл) в инстурмент "Ресурсы", чтобы использовать WebDAV. Более подробную информацию смотри в Ресурсы.

Сохранение файлов с помощью WebDAV (непример, напрямую из других программ, таких как Word, Excel, или Internet Explorer) ненадежно. Для лучших результатов, сначала сохраните файл на компьютере, а затем загрузите на сайт.

Внимание: Имена файлов, которые включают некоторые знаки, не могут быть сохранены. Список таких знаков смотри в Неподдерживаемые символы.

Использование Windows Vista

Внимание: Соединение с инстурментов "Ресурсы" на сайте с помощью WebDAV с компьютера с операционной систесой Windows Vista работает ненадежно. Разработчики в данный момент исследуют эту проблему. Тем не менее, Microsoft выпустил обновление, которое может помочь исправить эту проблему некоторым пользователям. Более подробную информацию об этом обновлении и как его загрузить смотри в статье 907306 в базе знаний Microsoft's.

Базу знаниц Microsoft можно найти по электронному адресу:

http://support.microsoft.com/default.aspx

Чтобы добавитьна компьютер ярлык, через который можно загружать или открывать файлы инструмента "Ресурсы" с помощью WebDAV:

  1. На рабочем столе дважды нажмите на значок Мой компьютер.

  2. Нажмите Подключить сетевой диск.

  3. В окне Подключить сетевой диск нажмите Подключение к веб-узлу, на котором можно хранить свои файлы и изображения, затем нажмите Далее.

  4. Нажмите Выберите другое сетевое размещений, затем нажмите Далее.

  5. В текстовом поле "Сетевой адрес или адрес в интернете" введите URL инструмента "Ресурсы" нужного сайта, нажмите Далее.

    Свяжитесь с администрратором, чтобы узнать URL. Скорее всего, он включает ID вашего сайта.

    Внимание: Инструкции по определениюID вашего сайта смотри в Определение ID сайта.

Теперь можно перетаскивать файлы и папки между вашим компьютером и сайтом, к которому выполнено подключение. Чтобы открыть папку, нажмите на него правой кнопкой мыши и выберите Открыть. Чтобы удалить элемент, нажмите на него правой кнопкой мыши и выберите Удалить.

Если у вас вознили проблемы с перетаскиванием файлов, нажмите правой кнопкой мыши на название папки, которую нужно скопировать и выберите Копировать. Затем нажмите правой кнопкой мыши на папку, куда необходимо ее вставить и выберите Вставить.

Возможно, нужно будет обновлять ваш браузер, чтобы файл появился в инструменте "Ресурсы".

Наверх

Использование Windows XP

В Windows XP можно использовать опцию "Добавить новый элемент в сетевое окружение" или Internet Explorer, чтобы загружать и открывать файлы с помощью WebDAV, если у вас есть необходимые права доступа.

Использование инструмента "Добавить новый элемент в мое сетевое окружение"

Чтобы использовать инструмент "Добавить новый элемент в мое сетевое окружение" для работы с WebDAV:

  1. На рабочем столе дважды нажмите на Мое сетевое окружение.

  2. В столбце "Сетевые задачи" нажмите Добавить новый элемент в сетевое окруженрие.

  3. На окне Мастера лобавления в сетевое окружение нажмите далее.

  4. Нажмите Выберите другое сетевое размещений, затем нажмите Далее.

  5. В текстовом поле "Сетевой адрес или адрес в интернете" введите URL инструмента "Ресурсы" нужного сайта, нажмите Далее.

    Свяжитесь с администрратором, чтобы узнать URL. Скорее всего, он включает ID вашего сайта.

    Внимание: Инструкции по определениюID вашего сайта смотри в Определение ID сайта.

Теперь можно перетаскивать файлы и папки между вашим компьютером и сайтом, к которому выполнено подключение. Чтобы открыть папку, нажмите на него правой кнопкой мыши и выберите Открыть. Чтобы удалить элемент, нажмите на него правой кнопкой мыши и выберите Удалить.

Использование Internet Explorer

Используя Internet Explorer, можно загружать и открывать файлы с помощью WebDAV:

  1. В браузере Internet Explorer в меню Файл нажмите Открыть.

  2. В появившемся окне отметьте флажок рядом с Открыть как веб-папку. В текстовом поле "Открыть" введите URL страницы инструмента "Ресурсы", затем нажмите OK.

    Свяжитесь с администрратором, чтобы узнать URL. Скорее всего, он включает ID вашего сайта.

    Внимание: Инструкции по определениюID вашего сайта смотри в Определение ID сайта.

Содержание папки назначения откроется в отдельном окне, где вы сможте открывать отдельные файлы, переходить к другим папкам на компьютере и передавать файлы между папками.

Когда вы закончили, закройте Internet Explorer, чтобы отключиться.
