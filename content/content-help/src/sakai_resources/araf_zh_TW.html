使用 Windows XP 或 Vista 的 WebDAV 上載檔案至資源

在本頁中，您將瞭解：

  • 簡介
  • 使用 Windows Vista
  • 使用 Windows XP

簡介

WebDAV 是指「網路分散式授權與版本控制」。它是一種 HTTP 的擴展協議，透過該協議使用者可協同編輯和管理遠端網路伺服器上的檔案。

設定 WebDAV 前，目標資源工具中必需至少含有一個物件（即檔案或資料夾）。如需説明，請參考資源。

直接將檔案儲存至 WebDAV（即從應用程式內，例如 Word、Excel 或 Internet Explorer），這種操作並不可靠。為了獲得最佳效果，請先將檔案儲存到您的電腦中，然後上載至網站。

註釋：不支援含有某些字元的檔案名。如需檢視不支援字元清單，請參考不支援的字元。

使用 Windows Vista

註釋：使用 WebDAV 在運行 Windows Vista 的電腦上連接網站資源，這種操作並不可靠。軟體發展商目前正在研究這一問題。但是，Microsoft 目前已經發佈了軟體更新，有助於解決一些使用者的連接問題。如需瞭解關於更新的資訊，並下載更新程式，請參考 Microsoft 知識庫中的 907306 號文章。

可存取下列網站搜尋 Microsoft 知識庫：

http://support.microsoft.com/default.aspx

如需在 Windows Vista 電腦中建立捷徑，以便使用 WebDAV 上載和存取資源中的檔案：

  1. 從桌面上按兩下電腦。

  2. 按一下連線網路磁碟機。

  3. 在連線網路磁碟機螢幕中，按一下連線至您可以用來儲存文件及圖片的網站，然後按一下下一步。

  4. 按一下選擇自訂網路位置，然後按一下下一步。

  5. 在「網際網路或網路位址」欄位中，輸入指向目標資源工具的 URL，然後按一下下一步。

    與網站管理員聯絡瞭解 URL 格式。該格式中可能包含網站的網站 ID。

    註釋：如需瞭解如何尋找網站 ID 的說明，請參考尋找網站的網站 ID

您現在可在電腦和您剛才連接的網站之間拖放檔案和資料夾。如需開啟資料夾，按右鍵並選擇檔案總管或開啟。 如需刪除項目，按右鍵該項目並選擇刪除 。

如果您有困難拖放，在您要複製的檔案或資料夾上按右鍵，並選擇複製。然後在您想要放置的目錄上按右鍵，並選擇貼上。

您可能需要重新整理瀏覽器來使檔案在資源的目錄中出現。

使用 Windows XP

在 Windows XP 中，如果您擁有足夠的權限，那麼就可使用新增網路上的芳鄰精靈或 Internet Explorer 透過 WebDAV 上載和存取檔案。

使用新增網路上的芳鄰精靈

如需使用新增網路上的芳鄰精靈為電腦新增捷徑，以便使用 WebDAV 上載和存取資源中的檔案：

  1. 在桌面上，按兩下網路上的芳鄰。

  2. 在「網路工作」窗格中，按一下新增一個網上鄰居。

  3. 在歡迎螢幕中，按一下下一步。

  4. 選擇選擇其他的網路位置，然後按一下下一步。

  5. 在「網際網路或網路位址：」欄位中輸入指向目標資源工具的 URL。

    與網站管理員聯絡瞭解 URL 格式。該格式中可能包含網站的網站 ID。

    註釋：如需瞭解如何尋找網站 ID 的說明，請參考尋找網站的網站 ID

您現在可在電腦和您剛才連接的網站之間拖放檔案和資料夾。但是，您無法透過將網站中的項目拖放至桌面上的回收站來刪除項目。如需刪除項目，在該頁目上按右鍵，並選擇刪除 。

使用 Internet Explorer

使用 Internet Explorer，您可以透過 WebDAV 上載和存取檔案。如需進行上述操作：

  1. 在 Internet Explorer 中，從檔案功能表選擇開啟...。

  2. 在隨後出現的視窗中，選中核取方塊開啟成網頁資料夾。在「開啟：」欄位中輸入指向目標資源工具的 URL，並按一下確定。

    與網站管理員聯絡瞭解 URL 格式。該格式中可能包含網站的網站 ID。

    註釋：如需瞭解如何尋找網站 ID 的說明，請參考尋找網站的網站 ID

如需在 Windows Vista 電腦中建立捷徑，以便使用 WebDAV 上載和存取資源中的檔案：

直接將檔案儲存至 WebDAV（即從應用程式內，例如 Word、Excel 或 Internet Explorer），這種操作並不可靠。為了獲得最佳效果，請先將檔案儲存到您的電腦中，然後上載至網站。
