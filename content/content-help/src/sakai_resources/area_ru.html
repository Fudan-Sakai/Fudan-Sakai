Ресурсы: обзор

Зачем нужен инструмент "Ресурсы"

Используя инструмент "Ресурсы" можно организовывать совместный доступ в различным материалам для участников сайта, или сделать их доступными для любых пользователей. Можно также использовать инструмент "Ресурсы" исключительно для своих целей в разделе "Мой сайт".

Можно загружать файлы (включая рабочие документы, электронные таблицы, презентации или видео) а также создавать и отправлять страницы HTML или простые текстовые документы или ссылки на важные веб-сайты.

Можно организовать структуру элементов инструмента "Ресурсы", используя папки, кроме того, можно указать, какие группы или типы пользователей будут иметь доступ к различным папкам.

Кроме того, можно в любой момент показывать или скрывать элемент, а также устанавливать временной промежуток, в который он будет открыт.

Основные возможности

Просмотр элементов: Нажатие на название инструмента Ресурсы на линейке ринструментов открывает список ресурсов сайта. Нажмите на название элемента, чтобы открыть его.

Чтобы просмотреть содержимое папки, нажмите на название папки или нажмите на иконку папки справа, чтобы открыть содержимое в том же списке. Чтобы открыть или закрыть все папки, нажмите на черные стрелки рядом с надписью "Название".

Можно сортировать элементы по названию, размеру, имени пользователя, загрузившего элемент или дате последнего изменения. Владелец сайта может изменять порядок, в каком элементы появляются в списке.

Добавление элементов: Чтобы добавить элемент, в меню Добавить рядом с названием папки, куда нужно добавить элемент, выберите тип элемента для добавления.

Организация структуры списка: Администраторы сайта могут структурировать элементы в папки, а также устанавливать порядок, в котором они появляются в списке. Чтобы изменить порядок, в меню Действия нажмите Упорядочить.

Публичный доступ: Каждый элемент доступен через URL, который можно показать другим пользователям.

Перетаскивания элементов: Можно перетакивать файлы с компьютера и на компьютер с помощью WebDAV. Более подробную информацию смотри в Загрузка файлов с помощью WebDAV в операционной системе Windows XP или Vista или Загрузка файлов с помощью WebDAV в операционной системе Mac OS X.

Групповой доступ: Можно установить доступ к нужным элементам для определенных групп, это дает возможность лучше контролировать доступ к файлам инструмента "Ресурсы".

Контроль доступа: Использую опцию "Права доступа", владельцы сайта могут устанавливать, какие типы пользователей могут читать, отправлять, изменять и/или удалять файлы в нужных папках. Владельцы сайта и преподаватели могут сделать ресурс доступным для всех, поставив флажок рядом с надписью "публичный доступ" при публикации.

Важные замечания

  • Многие преподаватели еженедельно отправляют данные на хранение в инструмент "Ресурсы". Они могут включать ссылки на веб-сайты и на другие типы документов, например файлы PDF. Некоторые группы также отправляют презентации или слацды, использованные на лекциях.

  • Устанавливая права доступа к папке, владелец сайта для совместной работы может тем самым создать место для хранения важной документации и разрешить только некоторым частникам изменять документы.
