資源：概述

功能

使用資源工具，您可以安全地與網站中的成員共用任何類型的資料，或公開這些資料。還可在我的工作區內設定自己的私人資源區。

透過資源工具，您可以上載檔案（例如文字處理文件、試算表、幻燈片和視訊），以及建立和發佈 HTML (web)頁面和簡單文字文件，並共用有用網站的連結。

您可以將資源項目組織到資料夾中，並且控制可以存取資源項目並將其新增到不同資料夾中的群組和使用者類型。

您還可以隨時顯示或隱藏項目，並設定項目可用性的開始和/或結束時間。

主要概念

瀏覽項目：當您按一下網站功能表欄中的資源時，您將看到該網站的資源項目清單。按一下項目標題開啟項目。

如需瀏覽資料夾的內容，按一下資料夾名稱進入資料夾，或按一下資料夾圖示在清單中展開內容。如需展開或摺疊全部資料夾，按一下頂部「標題」旁的黑色雙箭頭。

可按照標題、大小、資源建立人和上一次修改的日期為項目排序。網站負責人可自訂項目顯示的順序。

新增項目：如需新增項目，將滑鼠滑過您想要存儲項目的目標資料夾相應的新增連結。然後從隨後出現的下拉式清單中選擇您想要新增的項目類型。

組織項目：網站負責人可將項目組織到資料夾和子資料夾中，並且自訂項目顯示的順序。如需重新排序項目，將滑鼠滑過操作，並選擇重新排序。

共用資源：每個項目均可透過 URL 存取，並且與網站參與者共用。

拖放：您可以使用 WebDAV 將檔案拖放至桌面或從桌面拖放至其他位置。如需瞭解更多資訊，請參考使用 Windows XP 或 Vista 的 WebDAV 上載檔案至資源或使用 Mac OS X 的 WebDAV 上載檔案至資源

群組空間：您可以與預先定義的群組共用項目，以便更好地控制存取資源項目的使用者。

控制存取：網站擁有人可使用權限功能控制可閱讀、發佈、修改和/或刪除特定資料夾中檔案的使用者的類型。網站擁有人和講師可在選中「公共檢視」設定的情況下，透過選擇公佈資源使全部人都可存取該資源。

注意事項

  • 許多班級每週都在自己的資源中發佈閱讀資料。這些資料可包括網站連結，以及其他類型的文件，如 Adobe PDF 檔案。一些班級還發佈講課中使用的簡報內容或幻燈片。

  • 透過設定資料夾權限，一個大規模的協作網站可將資源中的一個資料夾作為空間使用，用於存檔重要文件，並僅允許某些網站參與者修改這些文件。
