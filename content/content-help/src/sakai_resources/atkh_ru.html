Установление публичного просмотра элементов инструмента "Ресурсы"

Когда вы используете инструмент "Ресурсы" для загрузки элемента, можно сделать его доступным только для участников сайта или установить публичный просмотр (то есть, сделать его доступным для всех в сети Интернет). Эта опция позволяет установить доступ к файлу для участников других сайтов.

Чтобы позволить участникам других сайтов просматривать ваши элементы инструмента "Ресурсы":

  1. Нажмите Мой сайт, чтобы выбрать личные ресурсы или нажмите на название сайта курса или проекта на линейке сайтов, чтобы выбрать ресурсы оттуда.

  2. На линейке инструментов нажмите Ресурсы.

  3. Найдите папку, для которой требуется установить публичный просмотр и справа от ее названия на выпадающем меню Действия нажмите Исправить детали.

  4. Под надписью "Доступность" выберите Этот файл для публичного просмотра. или, для папок Эта папка для публичного просмотра..

  5. Нажмите Обновить.

После того, как вы установили публичный просмотр для своих файлов, можно открывать их через Интернет. Чтобы найти URL файла инструмента "Ресурсы" смотри в Нахождение URL элемента инструмента "Ресурсы" .
