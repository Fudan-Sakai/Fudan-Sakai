Определение ID сайта

ID сайта курса, проекта или портфолио - это строчка букв и цифр, которая стоит после site/ в адресе сайта.

Например, возьмем следующий адрес:

http://sakai-deathstar.org/portal/site/836869b1-3997-4f29-803b-986be9fd270c

В приведенном примере ID сайта: 836869b1-3997-4f29-803b-986be9fd270c.

Кроме того, владельцы сайта могут найти ID сайта в инструменте "Ресурсы". На линейке инструментов нажмите Ресурсы, затем нажмите Загрузить - Скачать ресурсы, чтобы увидеть ID сайта.
