Создание папок для отдельных блоков пользователей в инстурменте "Ресурсы"

Внимание: Для того чтобы совершать все эти действия, вам должна быть назначена роль с соответствующими полномочиями. Для того чтобы определить, какая у вас роль, перейдите на страницу Роли участников. Основные сведения о ролях и правах доступа изложены также в Права доступа и роли: обзор.

Инстурмент "Ресурсы" позволяет создавать папки, доступные только для членов отдельного блока пользователей.

Создание блока пользователей

Чтобы создать папку для отдельного блока пользователей, нужна сперва создать блок пользователей. Более подробные инстуркции смотри в Управление блоками пользователей.

Создание папки блока пользователей

Чтобы создать папку блока пользователейr:

  1. На линейке инструментов нажмите Ресурсы. Рядом с папкой, в которой нужно создать папку блока пользователей в выпадающем списке Добавить выберите Создать папку.

  2. Введите название папки .

  3. Нажмите Добавить детали для данного элемента.

  4. Под надписью "Доступность" установите Показывать папку и ее содержимое только избранным группам. Отметьте группы, которым будет разрешен доступ к папке и нажмите Создать папки.

    Внимание: Если вы не видите опций под надписью "Доступность", необходимо настроить права доступа к родительской папке уровнем выше.

Превращение существующей папки в папку блока пользователей.

Чтоыб превратить существующую папку в папку блока пользователей:

  1. На линейке инструментов нажмите Ресурсы. В выпадающем списке "Действия" рядом с папкой, которую нужно превратить в папку блока пользователей, выберите Исправить детали.

  2. Под надписью "Доступность" установите Показывать папку и ее содержимое только избранным группам. Отметьте группы, которым будет разрешен доступ к папке и нажмите Обновить.

Управление правами доступа для папок блоков пользователей

Если вы создали папку блока пользователей, можно управлять правами доступа к ней, чтобы установить уровень доступа для участников сайта. Более подробные инструкции смотри в Изменение прав доступа к папке в инструменте "Ресурсы".
