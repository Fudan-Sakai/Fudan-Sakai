尋找資源項目的 URL

使用資源工具可將檔案儲存在中央位置，然後透過 URL 連結使其他人存取這些檔案。

如需尋找和共用資源中項目的 URL，請遵循下列步驟進行操作：

  1. 在功能表欄中，按一下資源。

  2. 在您想要連結的目標項目旁，從操作功能表按一下編輯詳細資料。

  3. 在「檔案詳細資料」下的「網址 (URL)」旁，高亮顯示項目的 URL。按 Ctrl-c(Windows)或 Cmd-c(Macintosh)複製 URL。

  4. 按 Ctrl-v(Windows)或 Cmd-v(Macintosh)將 URL 貼上至您想要輸入 URL 的位置。

註釋：如需使網站外部的其他人能夠看到 URL，請參考公開資源中的項目。
