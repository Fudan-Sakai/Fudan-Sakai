Unsupported characters

When naming an item such as a file, folder, discussion forum, or wiki page, avoid using either foreign characters or the unsupported characters below, as any of these characters can cause an error when you attempt to upload the file.

Important note: Even if the file appears to load correctly, it will not display if you include such symbols in the filename. It's possible that you could use one of these symbols and not receive an error while uploading; however, when others try to access the file, they will receive an error.

Character Description
\ Back slash
/ Forward slash
? Question mark
* Asterisk
" Quotation mark
: Colon
< Less than
> Greater than
# Hash mark
% Percent sign
= Equal sign
