Работа с файлами в инструменте "Ресурсы"

На этой странице:

  • Загрузка новых файлов
  • Изменение деталей файла
  • Изменение содержания файла

Внимание: Для того чтобы совершать все эти действия, вам должна быть назначена роль с соответствующими полномочиями. Для того чтобы определить, какая у вас роль, перейдите на страницу Роли участников. Основные сведения о ролях и правах доступа изложены также в Права доступа и роли: обзор.

Загрузка новыъ файлов

Внимание: Следующие инструции служат для загрузки новых файлов в инстурмент "Ресурсы". Чтобы заменить существующий файл новым, смотри Изменение содержания файла ниже.

Чтобы загрузить файлы в инструмент "Ресурсы", на линейке инстурментов нажмите Ресурсы и следуйте следующим шагам:

  1. Рядом с названием папки, в которую нужно добавить файл, в выпадающем списке Добавить нажмите Загрузить файлы.

  2. Нажмите Обзор, чтобы найти нужный файл на компьютере или в сети, затем нажмите Открыть.

    Внимание: Имена файлов, которые включают нераспознаваемые символы, не смогут быть обработаны. Список нераспознаваемых символов смотри в Нераспознаваемые символы.

  3. Рядом с надписью "Отобразить имя " введите имя файла, которое будет отображаться в инстурменте "Ресурсы". Если не нужно добавлять особое отображаемое имя, будет использовано имя файла.

    Внимание: Когда вы используете WebDAV для доступа к элементу, вы видите не отображаемое имя, а только имя файла.

  4. Чтобы добавить описание файла или другие детали, например, настройки доступа к файлу или период, на протяжени которого файл будет доступен, нажмите Добавить детали для данного элемента, это позволяет:

    • Ввести подробное описание файла в тестовом поле:

    • Выберите статус авторских прак для своего ресурса, введите информацию об авторских правах и настройке показ предупреждения об авторских правах:

      • Если выбрать Ввести информацию об авторских правах, появится текстовое поле, в которое можно ввести информацию об авторских правах.
      • Для информации об авторских правах, нажмите Больше информации .
      • Если нужно отображать информацию об авторских правах пользователям, когда она заходят на страницу с ресурсом, выберите Отображать предупреждение об авторских правах и требовать ознакомления от тех, кто пытается получить доступ. Эта опция запорашивает у участников согласие с авторскими правами для просмотра ресурса. Чтобы увидеть предупреждение, нажмите что это?.

    • Под надписью "Доступность " установаить контролько доступа к файлу и период, в который файл доступен:

      • Чтобы показать папку только участника сайта, выберите Только члены этого сайта могут видеть этот файл.. (Эта опция будет недоступна, если папка, в которую вы добавляете новую папку, имеет публичный доступ).
      • Чтобы разрешить доступ к этой папке участников других сайтов (например, для размещения приложений к элементу программы курса или другого проекта, которым вы управляете), нажмите Этот файл для публичного просмотра..
      • Чтобы открыть папку только для определенных групп, выберите Показывать файл только избранным группам. затем выберите группы, доступ которых нужно установить. Более подробную информацию о группах смотри в Управление группами.
      • Чтобы показать или скрыть ваши файлы или настроить дату начала или окончания срока доступнойти файла, выберите необходимые настройки (Эта опция отсутствует, когжа вы добавляете элемент в инструмент "Ресурсы" раздела Мой сайт) Администраторы сайта всегда могут просматривать скрытые элементы, даже когда они не видны другим участникам.

    • Чтобы добавить детальное описание, нажмите  Настраиваемые свойства . Можно ввести информацию в следующие текстовые поля:

      • Альтернативный заголовок
      • Автор
      • Автор
      • Предмет и ключевые слова
      • Дата создания
      • Дата выпуска
      • Резюме
      • Ассистент
      • Аудитория
      • Уровень образования аудитории

  5. Чтоыб загрузить более одного файла, нажмите Добавить другой файл.

    Можно добавить любой количество файлов за один раз, пока общий размер всех файлов не превысит лимита, установленного администратором системы. Этот лимит может зависеть от вашей организации, он показан внизу страницы добавления файлов.

    Внимание: Чтобы удалить файл из списка файлов для добавления, нажмите на красный крестик (X) рядом с его названием.

  6. Рядом с надписью " email Уведомление " укажите, хотите ли вы, чтобы участники сайта полчили уведомление о добвлении ресурса по электронной почте. (Эта опция не работает, если вы добавляете файл в инстурмент "Ресурсы" раздела Мой сайт.)

  7. Нажмите Загрузить файлы, чтобы закончить.

Внимание: Можно также использовать WebDAV, чтобы перетаскивать файлы со своего компьютера в папку "Ресурсы". Подробные инструкции смотри в Загрузка файлов с помощью WebDAV в операционной системе Windows XP или Vista или Загрузка файлов в инструмент "Ресурсы"в помощью WebDAV в операционной системе Mac OS X.

Добавление деталей файла

Чтоыб изменить имя файла, описание, настройки доступа или настраиваемые свойства, в выпадабщем меню Действия рядом с названием файла выберите Исправить детали. Внесите необходимые изменения и нажмите Обновить, чтобы сохранить их.

Наверх

Изменение содержания файла.

Когда вы открываете файл в инструменте "Ресурсы", например, документ Word document, компьютер создает временную копию файла и использует ее, а оигинал файла остается на сервере. Если произвести изменения в этой копии, они автоматически не сохраняются в оригинале. Если нужно изменить содержание оригинала, нужно сохранить изменения, а затем заменить оригинал сохраненной версией.

Чтобы заменить файл в инструменте "Ресурсы" с измененной копией с компьютера, В выпадающем списке Действия рядом с названием нужного файла выберите Загрузить новую версию. Нажмите Обзор, чтобы выбрать файл на компьютере и выбери Загрузить новую версию.
