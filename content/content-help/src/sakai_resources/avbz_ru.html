Работа с веб-ссылками в инструменте "Ресурсы"

Внимание: Для того чтобы совершать все эти действия, вам должна быть назначена роль с соответствующими полномочиями. Для того чтобы определить, какая у вас роль, перейдите на страницу Роли участников. Основные сведения о ролях и правах доступа изложены также в Права доступа и роли: обзор.

Добавление веб-ссылки

Чтобы добавить в качестве ресурса URL-ссылку на другой сайт:

  1. На линейке инструментов нажмите Ресурсы.

  2. Рядом с папкой, в которую нужно добавить ссылку, в выпадающем списке Добавить выберите Добавить веб-ссылки (URLs).

  3. Наберите URL для каждого ресурса в текстовом поле (Можно опустить префикс http://)

  4. Добавьте заголовок к ссылке UR. Если это не нужно, URL будет использоваться как заглавие ресурса.

  5. Чтобы добавить описание файла или другие детали, например, настройки доступа к файлу или период, на протяжени которого файл будет доступен, нажмите Добавить детали для данного элемента, это позволяет:

    • Ввести подробное описание сайта, на который установлена ссылка, в текстовом поле:

    • Под надписью "Доступность " установаить контролько доступа к файлу и период, в который файл доступен:

      • Чтобы показать ссылку только участника сайта, выберите Только члены этого сайта могут видеть эту папку и её содержимое. (Эта опция будет недоступна, если папка, в которую вы добавляете новую ссылку, имеет публичный доступ).
      • Чтобы разрешить доступ к этой ссылке участников других сайтов (например, для размещения приложений к элементу программы курса или другого проекта, которым вы управляете), нажмите Эта ссылка для публичного просмотра.
      • Чтобы открыть папку только для определенных групп, выберите Показывать папку и ее содержимое только избранным группам, затем выберите группы, доступ которых нужно установить. Более подробную информацию о группах смотри в Управление группами.
      • Чтобы показать или скрыть ваши папки или настроить дату начала или окончания срока доступнойти папки, выберите необходимые настройки (Эта опция отсутствует, когжа вы добавляете элемент в инструмент "Ресурсы" раздела Мой сайт) Администраторы сайта всегда могут просматривать скрытые элементы, даже когда они не видны другим участникам.

    • Чтобы добавить детальное описание, нажмите  Настраиваемые свойства . Можно ввести информацию в следующие текстовые поля:

      • Альтернативный заголовок
      • Автор
      • Автор
      • Предмет и ключевые слова
      • Дата создания
      • Дата выпуска
      • Резюме
      • Ассистент
      • Аудитория
      • Уровень образования аудитории

  6. Чтобы добавить другой URL, нажмите Добавить другую веб-сылку.

    Внимание: Чтобы удалить ссылку из списка ссылок для добавления, нажмите на красный крестик (X) рядом с ее названием.

  7. Рядом с надписью " email Уведомление " укажите, хотите ли вы, чтобы участники сайта полчили уведомление о добвлении ресурса по электронной почте. (Эта опция не работает, если вы добавляете файл в инстурмент "Ресурсы" раздела Мой сайт.)

    Внимание: Уведомление по электронной почте будет включать название и описание сайта (если они были введены) и URL.

  8. Нажмите Добавить веб-ссылки, чтобы завершить добавление.

Изменение деталей веб-ссылки

Чтоыб изменить имя ссылки, описание, настройки доступа или настраиваемые свойства, в выпадабщем меню Действия рядом с названием ссылки выберите Исправить детали. Внесите необходимые изменения и нажмите Обновить, чтобы сохранить их.

Обновление веб-ссылки

Чтобы изменить URL веб-ссылки, в выпадающем списке Добавить рядом с названием ссылки выберите Исправить содержимое. Сделайте изменения и нажмите Продолжить.
