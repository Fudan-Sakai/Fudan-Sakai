操作資源中的網路連結

註釋：如果要完成本頁中的操作，您必需擁有具有必要權限的角色。如需決定自己的角色，請遵循參與者角色中的說明進行操作。如需瞭解關於角色和權限的基礎知識，請參考權限和角色：概述。

新增網路連結

如需新增連結至網站的 URL 資源項目，請遵循下列步驟進行操作：

  1. 在網站功能表欄中，按一下資源。

  2. 在您想要新增 URL 的目標資料夾旁，用滑鼠滑過新增，並按一下新增網路連結(URL)。

  3. 在文字方塊中輸入 URL 新增各個資源。（也省略 http:// 字首）。

  4. 新增 URL 連結資源的名稱。如果沒有新增名稱，那麼將把 URL 作為資源的名稱。

  5. 如需新增描述，以及控制存取和可見性，按一下新增此項目詳細資料。

    • 在「描述」旁輸入您所連結的目標網站的描述性說明。

    • 在「可用性和存取」欄位下，設定可存取 URL 的使用者權限，以及可存取 URL 的時間，如下所示：

      • 如需僅向網站成員顯示檔案，選擇僅本網站成員可查看此檔案。
      • 如需與您所屬的另一個網站共用檔案（例如，作為您正負責的另一項課程或項目的課程大綱附件），選擇公開顯示此檔案。（如果您正在新增檔案的目標資料夾目前已經公開，那麼本選項不可用。）
      • 如需僅向所選群組顯示檔案，選擇僅向所選群組顯示此檔案，然後選擇有權存取的群組。如需瞭解關於群組的更多資訊，請參考管理群組。
      • 如需顯示或隱藏檔案，或為檔案設定開始和/或結束日期，按一下適當的選項。（當您向我的工作區內的資源新增項目時，此選項不顯示。)網點管理員始終有權檢視隱藏的項目，即使向其他使用者隱藏檔案時。

    • 如需新增詳細的描述，請按一下選擇性的屬性。然後您可以用以下欄位來新增資料：

      • 代用標題
      • 建立人
      • 公開人
      • 主題與關鍵字
      • 建立日期
      • 發行日期
      • 摘要
      • 貢獻者
      • 讀者
      • 讀者的教育水平

  6. 如需再建立一個 URL，按一下新增另一個網路連結。

    註釋：如需從資料夾清單中刪除 URL，按一下該 URL 旁的紅色X標誌。

  7. 在「電子郵件通知」旁，指定是否要在發佈資源時，透過電子郵件自動通知網站成員。（當您向我的工作區內的資源新增項目時，此選項不顯示。)

    註釋：通知郵件資訊將包括網站名稱和描述（如果您輸入了描述）和 URL。

  8. 按一下立即新增網路連結完成操作。

編輯網路連結詳細資料

如需編輯網路連結的名稱、描述或可用性和存取設定，用滑鼠滑過操作功能表，然後按一下編輯詳細資料。根據需要進行更改，然後按一下更新儲存更改。

更新網路連結

如需更改網路連結的 URL，用滑鼠滑過操作功能表，然後按一下編輯內容。根據需要進行更改，然後按一下繼續。
