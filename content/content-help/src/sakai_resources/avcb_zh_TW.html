新增 HTML 頁面

如需在資源區新增和顯示 HTML 頁面，請遵循下列步驟進行操作：

  1. 在網站功能表欄中，按一下資源。

  2. 在您想要新增 HTML 頁面的目標資料夾旁，用滑鼠滑過新增，並按一下建立 HTML 頁面。

  3. 使用 WYSIWYG 編輯器建立頁面，然後按一下繼續。

    註釋：在 WYSIWYG 編輯器中，您可以插入已編排格式的 Microsoft Word 文件。如需進行上述操作，按一下從 Word 貼上圖示，將遊標置於隨後出現的「從 Word 貼上」文字方塊中，然後在 Windows 中按Ctrl-v，或在 Macintosh 中按 Cmd-v，將複製的文字貼上到文字方塊中。使用 Ctrl-v（或 Cmd-v）進行貼上是保留原 Word 文件格式的最佳方法。按一下確定將文字新增到您想要編輯的項目中。

  4. 在文字欄位中輸入名稱和可選描述。

  5. 選擇檔案的版權狀態，新增版權資訊，並顯示版權警告：

    • 如果您選擇使用下方的版權，將顯示文字方塊，您可以在該文字方塊中新增關於版權的資訊。
    • 如需瞭解關於版權和公平使用的更多資訊，按一下更多資訊。
    • 如果您想要在使用者存取檔時顯示相關條款，選擇顯示版權警示，以及需要在他人存取檔案的時候確認版權狀態。該選項將要求使用者同意版權條款才能瀏覽資源。如需檢視警告，按一下這是什麼？。您不能編輯該警告。

  6. 在「可用性和存取」欄位下，設定可存取頁面的使用者權限，以及可存取頁面的時間，如下所示：

    • 如需僅向網站成員顯示檔案，選擇僅本網站成員可查看此檔案。
    • 如需與您所屬的另一個網站共用檔案（例如，作為您正負責的另一項課程或項目的課程大綱附件），選擇公開顯示此檔案。（如果您正在新增檔案的目標資料夾目前已經公開，那麼本選項不可用。）
    • 如需僅向所選群組顯示檔案，選擇僅向所選群組顯示此檔案，然後選擇有權存取的群組。如需瞭解關於群組的更多資訊，請參考管理群組。
    • 如需顯示或隱藏檔案，或為檔案設定開始和/或結束日期，按一下適當的選項。（當您向我的工作區內的資源新增項目時，此選項不顯示。)網點管理員始終有權檢視隱藏的項目，即使向其他使用者隱藏檔案時。

  7. 如需新增詳細的描述，請按一下選擇性的屬性。然後您可以用以下欄位來新增資料：

    • 代用標題
    • 建立人
    • 公開人
    • 主題與關鍵字
    • 建立日期
    • 發行日期
    • 摘要
    • 貢獻者
    • 讀者
    • 讀者的教育水平

  8. 在「電子郵件通知」旁，指定是否要在發佈資源時，透過電子郵件自動通知網站成員。（當您向我的工作區內的資源新增項目時，此選項不顯示。)

  9. 按一下完成.

編輯 HTML 頁面詳細資料

如需編輯 HTML 頁面的名稱、描述、版權狀態或可用性和存取設定，用滑鼠滑過操作功能表，然後按一下編輯詳細資料。根據需要進行更改，然後按一下更新儲存更改。

更新 HTML 頁面

如需更新 HTML 頁面的內容，用滑鼠滑過操作功能表，然後按一下編輯內容。根據需要進行更改，然後按一下繼續。
