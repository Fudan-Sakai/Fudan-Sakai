Об инструменте "Библиография"

При помощи инструмента "Библиография" вы можете осуществлять поиск по ресурсам библиотек (например, WorldCat, Academic Search [EBSCO] и JSTOR) и создавать и использовать ссылки на литературу и постоянные ссылки на источники в Интернете.

Вы можете осуществлять поиск ссылок на литературу по отдельным базам данных, таким как ProQuest и JSTOR, или по группе тематических баз данных. Ссылка на литературу обычно включает ссылку для доступа к полному тексту или информацию о физическом нахождении исчтоника. Вы можете использовать списки литературы в других инструментах (например, "Задания", "Обсуждения", "Календарь") на вашем сайте и экспортировать в библиографические приложения, такие как EndNote и Reference Manager.

Наверх

Добавление списка литературы

Для того чтобы добавить новый список литературы:

  1. На линейке инстурментов вашего сайта нажмите на Ресурсы. Рядом с названием папка, в которую вы хотите добавить список литературы, из выпадающего списка Добавить выберите Добавить список литературы.

  2. Теперь есть несколько способов добавить ссылку на источник в ваш список литературы:

    • Поиск по ресурсам библиотеки: ищите ссылки на литературу по библиотечному каталогу, журнальным базам и другим ресурсам.
    • Поиск в Google Scholar: ищите ссылки на источники во всемирной базе научной литературы и включайте их в список литературы.
    • Создать новую ссылку: добавьте ссылку на источник вручную.
    • Импортировать ссылки: bмпортируйте библиографию в формате RIS из EndNote, RefWorks или других программных приложений.

  3. После добавления ссылок на источники в список литературы назовите и сохраните его.

Поиск по ресурсам библиотеки

Внимание: Поиск по ресурсам библиотеки будет доступен только в том случае, если у вашей организации есть доступ к инструментам метапоиска по библиотекам (например, Sirsi SingleSearch, Ex Libris MetaLib).

  • Советы по использованию обычного поиска

    • Введите слова для поиска в текстовое поле и нажмите на кнопку Поиск.
    • Для того чтобы искать точную фразу, поместите ее в кавычки (например: "быть или не быть").
    • При поиске игнорируются служебные и общие слова, если они не являются частью фразы в кавычках. Например: а, и, или, в, на, за, из, для, по, я, ты, он, это, не, где, когда, что.
    • При поиске используется логический оператор И (то есть по запросу елки палки будут найдены результаты, содержащие одновременно "елки" И "палки"). Другие логические операторы, такие как ИЛИ и НЕ, в настоящий момент недоступны.
    • При поиске игнорируется пунктуация, если она не являются частью фразы в кавычках. Например: ? ! ; , ' : + - @ # $ % ^ & * () [] {}
    • При поиске игнорируются некоторые пробельные символы, такие как знак табуляции, лишние пробелы, знак перевода строки.

  • Соверты по использованию расширенного поиска

    • Для расширенного поиска по ресурсам библиотек нажмите на ссылку Расширенный поиск.
    • В появившемся списке выберите поля, по которым вы хотите выполнять поиск: Ключевые слова, Автор, Заглавие, Дисциплина или Год.
    • Введите слова для поиска в тестовом поле. Поиск будет выполнен по всем критерия (то есть при помощи логического оператора И), как показано выше. Для того чтобы искать точную фразу, поместите ее в кавычки, как показано выше.
    • Для того чтобы полностью очистить текстовые поля поиска, нажмите на Очистить форму.

  • Выбор категории поиска

    • По умолчанию установлена категория поиска Быстрый поиск по Academic Search (EBSCO) и ERIC (CSA).
    • Для установки другой категории поиска выберите ее из выпающего списка. В случае необходимости вы можете выбрать подкатегорию в списке баз данных.

  • Выбор баз данных

    Внимание: Описания баз данных по умолчанию скрыты. Вы можете осуществлять поиск только по восьми база данных одновременно.

    • Для просмотра описаний всех баз данных нажмите на Показать описания.
    • Для того чтобы скрыть описания всех баз данных, нажмите на Скрыть описания.
    • Для того чтобы просмотреть описание одной базу данных или скрыть его, нажмите на переключатель.
    • Вы можете осуществлять поиск по рекомендованным базам данных (уже выбранным) или изменить этот выбор. Для того чтобы выбрать базу данных или отменить ее выбор, нажмите на флажок для выбора около ее названия.

  • Выполнение поиска

    Когда вы ввели слова для поиска, выбрали категорию и базы данных, нажмите на кнопку Поиск внизу страницы.

  • Отмена поиска

    Во время выполнения поиска кнопка Поиск заменяется на кнопку Отмена. Для того чтобы остановить поиск, нажмите на нее.

Поиск в Google Scholar

Для того чтобы добавить ссылку на источник в список литературы при помощи Google Scholar, на странице "Добавить ссылку на лиературу", нажмите на кнопку Поиск в Google Scholar.

К сожалению, в настоящий момент Google Scholar недоступен на русском языке. Если вы не владеете английским языком, то знаете какой-либо другой иностранный язык, проверьте существует ли интерфейс на этом языке. Для этого нажмите на ссылку Scholar Preferences и там в пункте Interface Language в выпающем списке выберите нужный язык.

  • Вставка ссылок на источник в список литературы

    1. Выполните поиск в Google Scholar. Для вызова справки нажмите на Scholar Help около текстового поля для поиска.
    2. Выберите один из результатов поиска, на который вы хотите сослаться.
    3. Под заглавием источника вы увидите несколько ссылок. Для того чтобы импортирвоать ссылку на источник, нажмите Import into Sakai.
    4. Для того чтобы продолжить работу в Google Scholar, нажмите на кнопку Назад в Google Scholar. По завершении нажмите на кнопку Закрыть окно.
    5. Для того чтобы отредактировать список литературы, нажмите на кнопку Редактировать список литературы.
    6. Для того чтобы дат название вашему списку литературы и сохранить его, нажмите на кнопку Завершить.

Создание новой ссылки на источник

Для того чтобы добавить ссылку на источник вручную, на странице "Добавить ссылки на литературу" нажмите на кнопку Создать новую ссылку на литературу.

  • Выбор типа источника

    Выберите тип источника из выпадающего списка Статья из журнала, Книга, Часть книги, Отчет или Неизвестный формат.

  • Создание дополнительных полей

    Для того чтобы создать дополнительное поле для второго и последующих авторов, комментарией, дисциплины, ссылки в Интернете или сведений об авторских правах, нажмите под соответствующим полем Добавить еще.

  • Сохранение новой ссылки на источник

    Для того чтобы сохранить новую ссылку на источник, нажмите на кнопку Сохранить внизу страницы.

  • Отмена создания новой ссылки на источник

    Для того чтобы отменить создание новой ссылки на источник, нажмите на кнопку Отмена внизу страницы.

Импорт ссылок на литературу в формате RIS

  1. На странице добавления новой ссылки на источник нажмите на кнопку Импортировать ссылки на литературу.

  2. Рядом с надписью "Импортировать файл" укажите путь к файлу, который вы хотите импортировать, или нажмите Обзор...В , чтобы выбрать файл.

  3. Когда имя файла появится в поле, нажмите на кнопу Импортировать. Ссылки на литературу в файле будут включены в ваш список литературы.

  4. Другой способ: вы можете скопировать текст в формате RIS и вставить его в поле "Введите сюда данные в формате RIS", после чего нажмите на кнопку Импортировать.

  5. По завершении нажмите на кнопку Назад. Для того чтобы отменить импорт, нажмите на кнопку Отмена.

Добавление ссылки на литературу из результатов поиска в список литературы

Добавление ссылки на источник в ваш список литературы

Для того чтобы добавить ссылку на источник в ваш список литературы, нажмите на Добавить рядом с нужным результатом. Если вы успешно добавили ссылку на источник в ваш список литературы, то Количество источников возрастет на единицу, ссылка на источник будет выделена цветом, а Добавить изменится на Удалить.

Удаление ссылки на источник из вашего списка литературы

Для того чтобы удалить ссылку на источник из вашего списка литературы, нажмите Удалить рядом с нужным результатом. Если вы успешно удалили ссылку на источник из вашего списка литературы, то Количество источников снизится на единицу, ссылка на источник не будет больше выделена цветом, а Удалить изменится на Добавить.

Просмотр аннотаций

По умолчанию отображаются только заглавия ссылок на литературу. Для того чтобы просмотреть аннотации всех источников,нажмите на Показать аннотацию; для того чтобы скрыть аннотацию и информацию о дисциплинах, нажмите на Краткий вид.

Для того чтобы показать или скрыть аннотацию у отдельных результатов, нажмите на переключатель (маленькая черная стрелка между Добавить/Удалить и заглавием).

Отображение результатов поиска

По умолчанию на странице отображается по 10 результатов. Вы можете выбрать из выпадающего списка 10, 20, 30, 40 или 50 результатов на странице.

Навигация по страницам

  • Для того чтобы перейти на следующую страницу результатов поиска, нажмите > .

  • Для того чтобы перейти на предыдущую страницу результатов поиска, нажмите < .

  • Для того чтобы перейти на первую страницу результатов поиска, нажмите << .

Доступ к электронной или печатной версии

Для того чтобы получить доступ к полному тексту источника, нажмите на заглавие ссылки на литераутуру или на Получить! под заглавием. Если полный текст недоступен онлайн, то его, возможно, он есть в печатном виде.

Выполнение нового поиска

Для того чтобы вернуть форму для поиска к состоянию по умолчанию, нажмите на Новый поиск внизу страницы. Теперь вы сможете искать по другому набору баз данных.

Добавление ссылки на источник в текстовом редакторе

Внимание: Возможность добавления ссылок на источник прямо в текстовом редактора будет доступна только в том случае, если у вашей организации есть доступ к инструментам метапоиска по библиотекам (например, Sirsi SingleSearch, Ex Libris MetaLib).

  1. В инструменте "Ресурсы" выберите из выпадающего списка Add пункт Содать HTML-страницу.
  2. Нажмите на значок книга (рядом со значком якорь) на панели инструментов.
  3. Откроется поисковое окно. Подробнее о поиске в этом окне см. Поиск по ресурсам библиотеки выше.
  4. После того как вы нажмете на Добавить, ссылка на источник будет вставлена прямо в текстовый редактор.

Просмотр вашего списка литературы

Доступ к электронной или печатной версии

Для того чтобы получить доступ к полному тексту источника, нажмите на заглавие ссылки на литераутуру или на Получить! под заглавием. Если полный текст недоступен онлайн, то его, возможно, он есть в печатном виде.

Просмотр ссылок на источник

Для того чтобы окрыть ссылку на источник в режиме просмотра, нажмите на просмотр под заглавием.

Изменение ссылки на источник

Для того чтобы внести изменения в ссылку на источник, нажмите на редактировать под заглавием.

Выбор ссылок на источник

  • Для того чтобы выбрать все ссылки на литературу, нажмите на Выбрать все. Отмечены флажками для выбора и выделены цветом будут все ссылки на литературу.

  • Для отмены выбора нажмите на Отменить выбор. Флажки для выбора будут сняты и выделение цветом отменено у всех ссылок на литературу.

  • Для выбора отдельных ссылок на литературу отметьте их флажками для выбора, после чего они будут выделены цветом.

  • Для отмены выбора отдельных ссылок на литературу снимите флажки для выбора, после чего выделение цветом исчезнет.

Навигация по страницам

  • Для того чтобы перейти на следующую страницу списка литературы, нажмите > .

  • Для того чтобы перейти на предыдущую страницу списка литературы, нажмите < .

  • Для того чтобы перейти на первую страницу списка литературы, нажмите << .

  • Для того чтобы перейти на последнюю страницу списка литературы, нажмите >>В .

Экспорт ссылок на литературу

  • Для того чтобы экспортировать отдельные ссылки на литературу, отметьте их и из выпадающего списка Действия выберите Экспортировать отмеченные.

  • Для того чтобы экспортировать все ссылки на источники вашего списка литературы, из выпадающего списка Действия выберите Экспортировать весь список.

Экспортированные ссылки на литературу вы сможете скачать в виде файла формата RIS, который вы затем сможете открыть в библиографическом приложении, например EndNote, ProCite или Reference Manager.

Удаление ссылок на литературу

  • Для того чтобы удалить из списка отдельные ссылки на литературу, отметьте их и из выпадающего списка Действия выберите Удалить отмеченные. Другой способ: в можете нажать на Удалить под каждой ссылкой на литературу, которые вы хотите удалить.

  • Для того чтобы удалить все ссылки на источники из вашего списка литературы, из выпадающего списка Действия выберите Удалить весь список.

Добавление ссылок на источники в список литературы

  • Для того чтобы вернуться к результатам поиска, нажмите на кнопку Назад к результатам поиска.

  • Для того чтобы начать новый поиск или создать новую ссылку на источник, нажмите на кнопку Добавить ссылки на литературу.

Отмена изменений

Для того чтобы отменить внесенные изменения, нажмите на кнопку Отмена.

Сохранение вашего списка литературы

Для того чтобы сохранить ваш список литературы в "Ресурсы":

  1. Нажмите на Завершить.

  2. Введите название и дополнительную информацию в текстовые поля.

  3. Выберите соответствующие настройки в пункте "Доступ".

  4. Для того чтобы добавить полное описание, нажмите на Дополнительные настройки.

  5. Для того чтобы отправить участникам сайта извещение по электронной почте, выберите это в пункте "Извещение по электронной почте".

  6. Внизу страницы нажмите на кнопку Завершить.

Изменение параметров списка литературы

Для того чтобы просмотреть и изменить параметры вашего списка литературы, из выпадающего списка Действия выберите Изменить параментры.

  1. Вы можете изменить название, описание, настройки доступа и т.д.

  2. Если вы хотите изменить тип файла, нажмите на Изменить тип файлаи из выпадающего списка выберите нужный тип.

  3. Для того чтобы сохранить настройки, нажмите на Обновить внизу страницы.

Ваш список литературы будет доступен в инструменте "Ресурсы". Ссылки на литературу будут выстроены по заглавию в алфавитном порядке.
