關於引文清單

引文清單功能可讓您搜尋資源庫源（例如， WorldCat，學術搜尋[ EBSCO ] ， JSTOR ） ，以及建立和管理引文並連結文章和其他在線的項目。

您可以從個別的資料庫中搜尋和取出引文，如Proquest和JSTOR ，以及從特定主題的數據庫群組，建立引文清單。引文包含連結用來取出全文或資訊的實際位置。您可以在您的工作區內附加這些引文清單的副本至其他項目（例如，作業，討論的發佈，行事曆項目），而且還可以匯出引文書目管理工具，如 EndNote 和 Reference Manager。

新增引文清單

如需新建引文清單，請遵循下列步驟進行操作：

  1. 在網站功能表欄中，按一下資源。 在您想要新增引文清單的目標資料夾旁，用滑鼠滑過新增，並按一下新增引文清單 。

  2. 您現在有多個可以新增引文清單的選項：

    • 搜尋資源庫：搜尋目錄庫，文章資料庫，和其他資源。
    • 搜尋谷歌學術：搜尋世界各地的學術資源資料庫並儲存搜尋結果到此引文清單。
    • 建立新引文：輸入資料來新增引文。
    • 匯入引文從 EndNote，RefWorks 或其他引文管理工具中匯入RIS格式的引文。

  3. 一旦您加入引文到您的清單，請為您的清單命名並儲存。

搜尋資源庫

註釋：搜尋資源庫選項僅提供給具有圖書館 metasearch 工具﹙例如，Sirsi SingleSearch, Ex Libris MetaLib﹚存取的機構。

  • 基本搜尋秘訣

    • 輸入您的搜尋字串到文字欄位，然後按一下搜尋
    • 搜尋詞組用引號包圍詞組（例如， lyrics for "a day in the life"  ） 。
    • 搜尋忽視常見的字詞，除非他們是在引號內詞組的一部分。常用字詞包括：a, and, be, for, from, has, i, in, is, it, of, on, to, the, not, and or。
    • 使用布林 (Boolean) 搜尋和 (and) （即搜尋貓狗返回的資源含有兩種「貓」和 (and)「狗」 ） 。其他布林操作符，如或 (or) 和不 (not)此時無法使用。
    • 搜尋忽視無關的標點符號，除非它是在引號內詞組的一部分。忽略標點符號包括：? ! ; , ' : + - @ # $ % ^ & * () [] {}
    • 搜尋忽略空白字元（例如，tabs，多餘的空格，被迫(newline)換行字元） 。

  • 進階搜尋秘訣

    • 如需利用進階搜尋表單來搜尋資料庫，請按一下進階搜尋 。
    • 使用下拉式清單選擇您要搜尋的欄位： 關鍵字，作者， 標題 ，主題 ，或年份。
    • 在文字欄位中輸入搜尋字串 。搜尋將尋找符合全部搜尋準則的記錄（即布林和 ） ，如上所述。如需搜尋精確的詞組，請用引號包圍詞組，如上所述。
    • 如需清除搜尋欄位中的文字，請按一下清除表單。

  • 選擇搜尋類別

    • 預設搜尋類別是快速搜尋(Quick Search) ， 學術搜尋 (Academic Search) (EBSCO) ERIC (CSA)。
    • 如需使用不同的搜尋類別，請用下拉式表單選擇一個可用選用的類別。根據需要，請用下拉式清單選擇一個分類，直到您達到該預的建議資料庫清單。

  • 選擇資料庫

    註釋：資料庫的描述的預設是隱藏的，您可以同一時間搜尋不超過8個資料庫。

    • 如需檢視清單上每個資料庫描述，請按一下顯示描述 。
    • 如需隱藏清單上每個資料庫描述，請按一下隱藏描述。
    • 如需檢視或隱藏個別資料庫描述，請按一下切換按鈕。
    • 您可以對每一類別進行搜尋預選建議的資料庫，或改變的選擇。選擇或取消選擇資料庫，按一下資料庫標題旁邊的核取方塊。

  • 進行搜索

    一旦您滿意您的搜索條件，搜索類，和數據庫Search單擊搜索在頁面底部。

  • 取消您的搜尋

    在搜尋過程中， 搜尋按鈕變成取消按鈕。如需停止搜尋資源庫，請按一下取消。

搜尋谷歌學術

如需使用谷歌學術來新增引文至您的清單，請從新增引文頁面，按一下搜尋谷歌學術 。

  • 匯入引文至您的清單

    1. 使用谷歌學術搜尋界面進行搜尋。如需幫助，請按一下學術說明旁邊的搜尋方塊。
    2. 在搜尋結果中，尋找您想要引用的文章。
    3. 您會看到一些連結在文章描述下方。匯入引文至您的引文清單，請按一下匯入到Sakai (Import into Sakai) 。
    4. 如需繼續從谷歌學術匯入引文，按一下返回谷歌學術 (Back to Google Scholar)當您完成後，按一下關閉視窗 (Close Window) 。
    5. 如需編輯您的引文清單，請按一下編輯引文清單
    6. 如需為您的引文清單標題和儲存，請按一下完成完成。

建立新引文

如需使用輸入引文資訊方式來新增引文，請從新增引文頁面中按一下建立新引文。

  • 選擇引文類型

    如需選擇引文類型，請使用下拉式清單選擇期刊文章 ， 書籍 ， 書籍章節 ， 報告或不明的 。

  • 新增其他欄位

    如需建立其他欄位，以容納一個以上的作者，註釋，主題，版權，或連結，按一下相應的新增另一個連結。

  • 儲存新引文

    如需儲存新引文，請在頁面底部按一下儲存引文。

  • 取消新引文

    如需取消新引文，請在頁面底部按一下取消引文。

匯入RIS格式引文

  1. 從新增引文螢中，選擇匯入引文 。

  2. 在「匯入檔案：」旁 ，輸入您要匯入的檔案路徑，或按一下瀏覽…來瀏覽到它。

  3. 當檔案名出現在方塊中，請按一下 匯入。檔案中的引文會出現在引文清單中。

  4. 另外，您也可以剪下並貼上 RIS 格式的文字到下面「從以下文字匯入RIS格式的資料：」的文字方塊中 ，並按一下匯。

  5. 當您完成後，按一下返回新增引文 。如需取消匯入，請按一下取消引文清單

從搜尋結果中新增引文至引文清單

新增引文至您的引文清單

如需新增引文至您的新引文清單中，請在所需的引文旁按一下新增。一旦您已經成功地添加了一個引文到您的清單時，引文數目會增加一個，引文亦會高亮顯示。新增會變為刪除。

從您的引文清單中刪除引文

如需從您的新引文清單中刪除引文，請在所需的引文旁按一下刪除。一旦您已經成功地從您的清單中刪除了引文，引文數目會減少一個，引文不會再高亮顯示了。刪除會變為新增 。

檢視引文摘要

預設情況下，只有引文標題顯示出來。如需在頁面上為每個引文檢視摘要，請按一下摘要檢視；如需在頁面上為每一個引文隱藏摘要和主題資訊，請按一下標題檢視 。

如需為個別的引文在摘要和標題檢視之間進行切換，請按一下切換按鈕（在新增／刪除和引文標題之間的小黑色箭頭） 。

顯示搜尋結果

默認情況下，引用的10個每頁顯示。使用下拉列表中選擇每頁10、 20、30、 40、或 50引文。

在搜尋結果中翻頁

  • 如需到下一頁的搜尋結果，請按一下>  。

  • 如需返回上一頁，請按一下<  。

  • 如需到清單的首頁，請按一下<<  。

存取電子或列印內容

如需存取資源的全文，請按一下其引文標題，或在標題底部按一下取出它 (Get It!)。如果全文不可以在網上查閱，它或許可以列印。

開始新的搜尋

如需重設搜尋表單，請在頁面底部按一下新搜尋。然後您就可以搜尋一組不同的資料庫。

使用 WYSIWYG 編輯器新增引文

註釋：此從 WYSIWYG 編輯器中新增引文的選項僅提供給擁有存取圖書館 metasearch 工具（例如， Sirsi SingleSearch ，Ex Libris MetaLib ）的機構。。

  1. 在資源中，請按一下新增和選擇建立 HTML 頁面.。
  2. 請在工具欄上按一下書籍圖示（在錨點圖示旁）。
  3. 搜尋視窗將打開。關於怎樣使用搜尋視窗的指示，請查考上述搜尋資源庫。
  4. 當您按一下新增從搜尋結果頁面中新增引文時，引文將直接插入到 WYSIWYG 編輯器的方塊中。

檢視您的引文清單

存取電子或列印內容

如需存取資源的全文，請按一下其引文標題，或在標題底部按一下取出它 (Get It!)。如果全文不可以在網上查閱，它或許可以列印。

檢視引文

如需存取唯讀引文檢視，請在引文底部按一下檢視引文。

修改引文

如需修改引文詳細資料，請在引文底部按一下編輯。

選擇引文

  • 如需選擇全部引文，請按一下選擇全部。打勾標記將出現在核取方塊中和您清單上的全部引文將被高亮顯示。

  • 如需取消全部選取的引文，請按一下不選取。打勾標記將不再出現在在核取方塊中和您清單上將沒有引文被高亮顯示。

  • 如需選擇個別引文，在核取方塊中打勾；該引文將被高亮顯示。

  • 如需取消個別選取的引文，取消其核取方塊的打勾；該引文將不再被高亮顯示。

在您的引文清單中翻頁

  • 如需到下一頁的引文清單，請按一下>  。

  • 如需返回上一頁，請按一下<  。

  • 如需到清單的首頁，請按一下<<  。

  • 如需到清單的尾頁，請按一下>>  。

匯出引文

  • 如需從您的清單中匯出個別引文，選取要匯出的所需引文，並從引文操作下拉式清單中，選擇匯出已選取的 。

  • 如需從您的清單中匯出全部引文，從引文操作下拉式清單中，選擇匯出整個清單。

匯出引文將提示您下載一個 RIS 格式的簡單文字檔案，您可以匯入參照管理應用程式，如 EndNote， ProCite，和 Reference Manager。

刪除引文

  • 如需從您的清單中刪除個別的引文，選擇要刪除的引文，並從操作下拉式清單中，選擇刪除已選取的 。另外，您也可以在每一個個別的引文底部按一下刪除。

  • 如需從您的清單中刪除全部引文，從操作下拉式清單中，選擇刪除整個清單。

新增更多引文到您的清單

  • 如需返回您的搜尋結果中，請按一下返回搜尋結果 。

  • 如需開始新新的搜尋，或建立一個新的引文，請按一下新增引文到清單 。

取消您的引文清單

如需取消您的引文清單，按一下取消引文清單

儲存您的引文清單

如需儲存的清單到您的資源工具的工作區：

  1. 按一下完成。

  2. 在提供的文字方塊中輸入名稱和可選的描述。

  3. 在「可用性和存取」下，選擇合適的設定。

  4. 如需新增詳細資料描述，請按一下選擇性的屬性.

  5. 如需通知網站參與者，請使用下拉式功能表旁的「電子郵件通知」 。

  6. 在底部，按一下完成。
返回頂部

編輯您的引文清單詳細資料

如需檢視和更改您的引文清單中的詳細資料，從操作下拉式功能表中，選擇編輯詳細資料。

  1. 您可以更改名稱，描述，可用性，存取設定，或者引文清單的選擇性的屬性。

  2. 如果您想更改檔案類型，請按一下更改檔案類型，並使用下拉式功能表選擇相應的檔案類型。

  3. 如需儲存您的設定， 請在頁面的底部按一下更新。

您的引用清單將顯示在您的資源工具。引文將按引文標題字母順序排序來顯示。
