Удаление элемента в инструменте "Ресурсы"

Чтобы удалить элемент из инструмента "Ресурсы":

  1. Чтобы удалить один элемент, в выпадающем меня Действия выберите Удалить. Чтобы удалить несколько элементов за один раз, поставьте флажки напротив названи йэлементов, затем вверху страницы нажмите Удалить.

  2. Чтобы подтвердить удаление элементов, нажмите Удаить. Для отмены удаления, нажмите Отмена.

Внмание: Нельзя удалить папку, в которой содержатся элементы. Сначала необходиом убрать из нее содержимое (то есть удалить или переместить).
