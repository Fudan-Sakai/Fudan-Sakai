使用成績冊檢視和匯出學生成績

註釋：如果要完成本頁中的操作，您必需擁有具有必要權限的角色。如需決定自己的角色，請遵循參與者角色中的說明進行操作。如需瞭解關於角色和權限的基礎知識，請參考權限和角色：概述。

檢視和匯出學生成績

如需從成績冊檢視和匯出學生成績：

  1. 在網站功能表欄中，按一下成績冊。

  2. 在頂部按一下全部成績。

  3. 您將看到您有權為其打分的學生清單，他們全部的得分以及累計得分。可根據姓名或 ID 搜尋某個特定學生的記錄。如果您的課程規定了您有權為哪些班別或群組的成員打分，那麼您就可以過濾花名冊，以一次性顯示那些班別或群組。如果您的課程尚未註冊任何學生，那麼您將看到「無學生可顯示」資訊。

    使用「檢視」旁的下拉式清單根據班別、群組或類別瀏覽花名冊。

  4. 如需以 .csv 格式匯出成績，按一下匯出成績冊（位於頁面右上角）。

    註釋：將匯出全部可見的班級資料。如果在匯出前已過濾花名冊或按欄排序花名冊，那麼輸出資料將顯示這一結果。

  5. 匯出欄位以及各欄標題分別為「學生姓名」、「學生 ID」以及成績冊項目的名稱。將匯出學生的全名和成績冊項目完整的名稱。成績冊項目分數值僅匯出兩位小數點。得分將顯示在成績冊項目名稱欄中。

  6. 成績檢視和匯出完成後，按一下成績冊項目返回成績冊主功能表。
