新建／新增、編輯或刪除成績冊項目

在本頁中，您將瞭解：

  • 新建/新增成績冊項目
  • 編輯現有成績冊項目
  • 刪除成績冊項目

註釋：如果要完成本頁中的操作，您必需擁有具有必要權限的角色。如需決定自己的角色，請遵循參與者角色中的說明進行操作。如需瞭解關於角色和權限的基礎知識，請參考權限和角色：概述。

新建/新增成績冊項目

如需新建/新增成績冊項目：

  1. 在功能表欄中，按一下成績冊，然後按一下新增成績冊項目。

  2. 在新增成績冊項目螢幕中，填寫下列資訊：

    • 標題：全部成績冊項目都必須要有標題。標題最長為 255 個字元。
    • 成績冊項目分數值：必須為全部成績冊項目指定一個分數值。輸入一個大於0的數值。可輸入小數點但僅限於兩位小數點。
    • 截止日期：可指定成績冊項目的截止日期，這是可選項。可手動輸入截止日期（例如 09/28/07）或按一下行事曆圖示
      從彈出行事曆中選擇日期。
    • 類別：如果您必須建立類別，您可使用下拉式清單指定一個項目給類別。如需瞭解更多資訊，請參考使用成績冊中的類別和權重成績。

  3. 如果您想允許學生查看此項目的成績，請在向學生發佈此項目和在課程成績計算中包括此項目的核取方塊中打勾。您可稍後更改此選擇。

    註釋：如需此項目顯示給學生，那麼成績冊設定必須選取向學生發佈成績冊項目。如需課程成績顯示給學生，那麼課程成績選項必須被選中立即向學生顯示課程成績。如需瞭解更多指示，請參考使學生可查閱成績。

    如您不想包括此項目在課和成績計算中，在課程成績計算中包括此項目旁取消勾選核取方塊。您可稍後更改此選項。

    註釋：如果項目不包括在課程成績計算中，那麼學生的個別成績會出現在括號中（）。成績底下的圖例會解釋「括號內的成績不包括在課程成績計算中」。

  4. 如需新建項目，按一下新增項目。您將看到一條訊息，提示該項目已成功新增。或者按一下取消來放棄該項目，您會返回到成績冊項目頁面。

編輯現有成績冊項目

如需編輯現有成績冊項目：

  1. 從功能表欄中，按一下成績冊。

  2. 在成績冊項目清單中，按一下您想要修改的項目標題旁的編輯。

  3. 更改成績冊項目的設定後按一下儲存更改。您將看到一條訊息，提示該項目已成功更新。或者按一下取消來放棄該更改。

返回頂部

刪除成績冊項目

如需刪除成績冊項目：

  1. 從功能表欄中，按一下成績冊。

  2. 在成績冊項目清單中，按一下您想要刪除的項目的標題。

  3. 在隨後開啟的項目摘要頁面中，按一下「選項」旁的從成績冊中刪除成績冊項目。

  4. 在刪除成績冊項目確認頁中，按一下將本成績冊項目以及全部相關的分數從成績冊中刪除旁的核取方塊。

  5. 如需從成績冊刪除項目，按一下刪除。您將看到一條訊息，提示該項目已成功刪除。或者，按一下取消退出頁面，保留該項目。
