Сортировка таблиц в "Зачётной книжке"

Внимание: Для того чтобы совершать все эти действия, вам должна быть назначена роль с соответствующими полномочиями. Для того чтобы определить, какая у вас роль, перейдите на страницу Роли участников. Основные сведения о ролях и правах доступа изложены также в Права доступа и роли: обзор.

В зачётной книжке есть несколько таблиц, которые вы можете сортировать разными способами. В любой из этих таблиц просто нажмите заголовок колонки, по которой хотите сортировать. Повторное нажатие одного заголовка колонки включит сортировку по возрастанию или убыванию.

Внимание: В настоящий момент Зачётная Книжка не сортирует по имени. Если у вас есть несколько студентов с одинаковыми фамилиями, сортировка по имени не будет сделана правильно.

Резюмирующая таблица заданий

На странице Задания вы можете сортировать таблицу "Обзор Заданий" по следующим загловкам:

  • Заголовок
  • Средний Балл
  • Срок сдачи
  • Показать Студентам
  • Включено в Оценку за Курс
  • Веса Оценок (если применимо)
  • Редактор Оценок (например Зачётная Книжка, Задания)

Таблица оценок

На странице Все оценки для определённого задания мы можете сортировать "Таблицу Оценок" по следующим заголовкам:

  • Имя студента
  • ID студента
  • Баллы ( заработанные на задинии)

На старанице Оценки за Курс вы можете сортировать "Таблицу Оценок" по следующим заголовкам:

  • Имя студента
  • ID студента
  • Балы (все пункты заработанные за курс)
  • Оценка за Курс (автоматически выставленная оценка за курс)
  • Аннулированная оценка (аннулированная оценка за курс)

Все Оценки

На странице Все Оценки вы можете соритировать таблицы по следующим заголовкам:

  • Имя студента
  • ID студента
  • Оценка за Курс (автоматически выставленная оценка за курс)
  • Каждое Название Задания (отсортированное по количеству пуктов, зарабоатнному на задании)
