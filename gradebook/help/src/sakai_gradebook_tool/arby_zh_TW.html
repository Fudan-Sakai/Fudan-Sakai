排序成績冊表格

註釋：如果要完成本頁中的操作，您必需擁有具有必要權限的角色。如需決定自己的角色，請遵循參與者角色中的說明進行操作。如需瞭解關於角色和權限的基礎知識，請參考權限和角色：概述。

您可以透過各種方法為成績冊中的幾張表格進行排序。在其中任何表格中，只需按一下您想要排序的表格的欄標題。之後再次按一下相同的欄標題將在昇冪和降冪之間切換。

註釋：目前，成績冊不用名字來排序。如果您有幾名學生有相同的姓氏，成績冊不會正確地根據他們的名字來排序。

成績冊項目摘要表格

在成績冊項目頁面中，您可使用下列方法為「成績冊項目摘要」表格排序：

  • 標題
  • 班級平均分
  • 截止日期
  • 向學生發佈
  • 包括在課程成績內
  • 類別權重（若適用）
  • 成績編輯器工具（例如成績冊、作業）

成績表格

在課程成績摘要頁面中，可使用下列方法為「成績表格」排序：

  • 學生姓名
  • 學生 ID
  • 分數（該成績冊項目所獲得的分數）

在課程成績摘要頁面中，可使用下列方法為「成績表格」排序

  • 學生姓名
  • 學生 ID
  • 分數（獲得的課程總分）
  • 課程成績（成績冊自動計算的課程成績）
  • 成績覆蓋（經覆蓋的課程成績）

全部成績

在全部成績頁面中，可使用下列方法為表格排序：

  • 學生姓名
  • 學生 ID
  • 課程成績（成績冊自動計算的課程成績）
  • 各成績冊項目名稱（根據成績冊項目的得分排序）
