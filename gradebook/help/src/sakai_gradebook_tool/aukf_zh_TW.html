從試算表匯入成績冊項目

註釋：如果要完成本頁中的操作，您必需擁有具有必要權限的角色。如需決定自己的角色，請遵循參與者角色中的說明進行操作。如需瞭解關於角色和權限的基礎知識，請參考權限和角色：概述。

在成績冊中 ，您可用兩種不同的方式來匯入成績：

  • 匯入整個試算表。這種方法會為在成績冊中的欄位具有和現有成績冊項目相同名稱來覆蓋任何現有成績並為任何欄位不符合現有成績冊項目建立新的成績冊項目。

  • 匯入個別的成績冊項目。 這種方法會為每一個您匯入的試算表欄建立新的成績冊項目。您不能匯入任何欄具有和現有欄相同名稱。

您也可使用 Post'Em 工具來上載 學生可查看的.csv 成績試算表﹙和回饋，如需要﹚。如需瞭解更多資訊，請參考使用 Post'Em 工具。

註釋：您不能為成績冊項目匯入從作業工具傳送來的成績。

匯入整個試算表

  1. 在網站功能表欄中，按一下成績冊。

  2. 在頂部按一下匯入成績。

  3. 按一下下載試算表範本來下載試算表範本，選擇儲存，並按一下確定。

  4. 使用試算表應用程式編輯該試算表，如 Excel，然後把它儲存為 .csv 檔案。

  5. 按一下瀏覽…並選擇 .csv 檔案。

  6. 選擇了檔案後，按一下匯入試算表。

  7. 隨後頁面應顯示匯入的成績。當您完成時，按一下確定。

當使用此選項，最好在上述第三步驟中下載提供的試算表範本。您可上載另一個 .csv 格式的試算表來代替，如果您要這樣做，請緊密地遵循範本的格式。第一欄應包含個人的使用者名稱，而在此例中的第一列應包含標頭，與每個成績冊項目的標題的分數值：

匯入個別的成績冊項目

如需匯入成績冊項目，您必須先建立含分數得分欄的 .csv 試算表檔案，然後把它上載到成績冊裝載區。您的試算表的第一列必須包含標題欄。第一欄必須包含個人的使用者名稱。

一旦您的 .csv 檔案準備好上載，請遵循這些步驟：

將檔案上載至裝載區

  1. 在網站功能表欄中，按一下成績冊。

  2. 在頂部按一下從試算表匯入成績冊項目。

  3. 在匯入成績冊項目螢幕中，按一下上載試算表﹙csv 格式﹚至上載區.。

  4. 在標題欄位中，輸入要在上載區顯的唯一名稱。

  5. 如需選擇檔案上載，按一下瀏覽…。在您的電腦中尋找檔案，按一下檔案名稱，然後按一下開啟。

  6. 按一下儲存。

  7. 在確認上載螢幕上，檢查要上載項目和學生的數量，並查看檔案內容的顯示。如果內容是正確的， 按一下確定。如果您需要選擇另一個檔案，請按一下返回 。

一旦檔案上載到上載區，其內容變成可匯入的新成績冊項目。如需匯入已上載檔案內容為新成績冊項目：

從上載區匯入檔案

  1. 在匯入成績冊項目螢幕中，在上載區尋找您新上載的檔案，和該檔案旁按一下匯入。

  2. 在匯入預覽螢幕中，按一下選項按鈕選擇該欄，代表您要匯入的成績冊項目和相應分數。 （您每次只可選擇一個。 ）然後按一下匯入選中的 。

  3. 在匯入成績冊項目和分數螢幕中，輸入以下資訊：

    • 標題：您上述步驟提供的標題會自動填入。如果您之前沒有提供標題，您必須在此提供一個。最大長度是255個字元。
    • 成績冊項目分數值：請為成績冊項目輸入總分數值。此數必須大於零。十進制值是允許到小數點後兩位。
    • 截止日期：在月月/日日/年年的格式，表明該項目的截止日期，或按一下行事曆圖示（
      ）使用彈出式的行事曆工具選擇一個日期。此欄位不是必需的。
    • 選擇一個欄來匯入註解：如果在您的檔案中有一欄包含有註解，請使用下拉式清單選擇適當的欄，並新增註解至匯入。註解會在成績冊中的成績旁會出現。此欄位不是必需的。

  4. 如需使學生可見此項目得分，選擇向學牛發佈此項目選項。如需要，您稍後可取消此選項。

  5. 如需在計算課程成績時包括此項目，選擇在課程成績計算中包括此項目選項，如需要，您稍後可取消此選項。

  6. 當您完成後，按一下提交。您會返回匯入預覽螢幕，在那裡您會看見您的項已被添加的確認。您現可選擇另一個項目來匯入，或按一下重設圖示﹙
    ﹚反回主螢幕；請參考重設按鈕。您應看見您匯入的作業清單在「成績冊項目摘要」下。
