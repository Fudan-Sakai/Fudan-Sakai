АРХИВ: Импортирование оценок для существующих заданий

Зачётная книжка даёт возможность импортировать оценки для уже существующих заданий, а также для заданий, которые ещё не созданы. Для импортирования оценок и создания новых заданий, смотрите Импортирование оценок из электронной таблицы. Чтобы импортировать оценки для существующих заданий:

  1. Нажмите Зачётная книжка в линейке инструментов


  2. Нажмите Импортировать оценки вверху страницы

  3. Скачайте шаблон электронной таблицы, нажав Скачать шаблон электронной таблицы.

  4. Отредактируйте электронную таблицу в вашем любимом приложении для работы с электронными таблицами, таком как Excel, и сохраните ее в формате csv.

  5. Импортируйте таблицe, нажав Обзор... и выберете формат.csv.

  6. После того, как файл выбран, нажмите Импортировать Электронную Таблицу.

  7. Следующая страница должна показать импортированные оценки. Когда закончите, нажмите OK.

Примите во внимание

  • Вы не можете импортировать оценки для заданий, отправленных с инструмента Задание

  • Оценки, импортированные с использованием опции Импортировние оценок заменят все ранее существующие оценки. Всегда будьте уверены, используя шаблон электронной таблицы, скачанный в третьем шаге.
