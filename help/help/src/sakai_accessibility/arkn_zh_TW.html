已知存取性問題

Sakai 存在一系列目前尚未解決的存取性問題：

  • 必需啟用 JavaScript 軟體才能正常工作。

  • 按向上或向下鍵時，成績冊、學檔、喜好、課程大綱、Samigo、簡報、花名冊、Wiki 和班別下拉式功能表中的項目會自動載入。需按 Alt - 向上箭頭或 Alt - 向下箭頭在功能表之間捲動。
  • 由於存在單獨的頁面標題和內容框架，因此工具標題顯得多餘。

如果發現任何問題，請存取  elledge@msu.edu  通知 Sakai 可存取性群組。主題行使用 "Sakai Accessibility Issues"（"Sakai 可存取性問題"）。

同時請注意，強烈建議使用 JAWS 6.1 和 Window-Eyes 5.5 或更高版本，以便提高軟體的性能。

可在下列地址下載 JAWS 更新：

http://www.freedomscientific.com/fs_downloads/jaws.asp

可在下列地址檢查 Window-Eyes 更新：

http://www.gwmicro.com/Window-Eyes/Upgrades/

導覽提示和技巧

以下為導覽過程中的一些提示：

  • 如需在下拉式清單中捲動，按 Alt- 向上箭頭或 Alt- 向下箭頭。

  • 選擇工具後，使用 Alt-c 鍵將焦點移動至工具內容。

  • 如果無法找到焦點，按 Alt-c 鍵使焦點返回工具頂部。

  • 使用 Page Down 鍵轉至頁面上的底部按鈕（例如取消）。

  • 使用標題（帶有字母 h ）進行導覽。工作網站和工具部分，以及頁面內容都帶有標題。

  • 如需檢視頁面的全部標題，按 Insert-F6 找到清單，然後使用向上和向下箭頭在清單中捲動。

  • 按 Insert - 向下箭頭重複文字。

  • 如果與工具、內容或工作網站的連結失效，按 Insert-F6 調出標題清單，然後使用標題進行導覽。

如何設定 JAWS 和 Window-Eyes 設定

使用 Insert-v 命令，加上 MSAA 命令中的相關字母（如 h 表示標題），然後使用向上或向下箭頭，透過上述方法可在 Internet Explorer 執行大多數建議設定操作。

如需在 JAWS 內存取 HTML 設定（例如更改 JAWS 的預設設定），請遵循下列步驟進行操作：

  1. 按Alt-u 轉到實用程式。

  2. 按 c 選擇 JAWS 設定管理器。

  3. 按 s 選擇設定選項。

  4. 按 h 選擇 HTML 選項。

  5. 按 Ctrl-Tab 在頁面之間移動。

如需在 Window-Eyes 內存取 HTML 設定，請遵循下列步驟進行操作：

  1. 按 Ctrl-\ 選擇控制面板。

  2. 按Alt-l 選擇全域功能表。

  3. 按 v 選擇 詳細功能表。

  4. 按 m 選擇 MSAA。

標題

使用標題進行導覽。與工作網站、工具和內容之間的連結以標題形式顯示，便於導覽。同時使用 <h1>、<h2> 和 <h3> 有層次地組織工具的內容。

如需設定 JAWS 讀取「層級標題」，請遵循下列步驟進行操作：

  1. 在 Internet Explorer 中，按 Insert-v 轉到 MSAA 功能表。

  2. 按 h  轉到標題，然後按 e  執行操作。

Window-Eyes 預設啟用標題。

框架

由於框架標題往往會出現重複，因此建議您避免由螢幕讀者不必要地發佈標題。JAWS 的預設設定是不顯示框架名稱。

如果您想要設定 JAWS「在開始和結束處顯示框架名稱」，請遵循下列步驟進行操作：

  1. 在 Internet Explorer 中，按 Insert-v 轉到 MSAA 功能表。

  2. 輸入 f f ，然後按 e  執行操作。

Window-Eyes 預設啟用框架名稱。

快捷鍵

快捷鍵遵循適用的英國電子商務標準，具有幫助記憶的功能。在 PC 上使用 Alt 調用快捷鍵；在 Macintosh 上使用 Ctrl。每個連結的快捷鍵都在標題標籤中顯示。

入口網站快捷鍵

  • 可存取性頁面： Alt-0。（必需在您所在機構的權限中設定。可存取性下的幫助工具中也有選擇）。
  • 說明頁面： Alt-6
  • 跳轉至內容： Alt-c
  • 跳轉至工具清單： Alt-l
  • 跳轉至工作網站： Alt-w

工具的特定快捷鍵

大多數基於表單的工具都可使用快捷鍵。

  • 刪除、清除或取消： Alt-x
  • 編輯或修改： Alt-e
  • 説明： Alt-h
  • 重新整理： Alt-u
  • 儲存： Alt-s
  • 檢視或預覽： Alt-v

螢幕重新整理

由使用者控制頁面重新整理。如需重新整理頁面內容，按 F5。

如需設定 JAWS 關閉頁面重新整理，請遵循下列步驟進行操作：

  1. 在 Internet Explorer 中，按 Insert-v 轉到 MSAA 功能表。

  2. 輸入 r r ，然後按 e  執行操作。

如需設定 Window-Eyes，按 Alt-Shift-m 直到聽見「暫停全部」。

標題標籤

標題標籤提供附加資訊，有助於理解操作按鈕和連結。

如需設定 JAWS「使用標題」，請遵循下列步驟進行操作：

  1. 在 JAWS 中，按Alt-u轉到實用程式。

  2. 按 c 轉到設定管理器。

  3. 按 Alt-s 選擇設定選項。

  4. 按  h 選擇 HTML 設定。

  5. 按Ctrl-Tab Ctrl-Tab選擇詳細圖形。

  6. 按 Tab 一次，然後按向上箭頭，直到您聽見「使用標題」。

  7. 按 Enter。

Window-Eyes 預設啟用標題標籤。

表單標籤

表單包含常見的可存取性元素，包括 fieldset、egend、label for 和 id tags。

表格標籤

如需設定 JAWS 忽略佈局表，請遵循下列說明進行操作：

  1. 在 JAWS 中，按Alt-u轉到實用程式。

  2. 按 c 選擇 設定管理器。

  3. 按 Alt-s 選擇設定選項。

  4. 按  h 選擇 HTML 設定。

  5. 按 Ctrl-Tab Ctrl-Tab Ctrl-Tab 選擇清單和表格。

  6. 按Alt-y選擇資料表格。

  7. 如果尚未選中該選項，那麼按空白鍵選中。該選項預設選中。

Window-Eyes 預設忽略佈局表。

合規

應用程式開發者們的目標是確保應用程式符合 Section 508 和 WCAG 1.0 Priorities One and Two，並且盡可能改善軟體對於使用適應技術的使用者們的有用性。大多數工具都嚴格遵守 W3C 可存取性指導方針中對於標題、表單、表格、連接和影像的規定，但是不包括 JavaScript 的非腳本替代 元素，或彈出使用者資訊及已完成的操作。符合大多數指導方針要求的工具包括：

  • 通告
  • 聊天室
  • 討論
  • 電子郵件存檔
  • 閘道
  • 首頁
  • 我的工作區 - 成員資格
  • 我的工作區 - 喜好
  • Page Wrapper
  • 權限
  • Sakai 新聞
  • 日程表
  • 網站資訊
  • 網路內容
  • 工作網站設定
