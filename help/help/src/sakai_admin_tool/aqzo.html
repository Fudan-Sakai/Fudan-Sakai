Changing a user's account type

Note: To use this tool, you must have administrator rights.

To change a user's account type, follow these steps:

  1. From your My Workspace, click the Users tab. This will display a list of users defined within the system.

  2. Under "User Id", click the username of the user whose account type you want to change.

  3. In the "Type:" field, change the type as you wish, and then click Save.
