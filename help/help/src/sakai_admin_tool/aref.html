Deleting a user

Note: To use this tool, you must have administrator rights.

This document is for site administrators only. For information about removing participants from your project site, see Adding, editing, or removing participants.

To delete a user with the Users admin tool:

  1. In My Workspace, from the menubar, click Users.

  2. Click the User Id of the user you want to remove.

  3. At the top of the screen, click Remove User.

  4. Click Remove.
