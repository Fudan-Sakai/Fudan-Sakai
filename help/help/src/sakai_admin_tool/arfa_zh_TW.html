使用使用者工具新增新使用者

註釋：如需使用此工具，您必需擁有管理員權限。

正文件內容僅供網站管理員參考。如需瞭解關於新增成員至專案網站的訊息，請參考新增、編輯或刪除參與者或在工作網站設定中新增、編輯或刪除參與者。

如需使用使用者管理工具新增使用者，請遵循下列步驟進行操作：

  1. 在我的工作區中或管理工作區，從功能表欄按一下使用者。

  2. 在使用者螢幕中，按一下新建使用者。

  3. 輸入使用者的使用者 ID、名字、姓氏和電子郵件，並建立和驗證密碼。在「類型：」欄位中輸入適當的使用者類型：

    • 如需允許使用者建立工作網站，使用registered﹙已註冊﹚ 。
    • 如需禁止使用者建立工作網站，使用guest﹙訪客﹚，或將「類型：」欄位留空。

  4. 按一下儲存詳細資料。
