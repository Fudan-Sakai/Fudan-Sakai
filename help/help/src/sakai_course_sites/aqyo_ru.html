Добавление, изменение ролей и статуса, удаление участников с помощью инструмента "Мои сайты"

Внимание: Для того чтобы совершать все эти действия, вам должна быть назначена роль с соответствующими полномочиями. Для того чтобы определить, какая у вас роль, перейдите на страницу Роли участников. Основные сведения о ролях и правах доступа изложены также в Права доступа и роли: обзор.

С помощью инструмента "Мои сайты" раздела Мой сайт можно добавлять, изменять профили или удалять участников сайта. Чтобы сделать это, на линейке инстурментов раздела "Мой сайт" нажмите Мои сайты, затем поставьте флажок рядом с названием сайта, на котором нужно добавить, изменить профиль или удалить участников.

Внимание: Несмотря на то, что флажок можно поставить рядом с несколькими названиями, изменять настройки сайтов нужно по одному.

Нажмите Изменить и следуйте инструкциям:

Добавление участников

  1. Нажмите Добавить участников.


  2. Для добавления зарегистрированных участников в текстовом поле "Другие зарегистрированные участники" введите ID участника. Если нужно добавить нескольких участников, вводите каждый ID на отдельной строчке.

    Для добавления незарегистрированных участников в текстовом поле "Незарегистрированные участники" введите их адреса электронной почты, по одному на строчке.

  3. Под надписью "Роли участников" выберите, нужно ли присвоить новым участникам одинаковые или разные роли. Подробную информацию смотри в Роли участников. Нажмите Продолжить.

  4. На следующей странице, если вы решили присвоить участникам одинаковые роли, выберите нужную роль. Если вы решили присвоить участникам разные роли, используйте выпадающий список напротив каждого ID участника, чтобы присвоить нужную роль. Нажмите Продолжить.

  5. На следующей странице находится опция для автоматического уведомления новых участников об их добавлении на сайт по электронной почте. Выберите необходимую опцию с помощью переключателя и нажмите Продолжить.

  6. Нажмите Завершить.

Изменение ролей и статуса участников

  1. В списке участников инструмента "Настройки" используйте выпадающий список "Роль", чтобы изменить роль участника.

  2. Используйте выпадающий список "Статус", чтобы задействовать или деактивировать участника. Деактивированный участник остается участником, но перестает иметь доступ к сайту (эту опцию можно использовать, чтобы лишить участника доступа к сайту, сохранив его в списке участников. Такую возможность стоит использовать, например, для сохранения контакта со всеми бывшими студентами).

  3. Нажмите Обновить участников.

Удаление участников

  1. В списке участников инструмента "Настройки" в столбце "Удалить" поставьте флажок напротив ID участника, которого необходимо удалить.

  2. Нажмите Обновить участников.
