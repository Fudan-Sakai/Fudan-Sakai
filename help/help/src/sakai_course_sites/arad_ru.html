Изменение статуса сайта с помощью инструмента "Мои сайты"

Внимание: Для того чтобы совершать все эти действия, вам должна быть назначена роль с соответствующими полномочиями. Для того чтобы определить, какая у вас роль, перейдите на страницу Роли участников. Основные сведения о ролях и правах доступа изложены также в Права доступа и роли: обзор.

Когда сайт готов к публикации, его можно сделать доступным для пользователей (то есть опубликовать) с помощью инструмента "Мои сайты" раздела Мой сайт. Когда вы публикуете сайт, он появляется в новой вкладке вверху страницы каждого участника этого сайта.

Чтобы изменить статус сайта:

  1. На линейке инструментов раздела "Мой сайт" нажмите Мои сайты.

  2. Поставьте флажок рядом с нужным сайтом и нажмите Изменить.

    Можно зайти на страницу настроек и по-другому: выберите нужный сайт, на его линейке инструментов выберите Настройки.

  3. Нажмите Доступ к сайту.

  4. Под надписью "Статус сайта" опубликуйте свой сайт, поставив флажок рядом с надписью Опубликовать сайт, или скройте его, сняв флажок, затем нажмите Обновить.
