Поиск пользователя по профилю

Чтобы найти пользователя по профилю:

  1. В разделе Мой сайт на линейке инструментов нажмите Мой профиль.

  2. Под надписью "Поиск по профилю" введите имя, часть имени или ID пользователя, которого требуется найти. Нажмите Поиск.

  3. Найденные профили будут показаны в списке поиска. Нажмите на ID пользователя, чтобы просмотреть его или ее профиль.
