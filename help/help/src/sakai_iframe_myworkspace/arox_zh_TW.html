顯示、隱藏和重新排序課程和專案網站

註釋：當您登入時，您所屬的課程和專案網站將以分頁形式顯示在螢幕頂部。如需進入網站，按一下該網站的分頁。如果您所屬的網站沒有完全以分頁的形式顯示，那麼可從更多下拉式功能表存取其他未顯示的網站。

可使用我的工作區中的喜好工具選擇隱藏網站（例如，以前學期的課程網站）或重新排序網站（即，選擇顯示在分頁的網站）。如需進行上述操作，請遵循下列步驟進行操作：

  1. 從我的工作區功能表欄按一下喜好。

  2. 按一下自訂分頁。

  3. 在「自訂分頁」欄位下，您當前可見的網站列在「顯示在分頁的網站」方塊內。列在「不顯示在分頁的網站」方塊內的網站不會出現在 - 更多 - 分頁中。請使用右箭頭和左箭頭按鈕 ( < 和 > ) 來移動網站從一個方塊到另一個方塊。按一下雙箭頭按鈕 ( << 和  >> )來移動全部網站從一個方塊到另一個方塊。

    註釋：如果隱藏了網站的分頁，您仍然可以從我的工作區中的工作網站設定存取網站。

  4. 如需設定分頁顯示的順序，按一下「顯示在分頁的網站」中的網站，然後使用向上和向下箭頭更改順序。

    註釋：我的工作區分頁將始終顯示在其他網站的左側。

  5. 操作完成後，按一下更新喜好。

提示：如果您有大量啟用的網站，可以將某些網站快捷地移至清單的頂部而使其顯示在分頁中，使用下列方法而不是上下箭頭：

  1. 在自定分頁頁面，按一下 << 將您全部啟用的網站移到「不顯示在分頁的網站」的方塊中。

  2. 選擇您想要顯示在分頁的課程。﹙如需選擇多個課程，在 Windows 按住 Ctrl 或在 Mac 按住 Cmd，然後點擊。﹚

  3. 按一下 > 將選中的課程，移回到「顯示在分頁的網站」的方塊內。

  4. 如需要，按一下 << 將全部剩餘的網站移回「顯示在分頁的網站」的方塊中，他們將出現在您剛移動網站的後面。您也可以重複步驟2和3移回某些網站。

  5. 操作完成後，按一下更新喜好。
