Какой браузер стоит использовать?

Внимание: Чтобы определить, какую версию Internet Explorer или Firefox вы используете в операционной системе Windows, в программе браузера нажмите "Справка", затем нажмите " О программе" в Internet Explorer или " О Mozilla Firefox" в Mozilla Firefox. В операционной системе Macintosh в меню "Firefox" выберите "About Mozilla Firefox".

Используйте один из браузеров :

Windows

  • Internet Explorer 7.0 и выше
  • Mozilla Firefox 2.0 и выше

Macintosh

  • Mozilla Firefox 2.0 и выше

Другие операционные системы

  • Mozilla Firefox 2.0 и выше

Замечания:

Внимание: Если использовать два браузера для доступа к системе Sakai одновременно, может случиться что угодно. Для надежности, используйте для работы в системе Sakai одно окно браузера.
