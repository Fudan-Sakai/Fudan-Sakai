我應該使用哪種瀏覽器？

註釋：如需確定您微軟視窗電腦正在用哪個 Internet Exploere 或 Firefox 的版本，請在瀏覽器按一下說明，然後按一下關於 Internet Explorer或 關於 Mozilla Firefox。 在蘋果 Macintosh 電腦，從 Firefox 功能表中選擇關於 Mozilla Firefox。

請使用以下其中一種瀏覽器：

Windows

  • Internet Explorer 7.0 及更高版本
  • Mozilla Firefox 2.0 及更高版本

Macintosh

  • Mozilla Firefox 2.0 及更高版本

其他平臺

  • Mozilla Firefox 2.0 及更高版本

其他注意事項

註釋：如果您同時使用二個瀏覽器視窗存取同一個 Sakai 2.5 的工具，您可能收到意想不到的結果。 對於最佳的結果，請使用一個瀏覽器視窗在 Sakai 2.5 工作。
