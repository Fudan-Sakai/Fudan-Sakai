將文字貼上至 Firefox 中的 FCKeditor 時出現的問題

由於 Firefox 中的某些安全設定，當您嘗試透過按右鍵並選擇貼上將文字貼上至 FCKeditor 時，可能會出現問題。低版本的 Firefox 不允許使用 Ctrl-v 貼上文字。如需解決這些問題，從下列解決方案中選擇一種：

  • 從編輯功能表選擇貼上。

  • 按 Ctrl-v。如果問題仍然存在，升級至最新版本的 Firefox。

  • 更改安全設定，使得 Firefox 允許透過按右鍵進行貼上操作。如需在 Windows 完成上述操作，請遵循下列說明：

    1. 關閉全部 Firefox 視窗。

    2. 瀏覽至下列目錄，其中 username 是您的使用者名，ProfileName 是您的全名或是帶有 ".default" 尾碼的字串： C:\Documents and Settings\username\Application Data\Mozilla\Firefox\Profiles\ProfileName
    3. 使用文字編輯器在該目錄下建立名為 user.js 的檔案。

      註釋：確保文字編輯器沒有在檔案名最後新增 .txt 副檔名。檔案的副檔名必須為 .js。

    4. 在該檔中輸入下列內容： user_pref("capability.policy.policynames", "allowclipboard"); user_pref("capability.policy.allowclipboard.sites", "https://www.samplesite.com"); user_pref("capability.policy.allowclipboard.Clipboard.cutcopy", "allAccess"); user_pref("capability.policy.allowclipboard.Clipboard.paste", "allAccess");

      將 https://www.samplesite.com 替換為您想要允許貼上操作的網站的名稱。請勿使用 /（斜杠）。

    5. 重啟 Firefox。
