Настройки добавления приложений

Внимание: Для того чтобы совершать все эти действия, вам должна быть назначена роль с соответствующими полномочиями. Для того чтобы определить, какая у вас роль, перейдите на страницу Роли участников. Основные сведения о ролях и правах доступа изложены также в Права доступа и роли: обзор.

При создании нового элемента, некоторые инструменты предоставляют возможность добавления вложенных файлов (приложений). Можно добавить любое количество приложений, а также сочетать файлы со своего компьютера с веб-страницами и файлами инструмента "Ресурсы". Чтобы добавить приложение, следуйте инструкциям:

  • Чтобы добавить файл со своего компьютера, под надписью "Загрузить локальный файл" нажмите Обзор, чтобы найти и выбрать файл на компьютере.

  • Чтобы добавить URL, под надписью "URL (ссылка на веб-сайт)" наберите URL в текстовом поле и нажмите Добавить.

    Внимание: При наборе URL вводить http:// не нужно.

  • Чтобы добавить элемент из инструмента "Ресурсы", нажмите Вложить копию рядом с названием файла, который нужно добавить.

    Внимание: Можно приложить файл с другого сайта системы Sakai (включая раздел Мой сайт), если опции доступа к файлу установлены на Этот файл для публичного просмотра. Чтобы найти файлы на других сайтах, нажмите Показать остальные сайты.

Если вы добавляете приложения, они появляются в списке под надписью "Элементы для вложения". Когда вы добавили все необходимые файлы, проверьте список и нажмите Удалить, если найдете в списке ошибки.

Когда вы закончите, нажмите Продолжить внизу страницы. Вы увидите все свои файлы в списке "Приложения".
