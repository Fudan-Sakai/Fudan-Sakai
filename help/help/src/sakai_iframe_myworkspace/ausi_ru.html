Изменение статуса приватности

По некоторым соображениям, вы, возможно, захотите скрыть свое имя от других участников вашего сайта курса или сайта проекта. Кроме того, вы, возможно, не захотите, чтобы они могли связываться с вами через сайт.

Чтобы скрыть свое имя от других участников и запретить им связываться с вами, выберите "Скрытый" в меню "Статус приватности" в инструменте "Главная", а затем нажмите Обновить.

Чтобы изменить свой статус приватности на видимый и дать другим участникам возможноть контактировать с вами через сайт, выберите "Видимый", а затем нажмите Обновить.

Можно также изменить свой статус с помощью инструмента "Профиль" раздела Мой сайт. Чтобы это сделать:

  1. На линейке инструментов раздела "Мой сайт" нажмите Настройки.

  2. Нажмите на ссылку Статус приватности вверху страницы.

  3. Выберите сайт из выпадающего меню, а затем укажите Скрытый или Видимый, чтобы установить ваш статус приватности на этом сайте.

  4. Нажмите Обновить, чтобы сохранить изменения.

Короме того, можно нажать на Видимый на всех сайтах, чтобы стать видимым на всех сайтах, на которых вы участвуете, или Скрытый на всех сайтах, чтобы перестать быть видимым на всех сайтах, на которых вы участвуете.

Важные замечания

Изменяя свой статус приватности, имейте в виду:

  • Студенты, имеющие статус "Скрытый", остаются видимыми для преподавателей, администраторов и владельцев сайта.

  • Студенты, имеющие статус "Скрытый", не будут видны для других участников ни через инструмент "Реестр", ни в списке "Сейчас на сайте".

  • Студенты, имеющие статус "Скрытый", отсутствуют в списке адресатов в инструменте "Сообщения".

  • Персональный статус приватности не влияет на возможность просматривать группы.
