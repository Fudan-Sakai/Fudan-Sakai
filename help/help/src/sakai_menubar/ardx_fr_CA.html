Ajouter, éditer ou supprimer des outils

Note: Pour compléter les instructions contenues dans ce document, vous devez avoir un rôle ayant les droits d'accès requis. Pour déterminer votre rôle, suivez les instructions dans Rôles du participant. Pour une compréhension générale des rôles et droits d'accès, consultez Droits d'accès et rôles: Vue d'ensemble.

Note: Si le menu de votre site ne contient pas l'Information du site, consultez Ajouter, éditer ou supprimer des outils du menu à partir de la Configuration de Sites.

À l'aide de l'outil Information du site, vous pouvez ajouter, éditer ou supprimer des outils au menu. Pour ce faire, suivez les étapes ci-dessous:

  1. Dans votre site, dans le menu, cliquez Info du site.

  2. Cliquez Éditer les outils.

  3. Cochez les cases correspondant aux outils que vous désirez ajouter à votre menu, décochez la case si vous supprimez un outil.

  4. Cliquez Continuer.

  5. Si vous ajoutez ou utilisez déjà les outils suivants, l'écran Personnaliser les outils vous permettra de faire certaines modifications. Si vous n'ajoutez pas ou n'utilisez pas un des outils ci-dessous, allez à l'étape suivante. Pour les outils suivants, ces tâches sont possibles:

    • Archive de courriels: Créez ou modifiez l'adresse courriel de votre site
    • Nouvelles: Ajoutez ou modifiez les titres des sources de nouvelles et les URL; les titres apparaîtront dans votre menu
    • Contenu Web: Ajoutez ou modifiez les titres et les URL; les titres apparaîtront dans votre menu

    Lorsque vous aurez terminé, cliquez Continuer.

  6. Dans l'écran de confirmation, vous verrez une liste d'outils et tout changement apporté. Si la liste est exacte, cliquez Terminer. Pour éditer vos modifications, cliquez Retour, ou si vous désirez annuler, cliquez Annuler.
