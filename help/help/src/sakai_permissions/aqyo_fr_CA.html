Ajouter, éditer ou supprimer des participants à partir de la Configuration de Sites

Note: Pour compléter les instructions contenues dans ce document, vous devez avoir un rôle ayant les droits d'accès requis. Pour déterminer votre rôle, suivez les instructions dans Rôles du participant. Pour une compréhension générale des rôles et droits d'accès, consultez Droits d'accès et rôles: Vue d'ensemble.

À partir de Configuration de Sites dans Zone de Travail, vous pouvez ajouter, éditer ou supprimer les participants de votre site. Pour ce faire, à partir du menu de votre Zone de Travail, cliquez Configuration de Sites, puis cochez la case correspondant au site dans lequel vous désirez ajouter, éditer ou supprimer des participants.

Note: Même s'il vous est possible de cocher plusieurs cases, vous ne pouvez réviser les outils que d'un seul site à la fois.

Cliquez Réviser, puis suivez les étapes appropriées suivantes:

Ajouter des participants

  1. Cliquez Ajouter participants.

  2. Pour les participants qui ont des noms d'utilisateur officiels, sous "Nom(s) d'utilisateur" inscrivez le nom d'utilisateur du participant. Si vous désirez ajouter plus d'un participant, inscrivez chaque nom d'utilisateur sur une ligne séparée.

    Pour les participants qui n'ont pas de noms d'utilisateur officiels, sous "Adresse courriel de(s) invité(s) (participants externes, ex.: jdoe@yahoo.com)", inscrivez leur adresse courriel, une par ligne.

  3. Sous Rôles des participants, choisissez soit de donner le même rôle à tous vos nouveaux participants ou des rôles différents. Pour plus d'information à propos des rôles, consultez Rôles des participants. Cliquez Continuer.

  4. Dans la fenêtre suivante, si vous avez choisi de donner le même rôle à tous vos participants, sélectionnez le rôle approprié à donner à tout le monde. Si vous avez choisi de donner un rôle différent à chaque participant, utilisez la liste déroulante à côté de chacun des noms d'utilisateur pour choisir le rôle approprié. Cliquez Continuer.

  5. Sur la page suivante, vous avez la possibilité d'envoyer automatiquement un courriel aux nouveaux participants les avisant de la disponibilité du site. Sélectionnez le bouton radio approprié puis cliquez Continuer.

  6. Cliquez Terminer.

Éditer des participants

  1. Dans la liste de participants, utilisez la liste déroulante "Rôle" pour modifier le rôle d'un participant.

  2. Utilisez la liste déroulante "Statut" pour activer ou désactiver un participant.

  3. Cliquez Mettre à jour participants.

Supprimer des participants

  1. Dans la liste de participants, sous "Supprimer", cochez la(les) case(s) à côté du(des) participants que vous voulez supprimer.

  2. Cliquez Mettre à jour participants.
