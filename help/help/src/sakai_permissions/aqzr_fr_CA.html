Créer/ajouter, éditer ou supprimer des sites

Dans cette page:

  • Créer/ajouter un site
  • Éditer un site
  • Supprimer un site

Créer/ajouter un site

Pour créer un site, suivez les étapes suivantes:

  1. Dans le menu de votre Zone de Travail, cliquez Configuration de Sites.

  2. En haut de la page de configuration du site, cliquez Nouveau.

  3. Sous "Création d'un nouveau site", cliquez le bouton radio à côté de Site Web de cours ou Site Web de projet pour choisir le type de site que vous voulez créer. Pour un site de cours, vous devrez aussi utiliser la liste déroulante "Session de cours" pour sélectionner la session appropriée.

  4. Cliquez Continuer.

Sites de cours

Si vous choisissez de créer un site de cours, continuez en suivant les étapes suivantes (pour un site de projet, allez à Sites de projets):

  1. Sous "Création d'un nouveau site de classe", sélectionnez les classes/sections qui seront incluses dans le site du cours en cochant les cases appropriées.

  2. Pour ajouter une liste d'étudiants qui n'apparaît pas dans la liste affichée, cochez la case à côté de Je veux donner l'accès à une liste non énumérée ci-haut.

  3. Cliquez Continuer.

  4. Si vous choisissez d'ajouter une liste d'étudiants n'apparaissant pas dans la liste, vous devrez entrer manuellement l'information de la classe dans différents champs. Pour ajouter plus d'une liste, dans la liste déroulante "Ajouter plus de liste(s)", sélectionnez le nombre de listes supplémentaires que vous voulez ajouter. Vous devrez peut-être inscrire le nom d'utilisateur d'une personne autorisée responsable pour permettre l'accès au site à la (aux) liste(s) ainsi qu'un message contenant des instructions particulières au sujet de votre requête. Cliquez Continuer.

  5. Sous "Information de la classe", vous avez la possibilité d'inscrire une description longue et/ou une description courte de votre site. La description longue apparaîtra sur la page d'accueil de votre site; la description courte apparaîtra dans la liste publique de tous les sites du système.

  6. Pour modifier l'apparence de votre site, choisissez un habillage à partir de la liste déroulante "Apparence (icône)".

  7. Dans les champs "Nom de la personne-ressource du site:" et "Courriel de la personne-ressource du site:", inscrivez le nom et l'adresse courriel de la personne que les participants peuvent contacter s'ils ont des problèmes ou des questions.

  8. Cliquez Continuer.

  9. Sous "Outils", cochez les cases qui correspondent aux outils que vous voulez dans votre site. Si vous sélectionnez les outils suivants, un formulaire vous permettra d'inscrire de l'information supplémentaire:

    • Archive de courriels: Inscrivez un alias (c.-à-d. nom d'utilisateur) pour l'adresse courriel de votre site, puis cliquez Continuer.
    • Nouvelles: Inscrivez un titre et une URL pour votre source de nouvelles. Pour ajouter des sources supplémentaires, utilisez la liste déroulante "Plus d'outils Nouvelles?", puis cliquez Continuer.
    • Contenu Web: Inscrivez un titre et une URL. Pour ajouter des liens supplémentaires, utilisez la liste déroulante "Plus d'outils de contenu Web?"

  10. Vous avez la possibilité de réutiliser du matériel d'autres sites dont vous êtes le propriétaire. Choisissez une des options suivantes:

    • Non, merci.
    • Oui, de ces sites: . Utilisez le menu déroulant pour sélectionner les sites contenant le matériel que vous voulez réutiliser.

  11. Cliquez Continuer.

  12. Sous "Statut du site" sélectionnez Publier le site pour le rendre disponible aux participants du site. Si vous ne voulez pas publier votre site dès maintenant, vous pouvez le publier plus tard en passant par l'Information du site. Pour plus d'information, consultez Publier ou suspendre la publication d'un site.

    De plus, vous pouvez faire en sorte qu'il soit possible, pour quiconque a l'autorisation d'ouvrir une session, de s'inscrire à votre site. Pour ce faire, suivez les étapes suivantes:

    1. Cochez la case Possible de s'y inscrire pour quiconque a l'autorisation d'ouvrir une session.
    2. Utilisez la liste déroulante pour sélectionner un rôle à attribuer aux personnes qui s'abonnent à votre site.

  13. Cliquer Continuer.

  14. Sous "Confirmer la configuration de votre site", révisez l'information de votre site. Si des modifications sont requises, cliquez Retour. Si l'information est correcte, cliquez Demander site pour envoyer votre requête de site de cours.

Sites de projets

Si vous choisissez de créer un site de projet, continuez en suivant les étapes suivantes:

  1. Sous "Information du projet", dans le champ "Titre du site:", inscrivez un titre pour votre site.

  2. Dans les champs correspondants, vous avez la possibilité d'inscrire une description longue et/ou une description courte de votre site. La description longue apparaîtra sur la page d'accueil de votre site; la description courte apparaîtra dans la liste publique de tous les sites du système.

  3. Si vous voulez utiliser une icône, inscrivez son URL dans le champ "URL de l'icône".

  4. Dans les champs "Nom de la personne-ressource du site:" et "Courriel de la personne-ressource du site:", inscrivez le nom et l'adresse courriel de la personne que les participants peuvent contacter s'ils ont des problèmes ou des questions.

  5. Cliquez Continuer.

  6. Sous "Outils", cochez les cases qui correspondent aux outils que vous voulez dans votre site, puis cliquez Continuer.

    • Archive de courriels: Inscrivez un alias (c.-à-d. nom d'utilisateur) pour l'adresse courriel de votre site, puis cliquez Continuer.
    • Nouvelles: Inscrivez un titre et une URL pour votre source de nouvelles. Pour ajouter des sources supplémentaires, utilisez la liste déroulante "Plus d'outils Nouvelles?", puis cliquez Continuer.
    • Contenu Web: Inscrivez un titre et une URL. Pour ajouter des liens supplémentaires, utilisez la liste déroulante "Plus d'outils de contenu Web?"

  7. Vous avez la possibilité de réutiliser du matériel d'autres sites dont vous êtes le propriétaire. Choisissez une des options suivantes:

    • Non, merci.
    • Oui, de ces sites: . Utilisez le menu déroulant pour sélectionner les sites contenant le matériel que vous voulez réutiliser.

  8. Sous "Régler l'accès au site" sélectionnez Publier le site pour le rendre disponible aux participants du site. Si vous ne voulez pas publier votre site dès maintenant, vous pouvez le publier plus tard en passant par l'Information du site. Pour plus d'information, consultez Publier ou suspendre la publication d'un site.

  9. Pour indiquer qui peut accéder à votre site, choisissez l'une des options d'accès global suivantes:

    • Privé: Seules les personnes que vous ajoutez comme participants peuvent voir votre site.
    • Afficher mon site dans le répertoire, et partager les fichiers que je sélectionne: quiconque ayant le droit d'ouvrir une session dans le système peut voir votre site dans le répertoire de sites et peut accéder aux fichiers partagés.

    De plus, vous pouvez faire en sorte qu'il soit possible, pour quiconque a l'autorisation d'ouvrir une session, de s'inscrire à votre site. Pour ce faire, suivez les étapes suivantes:

    1. Cochez la case Possible de s'y inscrire pour quiconque a l'autorisation d'ouvrir une session.
    2. Utilisez la liste déroulante pour sélectionner un rôle à attribuer aux personnes qui s'abonnent à votre site.

  10. Cliquez Continuer.

  11. Sous "Confirmer la configuration de votre site", révisez l'information de votre site. Si l'information est correcte, cliquez Créer site. Si des modifications sont requises, cliquez Retour.

Votre nouveau site apparaîtra dans Zone de Travail dans la page Configuration de Sites.

Éditer un site

Pour éditer un site que vous avez créé, suivez les étapes suivantes:

  1. Dans le menu de Zone de Travail, cliquez Configuration de Sites.

  2. Cochez la case à côté du site que vous voulez éditer. Puis, en haut de la page Configuration de Sites, cliquez Réviser.

  3. À l'aide des liens près du haut de la page, sélectionnez une option pour éditer votre site:

    • Pour éditer le titre du site, la description, la description courte, l'URL de l'icône, le nom de la personne-ressource du site ou le courriel de la personne-ressource du site, cliquez Éditer l'information du site.

    • Pour ajouter ou supprimer des outils à votre site, cliquez Éditer les outils.

    • Pour modifier le statut de publication du site, qui peut voir ou s'inscrire à votre site, et déterminer le rôle par défaut des personnes qui s'inscrivent, cliquez Gérer accès.

    • Pour ajouter des participants et leur attribuer des rôles, cliquez Ajouter participants.

    • Pour créer un double exact de votre site incluant seulement le matériel publié par l'instructeur, cliquez Dupliquer site.

    • Pour ajouter du matériel provenant d'un autre site dont vous êtes le propriétaire, cliquez Importer du site.

      Note: Vous ne pouvez réutiliser du matériel que des sites dont vous êtes le propriétaire. Vous pouvez combiner du matériel de plusieurs sites.

    • Pour importer du matériel d'un fichier déjà existant, cliquez Importer du fichier.

Supprimer un site

Pour supprimer un site que vous avez créé, suivez les étapes suivantes:

  1. Dans le menu de Zone de Travail, cliquez Configuration de Sites.

  2. Cochez la case à côté du site que vous voulez supprimer. Puis, en haut de la page Configuration de Sites, cliquez Supprimer.

    Note: La suppression d'un site supprime tout le contenu du site et n'est pas récupérable. Personne d'autre ne sera capable d'accéder au site supprimé.

  3. Cliquez Supprimer.
