Создание/добавление сайта

Чтобы создать сайт:

  1. На линейке инструментов раздела "Мой сайт" нажмите Мои сайты.

  2. Вверху страницы, нажмите Новый.

  3. Под надписью "Создание нового сайта" используйте переключатель рядом с веб-сайт курса, веб-сайт проекта или веб-сайт портфолио, чтобы выбрать тип сайта, который надо создать. Для сайта курса также понадобится задать семестр: используйте выпадающий список для выбора.

  4. Нажмите Продолжить.

Сайт курса

Чтобы создать сайт курса, следуйте инструкциям (для создания сайта проекта, перейдите к Сайт проекта):

  1. Под надписью "Создание нового сайта курса" выберите группу, чтобы включить ее в курс, поставив флажок напротив нужной группы.

  2. Чтобы добавить список студентов, которого нет на странице, нажмите Добавить курс и/или группу, не приведенную на странице.

  3. Нажмите Продолжить.

  4. Если вы решили добавить список студентов, которого нет на странице, то нужно вручную ввести информацию о курсе в нескольких полях. Чтобы добавить более одного списка студентов, в выпадающем списке "Добавить еще список студентов", выберите списки, которые нужно добавить. Возможно, будет необходимо ввести ID пользователя, который подтвердит, что вы можете добавить этот курс или группу на сайт (например, глава учебного отдела или предыдущий преподаватель). Письмо с запросом авторизации будет отправлено этому пользователю. Нажмите Продолжить.

  5. Под надписью "Описание" необходимо ввести подробное и/или краткое описание сайта. Подробное описание появится на странице сайта, краткое будет видно в списке всех сайтов системы.

  6. Чтобы изменить внешний вид вашего сайта, выберите скин из выпадающего списка "Иконка".

  7. В тектовых полях "Контактное лицо:" и "Контактный адрес электронной почты:" введите ID и адрес электронной почты пользователя, с которым участники могут связываться, если у них возникнут проблемы или вопросы о работе сайта.

  8. Нажмите Продолжить.

  9. Под надписью "Инструменты" поставьте флажок напротив названий инструментов, которые необходимо добавить на сайт. Если вы выберете следующие инструменты, то появится форма для ввода дополнительной информации:

    • Архив электронной почты: Введите псевдоним электронной почты (например, ID пользователя) для почтового адреса сайта и нажмите Продолжить.
    • Новости: Выберите название и URL для страницы новостей. Чтобы добавить больше страниц, используйте выпадающий список "Добавить еще одну страницу новостей", затем нажмите Продолжить.
    • Веб-ресурсы: Введите название ресурса и URL. Чтобы добавить больше ресурсов, используйте выпадающий список "Добавить еще одну страницу веб-ресурсов".

  10. Вам будет предложено использовать материал с других сайтов, которыми вы владеете. Выберите один из вариантов:

    • Нет, спасибо.
    • Да, с этих сайтов: Используйте прокручивающееся меню для выбора сайтов, материалы которых нужно использовать.

  11. Нажмите Продолжить.

  12. Под надписью "Статус сайта " выберите Опубликовать сайт, чтобы сделать его доступным для участников. Если пока не нужно опубликовывать сайт, это можно сделать позднее через инструмент Настройки. Подробные инструкции смотри в Опубликование сайта.

    Можно также разрешить публичный доступ к сайту:

    1. Поставьте флажок рядом с Стать участником сайта может любой зарегистрированный в системе пользователь.
    2. Используйте выпадающий список для выбора ролей, которые будут назначаться пользователям, которые заходят на ваш сайт.

  13. Нажмите Продолжить.

  14. Под надписью "Подтвердите настройки вашего сайта" находится информация о создаваемом сайте. Если нужно внести изменения, нажмите Назад. Если информация верна, нажмите Запросить сайт, чтобы отправить запрос о создании сайта.

Изменение сайта

Чтобы изменить созданный сайт:

  1. На линейке инструментов раздела "Мой сайт" нажмите Мои сайты.

  2. Отметьте флажком название сайта, который нужно изменить и нажмите Изменить.

  3. Используя ссылки вверху страницы, выберите настройку, которую нужно изменить на сайте:

    • Чтобы изменить название сайта, подробное или краткое описание, иконку, контактное лицо или контактный адрес электронной почты, нажмите Информация о сайте.

    • Чтобы добавить или удалить инструменты на сайт, нажмите Инструменты.

    • Чтобы изменить настройки доступа к сайту, выбрать, кто может увидеть сайт в списке, и установить роли по умолчанию для пользователей, заходящих на сайт, нажмите Доступ к сайту.

    • Чтобы добавить участников и присвоить им определенные роли, нажмите Добавить участников.

    • Чтобы сделать точную копию сайта, которая содержит только материал, добавленный преподавателем, нажмите Дублировать сайт.

    • Чтобы добавить материал с другого вашего сайта, нажмите Импорт с другого сайта.

      Внимание: Можно использовать материал только с того сайта, которым вы владеете. Можно совмещать материал с разных сайтов.

    • Чтобы импортировать материал из файла, нажмите Импорт из файла.

Удаление сайта

Чтобы удалить свой сайт:

  1. На линейке инструментов раздела "Мой сайт" нажмите Мои сайты.

  2. Поставьте флажок рядом с названием сайта, который нужно удалить. Затем вверху страницы нажмите Удалить.

    Внимание: При удалении сайта безвозвратно удаляются все его материалы. Никто больше не будет иметь доступа к сайту.

  3. Нажмите Удалить.
