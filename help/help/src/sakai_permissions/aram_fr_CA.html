Régler les droits d'accès des annonces

Note: Pour compléter les instructions contenues dans ce document, vous devez avoir un rôle ayant les droits d'accès requis. Pour déterminer votre rôle, suivez les instructions dans Rôles du participant. Pour une compréhension générale des rôles et droits d'accès, consultez Droits d'accès et rôles: Vue d'ensemble.

Par défaut, tout membre d'un site peut lire les annonces. Par ajouter ou modifier les droits d'accès pour lire, créer, réviser ou supprimer n'importe quelle annonce, suivez les étapes ci-dessous:

  1. Entrez dans un site, cliquez Annonces, puis cliquez Droits d'accès.

  2. Cochez les cases appropriées pour régler les droits d'accès. Vous aurez les options suivantes pour les différents rôles:

    • read
    • new
    • delete.own
    • delete.any
    • revise.own
    • revise.any
    • read.drafts
    • all.groups

  3. Après avoir réglé les droits d'accès pour les annonces, cliquez Enregistrer.

  4. Si vous désirez quitter et revenir à votre configuration actuelle, cliquez Annuler.
