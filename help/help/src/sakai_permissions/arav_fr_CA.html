Ajouter, éditer ou supprimer des participants

Dans cette page:

  • Ajouter des participants
  • Éditer des participants
  • Supprimer des participants

Note: Pour compléter les instructions contenues dans ce document, vous devez avoir un rôle ayant les droits d'accès requis. Pour déterminer votre rôle, suivez les instructions dans Rôles du participant. Pour une compréhension générale des rôles et droits d'accès, consultez Droits d'accès et rôles: Vue d'ensemble.

Ajouter des participants

  1. Dans votre site, dans le menu, cliquez Info du site.

  2. Cliquez Ajouter participants.

  3. Pour les participants qui ont des noms d'utilisateur officiels, sous "Nom(s) d'utilisateur" inscrivez le nom d'utilisateur du participant. Si vous désirez ajouter plus d'un participant, inscrivez chaque nom d'utilisateur sur une ligne séparée.

    Pour les participants qui n'ont pas de noms d'utilisateur officiels, sous "Adresse courriel de(s) invité(s) (participants externes, ex.: jdoe@yahoo.com)", inscrivez leur adresse courriel, une par ligne.

  4. Sous Rôles des participants, choisissez soit de donner le même rôle à tous vos nouveaux participants ou des rôles différents. Pour plus d'information à propos des rôles, consultez Rôles des participants. Cliquez Continuer.

  5. Choisissez les rôles pour le(s) participant(s) que vous ajoutez. Si vous attribuez des rôles différents aux participants, utilisez la liste déroulante à côté de chacun des noms d'utilisateur pour choisir le rôle approprié. Si vous attribuez le même rôle à tous les participants que vous ajoutez, utilisez les boutons radio pour sélectionner le rôle approprié. Cliquez Continuer.

    Note: Selon la configuration de votre site, les rôles peuvent varier.

  6. Sur la page suivante, vous avez la possibilité d'envoyer automatiquement un courriel aux nouveaux participants les avisant de la disponibilité du site. Sélectionnez le bouton radio approprié puis cliquez Continuer.

  7. Confirmez que l'information du(des) participant(s) que vous ajoutez est correcte, puis cliquez Terminer.

Éditer des participants

  1. Dans votre site, dans le menu, cliquez Info du site.

  2. Dans la liste de participants, vous pouvez modifier le rôle d'un participant en utilisant la liste déroulante "Rôle".

  3. Sous "Statut", utilisez la liste déroulante pour modifier le statut d'un participant; sélectionnez Actif ou Inactif.

  4. Cliquez Mettre à jour participants.
Retour en haut

Supprimer des participants

  1. Dans votre site, dans le menu, cliquez Info du site.

  2. Dans la liste de participants, sous "Supprimer", cochez la case à côté de chacun des participants que vous voulez supprimer.

  3. Cliquez Mettre à jour participants.
