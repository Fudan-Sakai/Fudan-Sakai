Contrôler l'accès à votre site

Note: Pour compléter les instructions contenues dans ce document, vous devez avoir un rôle ayant les droits d'accès requis. Pour déterminer votre rôle, suivez les instructions dans Rôles du participant. Pour une compréhension générale des rôles et droits d'accès, consultez Droits d'accès et rôles: Vue d'ensemble.

Pour contrôler l'accès à votre site, suivez les étapes suivantes:

  1. Dans le menu de votre site, cliquez Info du site.

  2. Cliquez Gérer accès.

  3. Sous "Accès global", vous pouvez choisir une des options suivantes:

    • Pour les sites de projet seulement, vous pouvez choisir de garder votre site privé (cliquez le bouton radio à côté de Privé) ou vous pouvez sélectionner Afficher mon site dans le répertoire, et partager les fichiers que je sélectionne.
    • Si vous désirez rendre votre site de cours ou de projet largement accessible, cliquez la case à cocher à côté de Possible de s'y inscrire pour quiconque a l'autorisation d'ouvrir une session. À côté de "Rôle pour les personnes qui s'abonnent au site:", sélectionnez le rôle approprié pour tous les utilisateurs qui peuvent s'abonner à votre site.

      Pour plus d'information à propos des rôles, consultez Rôles du participant.

  4. Cliquez Mettre à jour.
