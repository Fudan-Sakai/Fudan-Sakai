Modifier les droits d'accès pour un outil dans un site de cours ou de projet

Note: Pour compléter les instructions contenues dans ce document, vous devez avoir un rôle ayant les droits d'accès requis. Pour déterminer votre rôle, suivez les instructions dans Rôles du participant. Pour une compréhension générale des rôles et droits d'accès, consultez Droits d'accès et rôles: Vue d'ensemble.

Les droits d'accès sont accordés pour un outil plutôt que pour un élément particulier créé en utilisant un outil. Par exemple, les droits d'accès sont réglés pour l'outil Annonces, plutôt que pour des annonces individuelles.

Note: Dans les ressources, vous pouvez régler les droits d'accès pour toutes les ressources, et vous pouvez aussi régler les droits d'accès pour des dossiers des ressources.

Pour modifier les droits d'accès pour un outil, suivez les étapes ci-dessous.

  1. Ouvrez le site en cliquant son onglet.

  2. Dans le menu, cliquez l'outil pour lequel vous voulez régler les droits d'accès.

  3. Cliquez le bouton Droits d'accès en haut de la fenêtre principale.

  4. Cochez ou décochez les cases pour accorder les droits d'accès.

  5. Cliquez Enregistrer. Pour modifier votre sélection, cliquez Annuler.

Note: Vous devez quitter la fonctionnalité des droits d'accès en cliquant Enregistrer ou Annuler avant de pouvoir régler les droits d'accès d'un autre outil.

Chaque outil a une série de droits d'accès différents. Pour vous donner une idée générale des droits d'accès disponibles, voici la liste des droits d'accès pour l'outil Calendrier:

  • nouveau: Créer un nouvel élément

  • supprimer: Supprimer un élément

  • importer: Importer un élément

  • lire: Lire un élément

  • réviser: Réviser un élément

  • all.groups (tout.groupes): Voir des articles publiés associés à n'importe quel groupe
