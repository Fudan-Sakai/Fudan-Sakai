更改課程或專案網站中工具的權限

註釋：如果要完成本頁中的操作，您必需擁有具有必要權限的角色。如需決定自己的角色，請遵循參與者角色中的說明進行操作。如需瞭解關於角色和權限的基礎知識，請參考權限和角色：概述。

權限是針對工具授予的，而不是針對使用工具建立的特定項目。例如，為通告工具，而不是個別的通告設定權限。

註釋：在資源中，您可以為全部資源設定權限，也可以為資源中的資料夾設定權限。

如需更改工具的權限：

  1. 按一下網站分頁開啟網站。

  2. 在功能表欄中，按一下您想要為其設定權限的工具。

  3. 按一下主視窗頂部的權限按鈕。

  4. 選中或取消選中核取方塊以授予權限。

  5. 按一下儲存。如需更改選擇，按一下取消。

註釋：設定另一個工具的權限前，您必需按一下儲存或取消退出權限功能。

每種工具都有一套不同的可用權限。如需瞭解關於可用權限的一般概念，請參考下文日程表工具權限清單：

  • new : 新建 – 建立新項目

  • delete : 刪除 – 刪除項目

  • import : 匯入 – 匯入項目

  • read : 閱讀 – 閱讀項目

  • revise : 修改 – 修改項目

  • all.groups : 全部群組 – 檢視與任何群組相關聯的發佈
