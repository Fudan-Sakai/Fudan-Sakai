Modifier les rôles des participants à partir de la Configuration de Sites

Note: Pour compléter les instructions contenues dans ce document, vous devez avoir un rôle ayant les droits d'accès requis. Pour déterminer votre rôle, suivez les instructions dans Rôles du participant. Pour une compréhension générale des rôles et droits d'accès, consultez Droits d'accès et rôles: Vue d'ensemble.

À partir de la Configuration de Sites dans Zone de Travail, vous pouvez modifier les rôles que vous attribuez aux participants du site. Pour ce faire, suivez les étapes suivantes:

  1. Dans le menu de Zone de Travail, cliquez Configuration de Sites.

  2. Cochez la case correspondant au site dans lequel vous désirez modifier les rôles des participants, puis cliquez Réviser.

    Note: Même si vous pouvez cocher plusieurs cases, vous ne pouvez réviser que les outils d'un site à la fois.

  3. Sous l'information de votre site, vous verrez la liste de ses participants, qui contient le nom de chaque participant, leur rôle et statut. Pour chacun des participants que vous désirez modifier, sélectionnez le(s) rôle(s) du (des) participant(s), puis cliquez Mettre à jour participants.
