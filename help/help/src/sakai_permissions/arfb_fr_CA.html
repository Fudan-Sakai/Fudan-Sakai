Modifier les rôles du participant

Note: Pour compléter les instructions contenues dans ce document, vous devez avoir un rôle ayant les droits d'accès requis. Pour déterminer votre rôle, suivez les instructions dans Rôles du participant. Pour une compréhension générale des rôles et droits d'accès, consultez Droits d'accès et rôles: Vue d'ensemble.

Pour modifier les rôles du participant, suivez ces étapes:

  1. Dans le menu du site de cours ou du site de projet pertinent, cliquez Info du site.

  2. À côté du(des) participant(s) dont vous voulez modifier le(s) rôle(s), sélectionnez le(s) rôle(s) approprié(s) à partir de la liste déroulante sous "Rôle".

  3. Cliquez Mettre à jour participants.

Pour plus d'information sur les rôles disponibles, consultez Rôles du participant.
