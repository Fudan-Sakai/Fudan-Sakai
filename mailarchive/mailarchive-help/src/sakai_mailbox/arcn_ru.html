Удаление писем из Архива электронной почты

Внимание: Для того чтобы совершать все эти действия, вам должна быть назначена роль с соответствующими полномочиями. Для того чтобы определить, какая у вас роль, перейдите на страницу Роли участников. Основные сведения о ролях и правах доступа изложены также в Права доступа и роли: обзор.

Чтобы удалить сообщение из Архива электронной почты:

  1. На линейке инструментов нажмите Архив электронной почты.

  2. Нажмите на заголовок письма, которое вы хотите удалить, чтобы открыть его, а затем нажмите Удалить.

  3. Нажмите Удалить для подтверждения удаления.
