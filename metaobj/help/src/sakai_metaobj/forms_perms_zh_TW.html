更改權限

全部網站參與者可以在資源中填寫用於學檔、矩陣和精靈的表單。但是，使用表單工具建立、更改、刪除、匯出和公開表單的權限則受角色的限制。如需設定權限，存取表單工具，並遵循下列步驟進行操作：

  1. 如果未顯示表單功能表欄，按一下重設按鈕。
  2. 按一下功能表欄上的權限按鈕。Sakai 隨後顯示權限螢幕。
  3. 請參考下表，選中相應的核取方塊為每個角色的使用者選擇適當的權限。

    權限 描述
    create - 建立 該角色的使用者可建立和匯入表單，以便自己和／或網站參與者使用。
    edit - 編輯 該角色的使用者可更改表單的名稱和說明。
    export - 匯出 該角色的使用者可以壓縮檔案的形式將表單匯出至桌面。
    delete - 刪除 該角色的使用者可刪除表單。
    publish - 公開 該角色的使用者可公開表單，以便網站參與者使用。
    suggest.global.publish - 建議全域公開 該角色的使用者可建議將表單全域公開（即在 CLE 內的全部網站中公開）。

  4. 如需儲存權限設定，按一下儲存。Sakai 根據上述操作儲存設定，並再次顯示表單首頁。或按一下取消返回螢幕，不儲存更改。
