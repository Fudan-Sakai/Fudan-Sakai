Поиск сообщений

Для выполнения тестового поиска по сообщениям:

  1. На линейке инструментов вашего сайта нажмите на Сообщения.

  2. Нажмите на папку, в которой вы хотите выполнить поиск.

  3. Рядом с надписью "Поиск текста" введите слова для поиска и нажмите на кнопку Поиск. При обычном поиске слова будут найдены только в теме сообщений, хранящихся в этой папке. Поиск чувствителен к регистру букв.

  4. Для изменения параметров поиска нажмите на Расширенный поиск и укажите соответствующие параметры.

    • Тема - тема сообщения
    • Автор - отправитель сообщения
    • Основная часть: - текст сообщения
    • Приоритет - приоритет сообщения
    • Диапазон дат - диапазон дат, когда сообщение было отправлено

  5. Параметры расширенного поиска будут остваться выбранными и при последующем выполнении поиска. Для того чтобы отменить поиск и очистить параметры, нажмите на Очистить поиск.

  6. Для того чтобы вернуться к обычному поиску, нажмите на Обычный поиск.
