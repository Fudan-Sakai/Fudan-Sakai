Просмотр и управление сообщениями

Инструмент "Сообщения" предназначен для обмена личными сообщениями между участниками или блоками участников на сайте. Сообщения размещены по стандартным папкам Входящие, Отправленные и Удаленные.

Основными функциями в инстурменте "Сообщеняи" являются следующие:

  • Для того чтобы написать сообщение, нажмите на Написать сообщение. Подробнее об этом см. Отправка, пересылка и ответ на сообщения.

  • Для того чтобы создать новую папку, нажмите на Новая папка.

  • Для того чтобы пересылать ваши сообщения на адрес электронной почты, нажмите на Настройки. Далее в пункте "Автоматически пересылать сообщения" выберите ответ Да. Введите адрес электронной почты в текстовом поле и нажмите на кнопку Сохранить сообщения.

  • Общее количество сообщений и число не прочитанных сообщений указано около названия каждой папки. Для того чтобы просмотреть письма в папке, нажмите на ее название.

Внутри папки вы увидете список сообщений, упорядоченных по дате.

  • Для того чтобы отсортировать сообщения по какому-либо признаку, нажмите на заголовок соответствующего столбца: Тема, Автор, Дата и Приоритет.

  • Для того чтобы изменить вид сообщений, в выпадающем списке "Просмотр" выберите либо Все сообщения либо Переписка.

  • Для просмотра другой папки нажмите на ссылку Предыдущая папка или Следующая папка.

  • Для того чтобы написать новое сообщения, нажмите на ссылку Написать сообщение. Подробнее об этом см. Отправка, пересылка и ответ на сообщения.

  • Для того чтобы просмотреть сообщение целиком, нажмите на его тему.

  • Для того чтобы переместить сообщение в другую папку, отметьте сообщения для перемещения и нажмите на ссылку Переместить. Выберите папку, в которую будут перемещены сообщения, и нажмите на кнопку Переместить сообщения.

  • Для того чтобы удалить сообщения, отметьте сообщения для удаления и нажмите на ссылку Удалить. В результате сообщения будут перемещены в папку Удаленные. Для того чтобы удалить сообщения навсегда, откройте папку Удаленные, выберите сообщения, чтобы стереть их, нажмите на Удалить и затем нажмите на кнопку Удалить сообщения для подтверждения стирания их.

В сообщении вы увидете текст сообщения и информацию о сообщении.

  • Для того чтобы просмотреть другое сообщение, нажмите на ссылку Предыдущее сообщение или Следующее сообщение.

  • Для того чтобы ответить на сообщение, нажмите на кнопку Ответить. Подробнее об этом см. Отправка, пересылка и ответ на сообщения.

  • Для того чтобы переслать сообщение, нажмите кнопку Переслать. Подробнее об этом см. Отправка, пересылка и ответ на сообщения.

  • Для того чтобы переместить сообщение в другую папку, нажмите на кнопку Переместить в папку. Выберите папку и нажмите на кнопку Переместить сообщения.

  • Для того чтобы удалить сообщение, нажмите на кнопку Удалить. Для подтверждения удаления опять нажмите Удалить.
