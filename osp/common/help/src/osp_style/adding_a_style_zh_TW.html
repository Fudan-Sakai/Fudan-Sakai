新增樣式

如果您是系統管理員，那麼提醒您，您所新增至 Portfolio Admin 網站中的任何樣式都可套用於 CLE 執行個體中的全部網站。

如需為網站新增新的樣式，請遵循下列步驟進行操作：

  1. 將樣式的 .css 檔案新增至資源中。
  2. 存取樣式工具。
  3. 如果未顯示樣式首頁，按一下重設按鈕。
  4. 在螢幕頂部的功能表欄中，按一下「新增」按鈕。Sakai 隨後顯示新增樣式螢幕。
  5. 在名稱方塊中，為樣式輸入一個描述性的名稱。
  6. 如有需要，在描述方塊中輸入完整描述。
  7. 按一下 CSS 檔案方塊右邊的選擇檔案連結。
  8. 根據選擇項目說明中的操作步驟選擇您所需要的 .css 檔案。按一下「繼續」後，Sakai 再次顯示新增樣式螢幕。您所選擇的檔案名稱將顯示在方塊中。
  9. 如需完成操作，按一下「新增樣式」。Sakai 隨後再次顯示樣式首頁。新建樣式已新增至清單中，以便網站參與者使用。或按一下「取消」返回首頁，不新增樣式。
