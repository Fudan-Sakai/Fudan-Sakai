樣式：概述

樣式是一套字體、顏色和圖形設定。網站組織者可使用樣式控制矩陣儲存格、精靈和精靈頁面的外觀。網站參與者可使用樣式控制他們自己所設計的範本的外觀。

樣式使用指導為了保持矩陣或精靈內部的統一，建議網站組織者在矩陣或的全部儲存格或精靈的全部頁面中套用同樣的樣式。學檔無須保持這種統一性。設計自己的學檔的網站參與者可能需要使用多個樣式。擁有多種樣式可供使用對於這些使用者來說大有幫助。

如果您是網站組織者，您可以從資源上載串接樣式表（即 .css 檔案），然後給樣式起一個有含義的名字，並且（如果需要）輸入樣式的描述，透過上述步驟使用樣式工具建立樣式。使用樣式工具還可以更改現有樣式的名稱和描述，刪除您不再需要的樣式，並且更改任何角色的全部使用者的使用權限。

一旦在網站內或使用 Portfolio Admin 網站全域建立樣式，那麼網站內的全部成員即可使用該樣式。

樣式工具通常位於學檔網站的設計和公開類別中。如需存取樣式工具，按一下樣式啟動按鈕。CLE 隨後顯示樣式首頁。

此頁面包含您在網站中所建立的全部樣式，以及已經公開以便全部成員使用的樣式的清單。對於清單中的每個樣式，您可以檢視描述、樣式建立人的姓名，以及該樣式是否已建議公開。

如果一個頁面無法顯示完整的樣式清單，那麼使用樣式功能表欄下方的按鈕顯示清單中的其他頁面。

  • 如需顯示清單中的第一頁，按一下第一頁。
  • 如需顯示清單中的上一頁，按一下上一頁。
  • 如需顯示清單中的下一頁，按一下下一頁。
  • 如需顯示清單中的最後一頁，按一下最後一頁。
