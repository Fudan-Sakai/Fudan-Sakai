更改權限

全部有權操作矩陣的參與者都可以顯示術語表定義各列和各欄的名稱。但是，新增、刪除、更改和匯出術語表術語的權限卻受角色限制，正如 Sakai 其他功能的權限一樣。

如需設定權限，存取術語表工具，並遵循下列步驟進行操作：

  1. 如果無法看到術語表功能表欄，按一下「重設」按鈕。
  2. 按一下功能表欄上的「權限」連結。Sakai 隨後顯示下列螢幕：
  3. 請參考下表，按一下相應的核取方塊為每個角色的使用者選擇適當的權限。



    權限 描述
    刪除 (Delete) 該角色的使用者可刪除術語表術語。
    新增 (Add) 該角色的使用者可新增和匯入術語表術語。
    編輯 (Edit) 該角色的使用者可更改術語表術語及其定義。
    匯出 (Export) 該角色的使用者可將術語表術語匯出為壓縮檔案。


  4. 如需儲存權限設定，按一下「儲存」。Sakai 根據上述操作儲存設定，並再次顯示術語表首頁。或按一下「取消」返回螢幕，不儲存更改。
