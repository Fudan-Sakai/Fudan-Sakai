更改權限

評估者的權限設定比網站其他成員要更為複雜。為了使您能夠作為評估者操作特定的矩陣儲存格和精靈頁面，必需滿足下列條件：

  • 您在該網站必需擁有評估者角色。
  • 在特定的矩陣儲存格或精靈頁面中，您的角色或您的使用者名必需指定為評估者。
  • 您的角色在矩陣或精靈工具和評估工具中都必需具備「評估」權限。

如需更改評估工具的預設權限，存取評估工具，並遵循下列步驟進行操作：

  1. 如果未顯示評估功能表欄，按一下「重設」按鈕。
  2. 按一下功能表欄上的「權限」按鈕。Sakai 隨後顯示權限螢幕。
  3. 請參考下表，選中相應的核取方塊為每個角色的使用者選擇適當的權限。



    權限 描述
    檢視擁有人 (View Owner) 該角色的使用者可以檢視正在透過矩陣儲存格或精靈頁面對其作業進行評估的使用者的姓名。
    評估 (Evaluate) 該角色的使用者可在矩陣和精靈中檢視網站參與者已完成的作業，並為作業新增回饋資訊。


  4. 如需儲存權限設定，按一下「儲存」。Sakai 根據上述操作儲存設定，並再次顯示評估首頁。或按一下「取消」返回螢幕，不儲存更改。
