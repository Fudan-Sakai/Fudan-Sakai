更改權限

如需更改矩陣工具的預設權限，存取矩陣工具，並遵循下列步驟進行操作：

  1. 如果未顯示矩陣功能表欄，按一下「重設」按鈕。
  2. 按一下功能表欄上的「權限」按鈕。Sakai 隨後顯示權限螢幕。
  3. 請參考下表，選中相應的核取方塊為每個角色的使用者選擇適當的權限。



    權限 描述
    使用 (Use) 該角色的使用者可存取矩陣，使用矩陣來收集關於他們工作的資訊、資料，並反映到工作中。
    建立 (Create) 該角色的使用者可建立新矩陣和匯入現有矩陣，以便自己和/或網站參與者使用。
    公開 (Publish) 該角色的使用者可公開矩陣，以便網站參與者使用。
    匯出 (Export) 該角色的使用者可以壓縮檔案的形式將矩陣匯出至桌面。
    檢閱 (Review) 該角色的使用者可在矩陣中檢查正在進行中的作業（使用下拉式功能表轉到任何網站參與者的矩陣），並為作業提供格式化的回饋意見。
    評估 (Evaluate) 該角色的使用者可在矩陣中檢視已完成的作業，並為作業新增回饋資訊。


  4. 如需儲存權限設定，按一下「儲存」。Sakai 根據上述操作儲存設定，並再次顯示矩陣首頁。或按一下「取消」返回螢幕，不儲存更改。
