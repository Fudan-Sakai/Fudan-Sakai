匯出和匯入

如果您擁有多個網站，那麼您可能需要將同一矩陣套用於多個網站。如需進行上述操作，先將精靈以壓縮檔案形式匯出至桌面，然後將其匯入您想要套用的目標網站。您還可以在公開匯入的矩陣前對其進行編輯，以便滿足新網站的需要。

匯出矩陣

如需將矩陣匯出為壓縮檔案，存取矩陣工具，並遵循下列步驟進行操作：

  1. 如果未顯示矩陣首頁，按一下「重設」按鈕。
  2. 找到您想要匯出的矩陣。
  3. 按一下矩陣名稱下方的「匯出」連結。Sakai 隨後顯示開啟檔案視窗。
  4. 遵循說明進行操作。Sakai 建立壓縮檔案，並將其下載至桌面上。
  5. 將壓縮檔案上載至資源。

匯入矩陣

如需從桌面匯入矩陣壓縮檔案，存取您想要匯入該矩陣的網站，並遵循下列步驟進行操作：

  1. 將檔案新增至資源。
  2. 存取矩陣工具。如果未顯示矩陣功能表欄，按一下「重設」按鈕。
  3. 按一下功能表欄上的「匯入」按鈕。Sakai 隨後顯示匯入矩陣螢幕。
  4. 按一下「選擇檔案…」。Sakai 隨後顯示選擇項目螢幕。
  5. 遵循選擇項目說明選擇矩陣壓縮檔案。按一下「繼續」後，Sakai 再次顯示匯入矩陣螢幕。您所選擇的檔案的名稱將顯示在匯入檔案方塊中。
  6. 按一下「匯入」按鈕。Sakai 從資源區匯入檔，並再次顯示矩陣首頁。匯入的矩陣將顯示在首頁中。該精靈的名稱和說明與原網站中的相同。或按一下「取消」返回首頁，不將檔案匯入矩陣工具。無論矩陣在原網站中處於何種狀態（公開或未公開），您所匯入的矩陣在新網站中將處於未公開狀態。您還可以在公開匯入的矩陣前對其進行編輯，以便滿足新網站的需要。
