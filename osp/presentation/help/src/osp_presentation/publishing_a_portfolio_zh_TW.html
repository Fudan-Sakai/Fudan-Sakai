公開學檔

建立學檔後，您可以公開學檔以便他人檢視。您可以選擇向網站中的群組、或是網站或 Sakai 環境外部的個人、或是任何人可透過網路瀏覽器存取的 URL 公開學檔。

如需公開學檔，存取學檔工具，並遵循下列步驟進行操作：

  1. 如果未顯示學檔首頁，按一下「重設」按鈕。
  2. 找到您想要公開的學檔工具，並按一下該學檔名稱下的「公開」連結。Sakai 隨後顯示選擇讀者螢幕，該螢幕包含兩個部分：向群組或個人公開部分以及向網路公開部分。
    • 各部分標題中的右箭頭表示該部分已關閉。如需開啟該部分以便檢視其中的欄位框，按一下箭頭。
    • 各部分標題中的下箭頭表示該部分已開啟（即顯示該部分中的欄位方塊）。如需關閉，按一下箭頭。
  3. 如需向全部擁有特定角色的網站成員公開學檔和/或向特定網站成員公開學檔，請遵循下列步驟進行操作：
    • 在向群組或個人公開部分中，選擇角色/姓名方塊中適當的角色或姓名。（先列出角色，然後是姓名。您需要向下捲動頁面才能看到全部姓名。）或在按一下多個角色和／或姓名的同時，按住 CTRL 鍵。
    • 按一下「新增」按鈕。Sakai 在所選讀者方塊中顯示您剛才選擇的角色和姓名。
  4. 如需與非網站成員共用學檔，在個人電子郵箱方塊中輸入該使用者的電子郵件地址，然後按一下「新增」。Sakai 將該地址新增至所選讀者方塊中。為您想要新增的每個電子郵件地址重複該步驟。
  5. 如需將您的學檔發佈到網路上，選擇向網路公開部分中的公開 URL 方塊。Sakai 然後在右側的方塊中顯示該網站學檔的預設 URL。如需指定另一個 URL，更改方塊中的內容。注意如果選中公開 URL 方塊，那麼存取網路的任何人都可以看到您的學檔。
  6. 選擇學檔讀者後，選擇下列選項中的一項：
    • 如需公開學檔，並通知讀者學檔已備妥可供瀏覽，按一下「完成和通知」。Sakai 隨後顯示下列螢幕：
    • 如需公開學檔但不通知讀者，按一下「完成」。Sakai 根據上述操作公開學檔，並顯示學檔首頁。
    • 如需返回至 3 步驟中的第 2 步驟：設計螢幕，按一下「返回」。
    • 或按一下「取消」返回首頁，不公開學檔。
  7. 如果按一下「完成和通知」，那麼如下所述填寫通知讀者螢幕中的欄位：
    • 選中您想要通知的每個角色或個人使用者相對應的核取方塊。
    • 在您的訊息欄位中，輸入您想要包含在電子郵件中的任何訊息。
  8. 如需將郵件傳送至所選使用者，按一下「傳送」。Sakai 根據上述操作傳送郵件，並再次顯示學檔首頁。或按一下「取消」返回首頁，不公開學檔，也不傳送郵件。

重要事項按一下「傳送」後，在您所選讀者清單中的非網站使用者將收到兩封郵件。第一封郵件通知使用者已經在網站上為他們設定了一個帳戶，並且提供存取帳戶的登入 ID 和密碼。第二封郵件包含存取該網站中您所建立學檔所需的資訊。這些使用者必須輸入兩封郵件中的相關內容才能瀏覽您的學檔。而在您的讀者清單中的網站成員只會收到一封郵件。該郵件通知他們您的學檔已備妥可供瀏覽，並提供存取您的學檔所需的資訊。
