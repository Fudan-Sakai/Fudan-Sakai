Deleting a Layout

You may delete a portfolio layout only if it is not being used in any portfolios on the site. If a layout is in use, you must delete all portfolios that use it before deleting the layout, itself.

To delete a portfolio layout, access the Portfolio Layouts tool and follow these steps:

  1. If the Portfolio Layouts home page is not displayed, click the Reset button.
  2. Find the layout you want to remove and click the Delete link below its name. Sakai displays a delete confirmation window.
  3. To delete the layout, click OK. Sakai deletes the layout and displays the Portfolio Layouts home page. Or, to return to the home page without deleting this layout, click Cancel.
