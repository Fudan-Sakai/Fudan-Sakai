Обратная связь: обзор

Для чего нужен инструмент "Обратная связь"

Используя инструмент "Обратная связь", Вы можете загружать электронные таблицы в файлах формата СSV, где разделителями являются запятые, для отправки отзывов участникам сайта.

Инструмент "Обратная связь" обладает следующим набором возможностей:

  • Показывает отзыв индивидуально участнику, которому он предназначен

  • Показывает отзывы на всех участников руководителю сайта

  • Позволяет руководителю сайта просматривать отзыв отдельному участнику

  • Подсчитывает и показывает статистику любых численных данных

  • Обновляет существующий файл отзыва

  • Загружает несколько файлов отзывов

Основные понятия

Формат электронных таблиц: Чтобы электронная таблица была совместима с инструментом "Обратная связь", ее необходимо сохранить в файле формата .csv. Первый столбец должен содержать имена пользователей, а первая строка - заголовки. В других графах должны содержаться данные в виде текста или цифр.

Важные замечания

  • Руководитель сайта проекта может использовать инструмент "Обратная связь" для предоставления отзывов членам сайта об их вкладе в отчет, делать доступным отзывы других членов сайта,ставить задачи и отслеживать их выполнение каждым отдельным участником.

  • Руководитель сайта курса может использовать инструмент "Обратная связь" для предоставления отзывов студентам об их успеваемости на занятиях. Отзыв может включать оценки, проценты, буквенные оценки и/ или комментарии.
