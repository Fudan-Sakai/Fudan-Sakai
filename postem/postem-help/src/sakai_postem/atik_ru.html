Использование инструмента "Обратная связь"

Внимание: Для выполнения этих действий Вам должна быть назначена роль с соответствующими полномочиями. Для того чтобы определить, какая у Вас роль, перейдите на страницу Роли участников. Основные сведения о ролях и правах доступа изложены также в Права доступа и роли: обзор.

Вы можете использовать инструмент "Обратная связь" для создания нескольких зачетных книжек, загрузки таблицы оценок в Excel или просто отправки комментариев.

Внимание: Инструмент "Обратная связь" скрыт по умолчанию. Для того чтобы добавить инструмент "Обратная связь" на сайт, перейдите на страницу Добавить, изменить или удалить инструменты.

Создать файл для отправки

Чтобы инструмент "Обратная связь" импортировал файлы, необходимо выполнить три требования:

  • Фаш файл должен быть сохранен в формате .csv. Вы можете сохранять таблицы Microsoft Excel как файлы с расширением .csv.

  • В первом столбце файла должны располагаться имена пользователей.

  • В первой строке должны быть заголовки. У каждого столбца должен быть заголовок.

Создав файл, вы можете изменять его при условии, что вышеописанные требования выполняются.

Отправить файл

Чтобы отправить файл:

  1. На линейке инструментов сайта, нажмите Обратная связь.

  2. Нажмите Добавить.

  3. В поле "Заголовок" введите название файла (максимальная длина - 30 знаков).

  4. В поле "Выбрать файл" нажмите Обзор , и найдите ваш файл с расширением .csv.

  5. Если Вы хотите сделать файл с отзывом доступным участникам (т.е. разрешить студентам посмотреть их оценки), отметьте флажком Отправить отзывы участникам?.

  6. Для сохранения файла нажмите Отправить; для отмены отправки нажмите Отмена.

  7. На странице "Проверить данные загружаемого файла" вы увидите пример того, что представляет из себя отправляемый файл. Чтобы продолжить и отправить файл, нажмите Сохранить. Чтобы вернуться на страницу загрузки, нажмите Назад.

Для загрузки дополнительных файлов повторите описанный выше алгоритм действий.
