Downloading, updating, or deleting a Post'Em file

Note: To complete these procedures, you must be assigned a role having the necessary permissions. To determine your role, follow the directions in Participant roles. For a basic understanding of roles and permissions, see Permissions and roles: Overview.

Once you have posted a file in Post'Em, you can download, update, or delete the file.

On this page:

  • Downloading a posted file
  • Updating a posted file
  • Deleting a posted file

Downloading a posted file

  1. In your site's menubar, click Post'Em.

  2. Find the file you want, and click Download. If prompted, save the file to your computer.

Updating a posted file

Note: A Post'Em file must meet the format requirements; see Using the Post'Em tool.

  1. In your site's menubar, click Post'Em.

    Note: Post'Em is hidden by default. To add Post'Em to your site, see Adding, removing, reordering, renaming, or hiding tools.

  2. Find the file to update, and click Update.

  3. In the field next to "Title", you can change the title if needed.

  4. In the field next to "Choose a file", click Browse... and navigate to your new .csv file.

  5. If you wish to release the feedback to participants (i.e., allow the students to see the grades), check Release feedback to participants?Â . This option may already be selected based on the original file.

  6. To save your file, click Post; to cancel, click Cancel.

  7. On the Verify Upload page, you can preview how the file will appear. To continue and post the file, click Save. To return to the upload screen, click Back.

Deleting a posted file

  1. In your site's menubar, click Post'Em.

  2. Find the file you want to delete, and click Delete.

  3. To confirm the deletion, click Delete. To cancel the deletion, click Cancel.
