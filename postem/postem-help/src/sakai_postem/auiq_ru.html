Скачать,обновить или удалить файл "Обратной связи"

Внимание: Для выполнения этих действий Вам должна быть назначена роль с соответствующими полномочиями. Для того чтобы определить, какая у Вас роль, перейдите на страницу Роли участников. Основные сведения о ролях и правах доступа изложены также вПрава доступа и роли: обзор.

После отправки файла "Обратной связи" вы можете скачать, Вы можете скачать, обновить или удалить его.

На этой странице:

  • Скачать отправленный файл
  • Обновить отправленный файл
  • Удалить отправленный файл

Скачать отправленный файл

Чтобы скачать отправленный файл:

  1. На линейке инструментов нажмите Обратная связь.

  2. Найдите нужный файл, затем нажмите Скачать. Сохраните файл на компьютере.

Обновить отправленный файл

Чтобы обновить отправленный файл:

Внимание: Файл "Обратной связи" должен соответствовать требованиям, предъявляемым к формату файлов. смотрите Использование инструмента "Обратная связь".

  1. На линейке инструментов нажмите Обратная связь.

    Внимание: Обратная связь скрыта по умолчанию. Чтобы добавить инструмент "Обратная связь" на сайт, смотрите страницу Добавить, изменить или удалить инструменты.

  2. Найдите файл, который нужно обновить и нажмите Обновить.

  3. В поле "Заголовок" по необходимости можно изменить заголовок.

  4. В поле "Выбрать файл" нажмите Обзор и найдите новый файл с расширением .csv.

  5. Если Вы хотите сделать отзыв доступным участникам(т.е. разрешить студентам посмотреть их оценки)отметьте флажком Отправить отзывы участникам?. Эта настройка может быть уже задана в настройках к оригинальному файлу.

  6. Чтобы сохранить файл, нажмите Отправить; для отмены нажмите Отмена.

  7. На странице "Проверить данные загружаемого файла" дается предварительный просмотр файла. Чтобы продолжить и отправить файл, нажмите Сохранить. Чтобы вернуться на страницу загрузки, нажмите Назад.

Удалить отправленный файл

Чтобы удалить отправленный файл:

  1. На линейке инструментов нажмите Обратная связь.

  2. Найдите файл, который Вы хотите удалить и нажмитеУдалить.

  3. Для пожтверждения удаления нажмите Удалить. Для отмены удаления нажмите Отмена.
