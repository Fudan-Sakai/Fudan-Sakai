These are typical example files that Sakai users (students, instructors)
might upload into the resource tool.  They are used by the JMeter script 
that uploads files into the resource tool.

These files were created and collected by Michael Beasley and put up on
the "CTools QC Team" (Quality Control) site:
https://ctools.umich.edu/portal/site/1097769640526-2853114/page/1097769640654-2853117
