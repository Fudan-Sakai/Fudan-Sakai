Новый пароль: Обзор

Для чего нужен инструмент "Новый пароль"

Инструмент "Новый пароль" позволяет получить новый пароль для входа в систему, просто введя ваш адрес электронной почты.

Важное замечание

Этот инструмент доступен только для гостевых входов в систему, то есть для пользователей, у которых нет постоянной учетной записи. Если при входе в систему вы вводите имя пользователя, отличное от адреса электронной почты, то вам не удастся получить новый пароль для входа с помощью этого инструмента.
