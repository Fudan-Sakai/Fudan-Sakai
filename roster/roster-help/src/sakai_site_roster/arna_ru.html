Обзор списка студентов

Чтобы просмотреть список студентов, на линейке инструментов нажмите Список студентов.

Внимание: Если Вы не видите "Список студентов" на линейке инструментов, это значит, что владелец сайта решил не использовать этот инструмент. Владельцы сайтов могут добавить инструмент "Список студентов" при помощи настроек. Соответствующие инструкции смотрите на странице Добавить, изменить или удалить инструменты.

Обзор

По умолчанию список участников составлен в алфавитном порядке. Число участников отображается справа над страницей.

Чтобы настроить обзор:

  • Используйте выпадающий список рядом с "Показывать" для выбора групп или блоков.

  • Чтобы отсортировать список в алфавитном порядке по фамилии, нажмите Имя. Чтобы отсортировать в обратном порядке, нажмите Имя еще раз.

  • Чтобы отсортировать список в алфавитном порядке по имени пользователя, нажмите ID пользователя. Чтобы отсортировать в обратном порядке, нажмите ID пользователя еще раз.

  • Чтобы отсортировать список в алфавитном порядке по роли, нажмите Роль. Чтобы отсортировать в обратном порядке, нажмите Роль еще раз.

Для просмотра профилей участников, нажмите на имя участника.

Изображения участников

Участники сайта могут просматривать изображения (т.е. фотографии или аватары), установленные другими участниками в их профилях. Для этого вверху страницы "Список студентов" нажмите Фотографии.

Вы можете использовать выпадающее меню Показывать, чтобы выбрать, каких участников показывать: всех или только участников определенного блока или группы. Вы можете найти участника по "Имени" или "ID пользователя". Обладая соответствующими полномочиями, Вы можете выбрать различные варианты просмотра Фотография в студенческом билете или Фотографии из профиля. Также можно выбрать функцию Скрыть имена.
