Создание вопросов к тесту

Внимание: Для того чтобы совершать все эти действия, вам должна быть назначена роль с соответствующими полномочиями. Для того чтобы определить, какая у вас роль, перейдите на страницу Роли участников. Основные сведения о ролях и правах доступа изложены также в Права доступа и роли: обзор.

Создание вопросов - только один из шагов, связанных в разработкой теста. Более подробную информацию о других аспектах разработки теста смотри в:

  • Создание нового теста
  • Настройки теста

Вожно выбрать один из нескольких типов вопроса во время разработки теста. Перед тем, как добавлять вопросы, ознакомьтесь с различными доступными типами вопросов, которые перечислены в Типы вопросов в тесте.

На линейке инструментов нужного сайта курса или проекта нажмите Тесты и экзамены. Чтобы создать новый тест под надписью "Создать новый тест" введите название теста в текстовом поле "Название". Можно использовать выпадающий список рядом с Выберите один из существующих типов тестов, чтобы выбрать тип теста. Можно также импортировать тест с помощью функции Импортировать тест. Более подробную информацию смотри в Импорт и экспорт тестов. Когда вы выбрали, каким способом нужно создать тест, нажмите Быстрое создание или создать, чтобы создать тест.

Опция Быстрое создание позволяет вводить вопросы теста или вставлять их из текстового файла. Нужно следовать инструкциям о формате текста внизу страницы.

Опция Создать открывает редактор вопросов. Чтобы начать добавление вопросов к новому тесту, выберите тип вороса, который нужно добавить из выпадающего списка Добавить попрос. Ниже вы найдете подробное описание каждого типа вопроса.

  • Выбор вариантов
  • Опрос/анкетирование
  • Развернутый ответ
  • Заполнить пропуски
  • Численный ответ
  • Парное соответствие
  • Истина/Ложь
  • Аудиозапись ответа
  • Загрузка файла
  • Копировать из фонда вопросов

Выбор вариантов

Чтобы создать вопрос с выбором вариантов:

  1. На странице Воспросы, из выпадающего списка Добавить тип вопроса выберите Выбор вариантов ответов. Откроется страница редактирования.

  2. В текстовом поле "Баллы" введите количество баллов для оценки вопроса. Введите 0 (ноль), если нужно, чтобы вопрос не оценивался.

  3. В текстовом поле "Формулировка вопроса" введите текст вопроса. Нажмите Показать/скрыть панель форматирования, чтобы открыть панель инструментов редактора WYSIWYG.

  4. Если нужно добавить приложение, нажмите Добавить приложения, затем ведите или укажите путь к файлу или URL удаленного ресурса.

  5. В разделе "Ответ:" выберите с помощью переключателя Единственный правильный ответr (по умолчанию) или Несколько правильных вариантов ответов. После тогоЮ как вы сделали выбор, страница обновится, чтобы показать или скрыть необходимые текстовые поля.

  6. В случае, если вы выбрали единственный правильны ответ, откроется четыре тектовых поля для ввода вариантов ответа и инструмент для добавления больего количества вариантов. Чтобы указать, какой вариант ответа правильный, нажмите на переключатель рядом с ним. В случае, если вы выбрали вариант с несколькими правильными ответами, отметьте флажком правильные ответы.

    Если нужно показать меньше вариантов ответа, нажмите Удалить под текстовыми полями, варианты которых нужно удалить. Если нужно предоставить больше вариантов, используйте выпадающий список рядом с Добавить еще варианты ответа, чтобы выбрать число вариантов, которые нужно добавить.

  7. Если нужно, чтобы варианты появлялись в случайном порядке, под надписью "Случайный порядок вариантов ответа" нажмите Да. Чтобы ответы появлялись в том порядке, в каком они находятся на экране (по умолчанию), нажмите Нет.

  8. Если нужно, чтобы пользователи обосновали свой ответ, под надписью "Потребовать обоснование ответа" нажмите Да. Если обоснование ответа не требуется (по усолчанию), нажмите Нет.

  9. Из выпадающего списка Включить в раздел выберите раздел (например, группу), которой нужно адресовать вопрос. Оставьте эту опцию по умолчанию, если еще не было создано никаких групп. t

  10. Если нужно добавить вопрос в фонд вопросов, из выпадающего списка Включить в фонд вопросов выберите название необходимого фонда.

  11. Под надписью "Отзыв на правильный/неправильный ответ" можно указать ответы системы, которые будут даваться пользователю в после выбора правильного или неправильного варианта.

  12. Когда вы закончили, нажмите Сохранить, чтобы венуться к экрану редактирования теста. Вы увидите, что новые вопросы добавлены к тесту.

Опрос/анкетирование

Процесс создания вопроса "Опрос/анкетирование" идентичен поцессу создания вопроса с выбором варианта ответа, кроме нескольких различий:

  • Значение поля "Баллы" по умолчанию 0 (ноль).

  • Чтобы ввести текст возможных вариантов ответа, нужно выбрать один из набора вариантов, наиболее часто используемых в опросах или анкетах. Чтобы выбрать набор вариантов, который нужно использовать, нажмите на соответствующий переключатель. Доступны следующие наборы вариантов:

    • Да, Нет
    • Не согласен, Согласен
    • Не согласен, Затрудняюсь ответить, Согласен
    • Ниже среднего -> Выше среднего
    • Категорически не согласен -> Полностью согласен
    • Категорически не согласен -> Полностью согласен
    • 1 -> 5
    • 1 -> 10

  • Отсутствует опция установки случайного порядка вопросов или запроса обоснования.

Заполнить пропуски

Чтобы создать вопрос типа "Заполнить пропуски":

  1. На странице "Вопрос" в выпаающем списке Добавить вопрос выберите Заполнить пропуски.

  2. В текстовом поле "Баллы" введите количество баллов для оценки вопроса. Введите 0 (ноль), если нужно, чтобы вопрос не оценивался.

    Внимание: Общее количество баллов, которое устанавливается, разделяется поровну между правильными ответами на заполненные пропуски. Например, возьмем следующий вопрос:

    Розы {красные}, а фиалки {синие}, но не всегда.

    Если вы оценили вопрос в 10 баллов, студент получит 5 баллов за каждый заполненный пропуск : "красные" и "синие"

  3. В текстовом поле "Формулировка вопроса" введите текст вопроса. Нажмите Показать/скрыть панель форматирования, чтобы открыть панель инструментов редактора WYSIWYG. Поставьте слова, которые будут пропущены, в фигурные скобки ( { } ), например "Дерни, деточка, за веревочку, {дверь} и откроется".

    Другие опции для создания вопроса с заполнением пропусков включают:

    • Специальный симпол: Вставьте звездочку (*) на место одного или более произвольных символов.
      Пример: В тридевят* ц*стве, в тридесят* гос*ве.
    • Синонимичные варианты ответа: Вставьте вертикальную черту | между эквивалентными (синонимичными) вариантами ответа.
      Пример: Император {Петр I|Петр Первый|Петр Великий}
    • С учетом регистра: Поставьте влажок рядом с надписью С учетом регистра?, если требует учитывать регистр в пропущенных словах. Например: Столица Калифорнии - город {Сакраменто}.
    • Взаимно исключающие варианты: Поставьте флажок рядом с надписью Взаимно исключающие? если у вопроса есть два или более пропуска с одинаковым набором ответов, но студент должен ответить на каждый по-своему. Например: "Стороны монеты называются {орел|решка} или {решка|орел" }

      Если ответы взаимно исключающие, студент должен ответить "орел" в одном пропуске и "решка" в другом, чтобы ответить на вопрос правильно. Если ответы не взаимно исключающие, правильным будет считаться, например, ответ "орел" в обоих пропусках.

      Можно использовать опцию "Взаимно исключающие ответы", когда она используется во все пропусках в вопросе. Например:

      Нужно смешать {красную|синюю} и {красную|синюю} краску вместе, чтобы получить {фиолетовую}.

      Если выбрать опцию "Взаимно исключающие ответы" в этом вопросе, она будет автоматически снята при сохранении вопроса.

  4. Если нужно добавить приложение, нажмите Добавить приложения, затем ведите или укажите путь к файлу или URL удаленного ресурса.

  5. Из выпадающего списка Включить в раздел выберите раздел (например, группу), которой нужно адресовать вопрос. Оставьте эту опцию по умолчанию, если еще не было создано никаких групп.

  6. Если нужно добавить вопрос в фонд вопросов, из выпадающего списка Включить в фонд вопросов выберите название необходимого фонда. .

  7. Под надписью "Отзыв на правильный/неправильный ответ" можно указать ответы системы, которые будут даваться пользователю в после выбора правильного или неправильного варианта.

  8. Когда вы закончили, нажмите Сохранить, чтобы венуться к экрану редактирования теста. Вы увидите, что новые вопросы добавлены к тесту.

Загрузка файла

Чтобы создать вопрос типа "Загрузка файла", в качестве ответа на который пользователи должны загрузить файл:

  1. На странице "Вопрос" в выпаающем списке Добавить вопрос выберите Загрузка файла.

  2. В текстовом поле "Баллы" введите количество баллов для оценки вопроса. Введите 0 (ноль), если нужно, чтобы вопрос не оценивался.

  3. В текстовом поле "Формулировка вопроса" введите текст вопроса. Нажмите Показать/скрыть панель форматирования, чтобы открыть панель инструментов редактора WYSIWYG.


  4. Если нужно добавить приложение, нажмите Добавить приложения, затем ведите или укажите путь к файлу или URL удаленного ресурса.

  5. Из выпадающего списка Включить в раздел выберите раздел (например, группу), которой нужно адресовать вопрос. Оставьте эту опцию по умолчанию, если еще не было создано никаких групп.

  6. Если нужно добавить вопрос в фонд вопросов, из выпадающего списка Включить в фонд вопросов выберите название необходимого фонда. .

  7. Под надписью "Отзыв на правильный/неправильный ответ" можно указать ответы системы, которые будут даваться пользователю в после выбора правильного или неправильного варианта.

  8. Когда вы закончили, нажмите Сохранить, чтобы венуться к экрану редактирования теста. Вы увидите, что новые вопросы добавлены к тесту.

Копировать из фонда вопросов

Чтобы скопировать вопрос из фонда вопросов:

  1. На странице "Вопрос" в выпаающем списке Добавить вопрос выберите Копировать из фонда вопросов.

  2. Нажмите на название фонда, откуда нужно скопировать вопрос.

  3. Перейдите к нужному фонду и/или используйте фладки, чтобы выбрать вопросы, которые нужно вставить в тест, затем нажмите Копировать.
