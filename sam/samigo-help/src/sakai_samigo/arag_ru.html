Настройки теста

Внимание: Для того чтобы совершать все эти действия, вам должна быть назначена роль с соответствующими полномочиями. Для того чтобы определить, какая у вас роль, перейдите на страницу Роли участников. Основные сведения о ролях и правах доступа изложены также в Права доступа и роли: обзор.

Чтобы открыть страницу настроек теста, на линейке инструментов нужного сайта нажмите Тесты и экзамены. На странице со списками тестов нажмите Настройки под названием нужного теста. Нажмите черную стрелку напротив типа настроек, чтобы увидеть доступные возможности настроек.

Внимание: Изменение настроек для одного теста не влияет на настройки остальных тестов. Изменение одних опций не влияет на другие.

Набор доступных настроек зависит от того, был ли опубликован данный тест. Также они зависят от типа созданного теста. Тип теста по умолчанию не включает всех возможных настроек. Когда тест опубликован, можно изменять только настрйки "Сроки предьявления" и настройки, связынные с переносов оценок в инструмент "Зачетная книжка" (в разделе "Проверка").

Если нужно включить больше различных настроек, чем доступно в неопубликованном тесте, нужно будет создать новый тип теста. Более подробные инструкции смотри в Добавление/создание, изменение или удаление типа теста.

Настройки теста

Внимание: Не нужно использовать опцию отзыва системы на ответы в инструменте "Тесты и Экзамены" с тестами на оценку, так как она может предоставить студентам информацию, которую она смогут использовать для исправления ответов перед отправкой теста.

Существуют следующие возможные настройки:

  • Тип теста: Если вы выбрали в качестве основы для теста существующий шаблон, его название, автор(ы) и краткое описание будет отображаться в этой опции.

  • Введение к тесту: Укажите название и автора(ов) теста, а также введите описание или введение к тесту. Автор теста (то есть, пользователь, через учетную запись которого был создан тест) автоматически указан здесь, эту информацию именить нельзя.

  • Сроки предъявления: Укажите даты, когда тест станет открыт для сдачи, когда прием работ будет завершен и когда тест будет изъят из доступа. Чтобы установить даты и время, используйте иконку "Расписание" рядом с каждым текстовым полем.

  • Аудитория теста: Выберите, будет ли тест открыт для анонимных пользователей, или только для участников данного сайта.

    Внимание: Когда тест опубликован, автору будет указан его URL. Автор может по своему усмотрению передавать URL любым лицам, чтобы тест могли пройти также и пользователи. не являющиеся участниками сайта.

  • Ограничение доступа: Установите доступ только с указанных IP адресов, или укажите дополнительный ID пользователя и пароль для доступа к тесту.

  • Ограничение по времени: Можно превратить тест в тест с ограничением по времени и установить время, отпускаемое на тест. Тест будет сдан автоматически, когда отведенное на его выполнение время истечет.

  • Структура и организация теста: Установить вид и способ появления заданий теста. Доступны следующие возможности:

    • Навигация : Выберите одну из настроек:
      • Жесткий порядок ответа на вопросы: последовательный переход от одного вопроса к следующему, вернуться к предыдущим вопросам невозможно, список вопросов недоступен.
      • Свободный порядок ответа на вопросы: позволяет произвольный доступ к вопросам в списке, участники, проходящие тест, могут пользоваться кнопками Следующий и Предыдущий, чтобы ознакомиться со списком вопросов.
    • Предъявление вопросов : Выберите, отображать ли каждый вопрос на отдельной странице или показывать весь тест на однойстранице.
    • Нумерация вопросов : Выберите, установить ли отдельную нумерацию для каждого раздела теста или установить сквозную нумерацию для целого теста.

  • Сдача теста: Укажите количество разрешенных попыток для сдачи теста; используйте переключатель рядом с надписью "Сдача с опозданием" чтобы указать, будут ли тесты приниматься после окончания срока предъявления (более подробно об опции смотри Сроки предъявления). Если было выбрано принимать тесты после окончания сроков предъявления, такие тесты будут выделены во время проверки.

  • Сообщение студенту перед сдачей теста: Введите сообщение или укажите URL, который будет показан студенту перед сдачей теста.

  • Отзыв: Выберите одну из следующих опций: :

    • Немедленный отзыв
    • Отзыв после сдачи (Внимание: лучше использовать вместе с параметром "Проверки" - "Засчитывать последнюю попытку")
    • Отзыв будет предъявлен студенту в определенный день (нажмите на иконку "Расписание", чтобы указать день))
    • Студенту не будет предъявлен отзыв

    Кроме того, можно также использовать комбинацию из нижеследующих опций, чтобы включить в отзыв:

    • Ответ студента
    • Правильный ответ
    • Полученный балл за тест
    • Полученный балл за ответ на вопрос
    • Отзыв на конкретные вопросы
    • Отзыв только на отпреденные варианты ответа
    • Комментарий преподавателя
    • Статистика и гистограммы

  • Проверка: Укажите, будет ли показано авторство работ преподавателю или оценка будет выставляться анонимно, будут ли оценки переноситься в зачетную книжку или нет и как будет выставляться оценка, если разрешены неоднократные сдачи теста. Если вы выбрали анонимную проверку, пользователь, проверяющий тест, не будет видеть ID студента, сдавшего тест.

  • Оформление: Измените оформление теста:

    • Нажмите на иконку палитры рядом с надписью " Цвет фона ", чтобы открыть набор цветов. Используйте выпадающий список, чтобы определить, какой набор цветов необходимо использовать. (Безопасные цвета, Палитра Windows, или Оттенки серого), затем нажмите на цвет, который требуется использовать в фоне теста. Шестнадцатиричное обозначение выбранного оттенка отобразится в текстовом поле "Цвет фона".
    • В текстовом поле "Фоновая картинка" введите URL файла с изображением, которое должно использоваться в качестве фоновой картинки теста.

  • Метаданные: Укажите ключевые слова, цели или рубрики, чтобы облегчить или уточнить задания тестов.

Дополнительно существует возможность сбора метаданных для теста.

Чтобы сохранить настройки, нажмите Сохранить параметры внизу страницы. Нажмите Отмена для отмены всех несохраненных изменений.

Внимание: Тест должен содержать хотя бы один вопрос, чтобы быть опубликованным.
