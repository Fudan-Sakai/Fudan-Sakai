評估設定

註釋：如果要完成本頁中的操作，您必需擁有具有必要權限的角色。如需決定自己的角色，請遵循參與者角色中的說明進行操作。如需瞭解關於角色和權限的基礎知識，請參考權限和角色：概述。

如需進入評估設定頁面，從適當工作網站的功能表欄按一下測驗和小測。在評估頁面中，按一下評估標題下方的設定。按一下設定類型黑色箭頭檢視選項。

註釋：更改某個評估的設定不會對您所建立的其他評估的設定產生影響。更改評估設定中的選項不會對任何其他選項產生影響。

可用設定取決於評估是否已公開。同時還取決於您所建立的評估類型。預設評估類型不包含全部設定。一旦評估公開後，您就只能編輯三個傳送日期設定和成績冊選項（「打分」欄位下）設定。

如需您想要獲得比目前所看到的未公開評估可用設定更多的設定，那麼您就需要建立並編輯一個新的評估類型。如需執行上述操作，請參考新增/建立、編輯或刪除評估類型。

評估設定

註釋：評估學習時不應使用測驗和小測中的回饋選項，如果使用這一選項，那麼學生就可以使用其中提供的資訊在提交測試前，更正問題的答案。

以下是全部可用設定清單：

  • 評估類型資訊：如果您選擇基於現有自訂範本建立評估，那麼將在此處自動顯示評估的標題、作者和簡要介紹。

  • 評估介紹：指定評估的標題和作者，以及新增評估的描述和介紹。還將列出評估建立人（即評估建立時登入的使用者），但是您無法修改此資訊。

  • 傳送日期：指定評估可用、到期和撤銷時間（日期和時間）。如需設定日期和時間，按一下每個欄位旁的行事曆圖示。

  • 評估發佈對象：選擇向匿名使用者發佈評估，或僅向網站成員發佈。

    註釋：一旦評估公開後，評估的作者將收到評估的 URL 地址。由評估作者決定是否將 URL 傳送給非網站成員使用者。

  • 高安全度：指定只有某些 IP 地址才能存取您的評估，或建立專用於存取評估的副使用者名和密碼。

  • 定時評估：您可以將評估設定為定時評估，並指定允許的時間。還可以選擇在您所設定的時間期滿後，自動提交定時評估。

  • 評估結構：管理評估的外觀和操作。有下列選項可供選擇：

    • 導覽：從下列選項中進行選擇：
      • 線性存取：限制使用者存取目錄，只能按照順序從前往後處理評估。
      • 隨機存取：允許使用者使用上一個和下一個按鈕或目錄在評估中的各問題之間移動。
    • 問題配置：選擇在單獨的網頁中顯示每個問題、在單獨網頁中顯示每個部分或在一個網頁中顯示完整的評估內容。
    • 編號：選擇評估的全部部分採用連續編號，或選擇每個部分重新開始編號。

  • 提交：指定允許的提交次數，並使用「延期處理」選項設定是否接受在截止日期之後提交評估（請參考上文傳送日期部分瞭解關於到期日期選項的資訊。）如果選擇接受延期提交，那麼將在打分過程中為這些提交內容打上標記。

  • 提交訊息：輸入資訊或指定 URL 地址，以便在使用者提交後將這些內容傳送給他們。

  • 回饋： 選擇下列選項中的一項：

    • 立即回饋
    • 提交後回饋（注意：與「記錄上次得分」打分選項結合使用最佳。）
    • 將在指定日期向學生顯示回饋（按一下行事曆圖示指定日期）
    • 不向學生顯示任何回饋

    對於回饋中的內容，您還可以選擇下列選項的任意組合：

    • 學生回答
    • 正確的回答
    • 學生的評估得分
    • 學生的問題得分
    • 問題層級回饋
    • 選擇層級回饋
    • 評分者的註解
    • 統計和長條圖

  • 打分：指定是否匿名打分，是否向成績冊傳送成績，以及如果允許多次提交，那麼如何處理得分。如果選擇匿名打分，那麼評分者將無法看到與提交評估相關的使用者名。

  • 圖形：更改評估外觀：

    • 按一下「背景顏色」欄位旁的調色板圖示開啟顏色選擇器。從下拉式清單選擇您想要使用的調色板（Web 安全調色板、Windows 系統調色板或灰度調色板），然後按一下您想用於評估背景的顏色。代表您所選擇顏色的十六進位值將自動填入「背景顏色」欄位。
    • 在「背景影像」欄位中，您可以輸入作為背景影像使用的影像檔的 URL 地址。

  • 詮釋資料：指定關鍵字、目的和問題，以便搜尋和分類評估。

此外，您還可以選擇收集問題的詮釋資料。

如需儲存評估設定，按一下頁面底部的儲存設定。按一下取消取消任何未經儲存的更改。

註釋：評估必需至少包含一個問題才能公開。
