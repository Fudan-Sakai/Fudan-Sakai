О типах тестов

Тип теста - это предустановленный набор настроек теста, который призван сделать проще процесс создания теста. Типы теста автоматически становятся общими для всех ваших сайтов курса или сайтов проекта.

Когда вы создаете или изменяете тип теста, вы задаете, какие изменения в настройках можно будет произвести, когда этот тип будет использоваться для создания теста. Подробные инструкции по использованию типа теста при создании нового теста смотри в Создание нового теста. Подробную информацию о настройках теста смотри в Настройки теста.

Создание типа теста

Чтобы создать тип теста:

  1. На линейке инструментов нажмите Тесты и экзамены.

  2. На странице со списком типов тестов нажмите Типы тестов.

  3. Под надписью "Новый тип теста" в текстовом поле введите название нового типа теста и нажмите Создать.

  4. Под надписью "Изменить тип теста:" находится список настроек. Чтобы воспользоваться опциями одного из наборов настроек, нажмите на его название или на трегольник рядом. Затем поставьте флажки, чтобы отметить необходимые настройки, которые нужно изменить. Кроме того, некоторые из настроек имеют настраиваемые параметры. Подробную информацию о доступных настройках и опциях смотри в Настройки типа теста ниже.

  5. Когда вы закончили, нажмите Сохранить. В списке "Сохраненные типы тестов" появится новый тип теста.

Изменение типа теста

Чтобы изменить тип теста, на странице "Типы тестов" в списке "Сохраненные типы тестов" нажмите на название типа. Подробную информацию о настройках и опциях смотри в Настройки типа теста ниже. Когда вы закончили, нажмите Сохранить.

Удаление типа теста

Чтобы удалить тип теста на страние "Типы тестов" в списке "Сохраненные типы тестов" нажмите на ссылку Удалить под названием типа, который нужно удалить. Нажмите Удалить еще раз для подтверждения.

Настройки типа теста

Когда создается новый тип теста, можно определить, какие изменения настроек необходимо произвести, когда вы создаете тест на базе шаблона типа теста. Эти настройки и опции включают в себя:

  • Информация о типе теста: В текстовых полях укажите название, автора и описание данного типа теста. Чтобы эта инфомация появлялась в настройках каждого теста, который создается на основе этого типа, поставьте флажок рядом с надписью "Возможен просмотр?".

  • Введение к тесту: Поставьте флажки, чтобы иметь возможность изменять текстовые поля "Автор" и "Описание/введение к тесту".

  • Аудитория теста: Если вы отметите флажками эти опции, вы сможете указывать, кто может проходить тест, основанный на этом типе.

  • Ограничение доступа: Поставьте флажки, чтобы сделать возможной опциональные установки ограничения доступа для тестов. Если не поставить флажки, опции ограничений доступа будт недоступны в тестах, которые создаются на базе этого типа.

  • Ограничение по времени: Можно превратить тест в тест с ограничением по времени и установить время, отпускаемое на тест. Тест будет сдан автоматически, когда отведенное на его выполнение время истечет.

  • Структура и организация теста: Эти настройки устанавливают порядок прохождения теста студентами. Используйте переключатель, чтобы задать тип настроек по умолчанию, используйте флажки, чтобы разрешить изменения в дальнейшем. Если не поставить флажки, то настройки по умолчанию будут действовать во всех тестах, созданных на базе этого типа.

  • Сдача теста: Эти настройки устанавливают, сколько раз студент может проходить тест, и будут ли тесты приниматься после окончания срока предъявления. Используйте переключатель для установки настроек по умолчанию, поставьте флажки, чтобы разрешить изменения в дальнейшем. Если не поставить флажки, то настройки по умолчанию будут действовать во всех тестах, созданных на базе этого типа.

  • Сообщение студенту перед сдачей теста: Сообщение перед сдачей теста и URL появляются на послежней странице теста, после того, как студент нажимает Отправить на оценку. Поставьте флажки, чтобы разрешить вставку сообщений и URL. Если не поставить флажки, то настройки по умолчанию будут действовать во всех тестах, созданных на базе этого типа.
  • Отзыв: Под надписью "Отзыв" можно выбрать Отзыв на конкретные вопросы, Отзыв только на отпреденные варианты ответа (A,B,C...), или Оба способа. Под надписью "Предъявление отыва" можно выбрать, будут ли студенты видеть отзыв и в какой момент они будут его видеть. Под надписью "Состав отзыва" можно выбрать, какие компоненты отзыва будут доступны студентам. Можно также установить, будет ли возможно в дальшейшем изменять эти настройки.

    Внимание: Не нужно использовать опцию отзыва в инструменте "Тесты и экзамены" в тестах на оценку, так как эта информация может помочь студунтам проверить свои ответы перед отправкой теста.

  • Проверка: Опции проверки устанавливают следующее:

    • Будут ли пользователи, оценивающие работы, видеть имена студентов, или работы будут анонимными
    • Будут ли оценки отправляться в инструмент "Зачетная книжка"
    • Как будут засчитываться результаты после множественных попыток сдачи теста

    Используйте переключатель для установки настроек по умолчанию, поставьте флажки, чтобы разрешить изменения в дальнейшем. Если не поставить флажки, то настройки по умолчанию будут действовать во всех тестах, созданных на базе этого типа.

  • Оформление: Поставьте флажки, чтобы разрешить изменения цвета или картинки фона в тесте. Если не поставить флажки, опция "Оформление" будет недоступна в тестах, созданных на основе этого типа.

  • Метаданные: В настройке опции "Метаданные" существуют текстовые поля для записи нужной информации для теста или вопроса. Три текстовые поля включают "Рубрики", "Цели" и "Ключевые слова". Если не поставить флажки, метаданные не смогут быть записаны в тесте, созданном на основе этого типа.
