Можно ли передать другим пользователям фонд фопросов?

Передать фонд вопросов другим преподавателям через интерфейс системы нельзя. Фонды вопросов доступны только для пользователя, который их создал. Тем не менее, можно экспортировать фонд вопросов и передать файл другому преподавателю, чтобы он смог его импортировать.
