Типы вопросов, доступные в тесте

Внимание: Для того чтобы совершать все эти действия, вам должна быть назначена роль с соответствующими полномочиями. Для того чтобы определить, какая у вас роль, перейдите на страницу Роли участников. Основные сведения о ролях и правах доступа изложены также в Права доступа и роли: обзор.

Существует гибкий набор возможностей при создании теса. Можно использовать любой из типов вопросов. описанных ниже по очереди или в любой комбинации в тесте. Более подробные инструкции о создании каждого типа вопросов смотри в Создание вопросов к тесту

На этой странице приводится обзор доступных типов вопросов:

  • Выбор вариантов ответа: Предоставляет участнику вопрос со списком возможных ответом. Можно выбрать, будет ли правильным только один или несколько ответов.

  • Опрос/анкетирование: Разработан для сбора данных в форме анкеты: предоставляет участникам вопрос или утверждение, на которое они отвечают, используя набор ответов, который вы выбрали:

    • Истина - ложь: Выбрать из "Да" - "Нет" или "Согласен" - "Не согласен".
    • Шкала Лайкерта: Выбрать из смысловой или числовой шкалы (например, "Категорически несогласен - Полностью согласен", 1-5)

  • Развернутый ответ: Предоставляет пользователям вопрос, снабженный текстовым полем для ответа.

  • Заполнить пропуски: Предоставляет пользователям вопрос, снабженный текстовым полем, в котором они вводят ответы, при мем каждый ответ сравнивается со списком возможных ответов.

  • Численный ответ: Предоставляет пользователям вопрос, снабженный текстовым полем, в котором они вводят числовой ответ, каждый ответ сравнивается со списком возможных ответов.

  • Парное соответствие: Предоставляет пользователям серию возможных парных соответствий.

  • Истина/Ложб: Предоставляет пользователям утверждение, о котором они должны указать, истинным или ложным оно является.

  • Аудиозапись: Предоставляет пользователям вопрос, во время ответа на который открывается приложение для звукозаписи и позволяет пользователю использовать микрофон.

  • Загрузка файла: Предоставляет пользователям ворпос, в качестве ответа на который они должны загрузить файл.

  • Вставить из фонда вопросов: Дает возможность импортировать вопросы из имеющихся фондов вопросов. Более подробную информацию смотри в Создание, перемещение, копирование или даление фонда или субфонда вопросов.

Можно установить любую стоимость вопроса в баллах для каждого вопроса. Вопросы типа "Опрос/Анкетирование по умолчанию оценены в 0 (ноль) баллов. но можно оценить и этот тип вопроса. Кроме того, каждый из перечисленных типов вопроса оценивается автоматически, кроме типов "Развернутый ответ", "Аудиозапись" и "Загрузка файла"
