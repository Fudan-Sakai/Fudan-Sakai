Просмотр оценок и ответов участников в инструменте "Тесты и экзамены"

Чтобы просмотреть оценки и ответы студента на отдельный вопрос в инструменте "Тесты и экзамены":

  1. На линейке инструментов соответствующего сайта курса или сайта проекта нажмите Тесты и экзамены.

  2. На странице со списками тестов в списке "Опубликованные тесты" нажмите нужный тест, под его названием нажмите Баллы. По умолчанию, откроется страница "Суммарные баллы", показывающая таблицу со следующей информацией по каждому участнику:

    • ID сданной работы
    • Роль
    • Дата сдачи (дата, когда участник отправил тест)
    • Статус (либо "Отправлено", либо "Не отправлено", либо "Все оценено")
    • Суммарный бал
    • Настройка (текстовое поле, в котором можно ввести баллы, которые будут прибавлены к суммарному балу. Нажмите Обновить, чтобы сохранить изменения).
    • Последняя (последняя оценка студента, посчитанная путем сложения Суммарного балла и балла, добавленного с использованием опции "Настройка")
    • Кломментарий (текстовое поле, в котором можно ввести комментарий, которй будет появляться перед участником, когда он будет просматривать сданную работу. Нажмите Обновить, чтобы сохранить изменения)

    Внимание: По умолчанию, информация на странице "Суммарный балл" отсортирована в алфавитном порядке. Чтобы применить сортировку по другой колонке, нажмите на название колонки (например, сортируйте информацию по ролям участников, нажав Роль). Если вы создали группы с использованием инструмента "Группы", можно использовать выпадабщий список вверху экрана, чтобы установитрь отображение информации по отдельной группен. Если вы разрешили повторную сдачу, можно использовать выпадающий список вверху экрана, чтобы посмотреть только работы с наивысшей оценкой.

  3. Чтобы просмотреть вопросы теста и ответы участника на каждый вопрос, вверху экрана нажмите Вопросы. Переходите по вопросам, нажимая на ссылку вопроса вверху экрана.

    Внимание: Эта опция помогает быстро сопоставить ответы студентов на отдельный вопрос, но просмотреть развернутые ответы студентов можно только по одному.

  4. Чтобы просмотреть информацию об общем количестве сданных работ, возможный наивысший балл и статистическую информацию (такую как среднее значение, медиана, мода и диапазон баллов), вверху страницы нажмите Статистика.

    Под обобщенными данными приведен каждый вопрос с диаграмой, показывающей количество студентов, выбравших конкретный вариант ответа.
