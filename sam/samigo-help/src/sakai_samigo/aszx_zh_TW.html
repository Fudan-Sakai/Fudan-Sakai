檢視測驗和小測中的參與者得分和對各問題的回答

如需檢視測驗和小測中的參與者得分和對各問題的回答：

  1. 在相關課程或專案網站的功能表欄中，按一下測驗和小測。

  2. 在評估頁面中的「已公開評估」下找到相關評估，按一下該評估標題下的得分。預設設定下，隨後將開啟總分評估，顯示包含各參與者下列資訊的表格：

    • 姓名
    • 使用者 ID
    • 角色
    • 日期（參與者提交評估的日期）
    • 狀態（「已提交」、「未提交」或「全部已打分」）
    • 總分
    • 調整（用於輸入新增到總分中的調整量的欄位）；按一下更新儲存更改）
    • 最終得分（參與者的最終得分；該欄位是將總分和調整欄位中的內容相加得出）
    • 註解（您可在該文字方塊中輸入註解，當參與者檢視各自的已提交測試時，將看到您新增的註解；按一下更新儲存更改）

    註釋：預設設定下，總分螢幕上的資訊將按照參與者姓名的字母順序排列。如需按照其他列的順序排列，按一下該欄名稱（例如，按一下角色按照參與者角色排列資訊）。如果您使用班別資訊工具建立班別，那麼可使用螢幕頂部的下拉式清單選擇僅檢視特定班別的資訊。如果允許多次提交，那麼可使用螢幕頂部的下拉式清單選擇僅檢視獲得最高得分的提交內容。

  3. 如需檢視測試中的問題，以及參與者對每個問題作出的回答，按一下螢幕頂部的問題。按一下螢幕頂部的問題連結逐個瀏覽問題。

    註釋：透過這種方法可快速地比較全部學生對同一問題的回答，但是如果要檢視書面回答，那麼只能逐個學生進行檢視。

  4. 如需檢視關於提交總數、總分的資訊以及相關統計資訊（如得分的平均值、中值、模式和範圍等），按一下螢幕頂部的統計。

    在每個問題的累計資訊下方，還顯示一個長條圖，用於指出有多少名學生選擇了同一個回答。
