Создание дубликата сайта

При создании дубликата сайта курса Вы создаете копию сайта, которая не в полной мере повторяет оригинал. Дубликат сайта не будет содержать материалы, размещенные студентами; а только материалы, выложенные преподавателями.

При создании дубликата сайта проекта Вы создаете копию сайта, в полной мере повторяющую оригинал. Дубликат сайта будет содержать все материалы, выложенные участниками.

Чтобы создать дубликат сайта:

  1. Откройте сайт, на линейке инструментов, нажмите Настройки.

  2. Нажмите Создать дубликат сайта.

  3. В поле "Название сайта:" напечатайте название дубликата сайта, выберите Академический семестр, затем нажмите Создать дубликат.

    Внимание: Если вы не обладаеме достаточными правами для создания сайта, Вы получите сообщение: "Предупреждение: Вы не можете создать новый сайт".

  4. Нажмите Завершить.
