Добавление, редактирование и удаление участников

На этой странице:

  • Добавить участников
  • Редактировать участников
  • Удалить участников

Внимание: Для выполнения этих действий Вам должна быть назначена роль с соответствующими полномочиями. Для того чтобы определить, какая у Вас роль,перейдите на страницу Роли участников. Основные сведения о ролях и правах доступа изложены также в Права доступа и роли: Обзор.

Добавить участников

  1. На линейке инструментов Вашего сайта нажмите Настройки.

  2. Нажмите Добавить участников.

  3. Для участников с официальными именами пользователей в поле "Другие официальные участники" введите имя пользователя для каждого участника. Если Вы хотите добавить несколько участников, введите имя каждого пользователя с новой строки.

    Для участников с неофициальными именами пользователей в поле "Неофициальные участники" введите их электронные адреса, каждый с новой строки.

  4. В меню "Роли участников" выберите, какие роли необходимо присвоить новым участникам: одинаковые или индивидуальные. Дополнительные сведения о ролях смотрите в Роли участников. Нажмите Продолжить.

  5. Назначьте роли участнику (ам), которых Вы добавили. Если Вы присваиваите участникам индивидуальные роли, используйте для выбора соответствующей роли "выпадающий" список рядом с каждым именем. Если Вы хотите назначить всем участникам одинаковую роль, используйте кнопки для выбора соответствующей роли. Нажмите Продолжить.

    Внимание: В зависимоти от конфигурации Вашего сайта, роли могут различаться.

  6. Следующая страница предоставляет возможность автоматически отправить новым участникам сообщение с уведомление о том, что сайт доступен для использования. Выберите соответствующую кнопку и затем нажмите Продолжить.

  7. Подтвердите правильность информации об участниках, которых Вы добавили, и нажмите Завершить.

Редактирование участников

  1. На линейке инструментов Вашего сайта нажмите Настройки.

  2. В Списке участников вы можете изменить роль участника, используя "выпадающий" список в меню "Роль".

  3. В меню "Статус" можно использовать "выпадающий" список для изменения статуса участника; Выберите Действительный или Недействительный. Студент с недействительным статусом остается членом сайта, однако лишается доступа к сайту. (Вы можете использовать эту функцию для исключения с сайта студентов, которые имеют доступ через взаимодействующий список студентов и, следовательно, не могут быть удалены).

  4. Нажмите Обновить участников.

Удалить участников

  1. На линейке инструментов Вашего сайта нажмите Настройки.

  2. В Списке участников в меню "Удалить" отметьте флажком каждого участника, которого Вы бы хотели удалить.

  3. Нажмите Обновить участников.
