Изменение информации о сайте с помощью Настроек

Внимание: Для выполнения этих действий Вам должна быть назначена роль с соответствующими полномочиями Для того чтобы определить, какая у Вас роль, перейдите на страницу Роли участников. Основные сведения о ролях и правах доступа изложены также в Права доступа и роли: Обзор.

Для изменения информации о сайте с помощью Настроек следуйте перечисленным ниже указаниям:

  1. В меню инструментов Вашего сайта нажмите Настройки.

  2. Нажмите Изменить Информацию о сайте.

  3. В поле "Название сайта" измените название сайта.

    Внимание: Вы не можете изменять название сайта курса.

  4. В поле "Описание:" введите информацию, которая, по Вашему мнению, должна отображаться в разделе "Информация о сайте" Вашего сайта

    Дополнительные сведения о добавлении материалов в раздел "Информация о сайте" смотрите в Добавить материалы в раздел "Информация о сайте".

  5. Если Ваш сайт открыт для публичного доступа, в поле "Краткое описание:" введите информацию, которая, по Вашему мнению, должна отображаться, когда сайт появляется в публичном списке сайтов.

  6. У Вас также может иметься возможность выбрать иконку для отображения на сайте.

  7. Введите соответствующую информацию в полях "Контактное лицо:" и "Контактный адрес электронной почты:" и затем нажмите Продолжить.

  8. Для подтверждения нажмите Завершить. Чтобы отменить изменения, нажмите Отмена.

Внимание: Изменяя информацию о сайте, Вы также можете изменить его внешний вид. Дополнительные сведения смотрите в Изменение внешнего вида сайта.
