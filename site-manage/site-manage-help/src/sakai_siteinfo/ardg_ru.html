Изменение внешнего вида сайта

Note: Для выполнения этих действий Вам должна быть назначена роль с соответствующими полномочиями. Для того чтобы определить, какая у Вас роль, перейдите на страницу Роли участников. Основные сведения о ролях и правах доступа изложены также в Права доступа и роли: Обзор.

Сайты курсов

Для того чтобы изменить внешний вид сайта курса, с помощью Настроек выберите один из предлагаемых в вашем учебном заведении видов:

  1. На линейке инструментов Вашего сайта выберите Настройки, затем выберите Изменить информацию о сайтеn.

  2. Используйте "выпадающее" меню Внешний вид (Иконка): для выбора одного из доступных видов, а затем нажмите Продолжить.

  3. Для подтверждения нажмите Завершить.

    Внимание: Возможно, Вам понадобится обновить сайт для того чтобы внесенные изменения отобразились.

Для того чтобы вернуться к внешнему виду по умолчанию:

  1. В "выпадающем" меню Внешний вид (Иконка): выберите *по умолчанию*, затем нажмите Продолжить.

  2. Для подтверждения нажмите Завершить.

Сайты проектов

Чтобы изменить внешний вид сайта проекта выберите иконку, которая будет отображаться сверху линейки инструментов:

  1. На линейке инструментов Вашего сайта выберите Настройки, затем выберите Изменить информацию о сайте.

  2. В поле "Ссылка на иконку:" введите ссылку на графический файл, который, по Вашему мнению, должен служить иконкой сайта, затем нажмите Продолжить.

  3. Для подтверждения нажмите Завершить.

    Внимание: Возможно, Вам понадобится обновить сайт, для того чтобы внесенные изменения отобразились.

Для того чтобы вернуться к внешнему виду по умолчанию(без иконки):

  1. Из поля "Ссылка на иконку:" удалите ссылку, которую Вы ввели. Поле должно остаться пустым.

  2. Нажмите Продолжить, для подтверждения нажмите Завершить.
