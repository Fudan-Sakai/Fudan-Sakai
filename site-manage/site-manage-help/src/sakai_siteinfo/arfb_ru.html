Изменение роли участников

Note: Для выполнения этих действий Вам должна быть назначена роль с соответствующими полномочиями. Для того чтобы определить, какая у Вас роль, перейдите на страницу Роли участников. Основные сведения о ролях и правах доступа изложены также в Права доступа и роли: обзор.

Для изменения роли участников:

  1. На линейке инструментов соответствующего сайта курса или сайта проекта нажмите Настройки.

  2. Рядом с именем участника (ов), роль которого (ых) Вы хотите изменить, выберите соответствующую роль из "выпадающего" меню в разделе "Роль".

  3. Нажмите Обновить участников.

Дополнительные сведения о том, какие роли доступны, смотрите в Роли участников.
