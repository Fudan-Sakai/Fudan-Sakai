Обзор

Блоки пользователей являются подгруппами участников данного сайта. Блоки пользователей не задаются в числе официальных данных о курсе и, следовательно, могут быть созданы преподавателем. Блоки пользоватей полезно использовать для организации блоков студентов, команд для работы над проектами, и других неофициальных подгрупп из числа участников сайта.

Внимание: При объединении групп на сайте верхнего уровня блоки из оригинальных групп не будут перенесены. Тем не менее, каждая группа на сайте верхнего уровня действует в качестве блока, например инструменты для блоков пользователей распознают каждую группу в качестве блока.

Вы можете использовать блоки при работе со следующими инструментами:

  • Объявления: Разместить Объявления для Вашего блока.

  • Задания: Разместить задания для Вашего блока. Однако нельзя добавлять Задания с доступом только определенных блоков пользователей в Оценки.

  • Сообщения: Отправить скрытые сообщения Вашему блоку пользователей.

  • Ресурсы: Разрешить Вашему блоку особый доступ к файлам и папкам.

  • Календарь: Отмечать в календаре события только для определенных блоков пользователей.

  • Настройки: Управление блоками пользователей.

Доступ к блокам пользователей

Вы пожете получить доступ к блокам пользователей при помощи инструментов Настройки и Мои сайты.

Чтобы получить доступ к группам с помощью инструмента Настройки:

  1. На линейке инструментов сайта нажмите Настройки.

  2. Нажмите Управление блоками пользователей.

Чтобы получить доступ к группам с помощью инструмента Мои сайты:

  1. Нажмите Мой сайт.

  2. На линейке инструментов нажмите Мои сайты.

  3. Отметьте флажком курс, в котором вы хотите управлять блоками пользователей, наверху нажмите Изменить.

  4. Нажмите Управление блоками пользователей.

Создание блока пользователей

Чтобы создать новый блок пользователей:

  1. Получите доступ к разделу управления блоками на сайте (смотри выше).

  2. Нажмите Добавить.

  3. В поле "Название" введите название блока пользователей (например, Команда для работы над проектом 1). Также можно добавить текстовое описание блока.

  4. Из окна слева выберите участников сайта, которых Вы хотите добавить в блок, затем нажмите Добавить в блок. Повторите этого действие для каждого участника сайта, которого Вы хотите сделать членом блока. Чтобы выбрать сразу несколько членов блока, удерживая нажатой клавишу Ctrl (для Windows) или клавишу Cmd (для Mac OS X), выберите членов блока, которых Вы хотите добавить.

  5. Когда Вы закончите добавлять членов блока, нажмите Обновить.

Внимание: Невозожно создать новый блок пользователей на основнии уже существующего блока (т.е. сделать дубликат существующего блока).

Изменение блока

Чтобы изменить блок пользователей:

  1. Получите доступ к разделу управления блоками на сайте (смотрите выше).

  2. Рядом с названием группы нажмите Изменить.

  3. Вы можете изменить "Название" и "Описание". Вы также можете добавлять или удалять членов блоков, нажав Добавить в блок или Удалить.

  4. Когда Вы закончите вносить изменения, нажмите Обновить.

Удаление блоков

Чтобы удалить блок:

  1. Получите доступ к разделу управлению блоками на сайте (смотрите выше).

  2. Рядом с группой, которую Вы хотите удалить, поставьте флажок в меню Удалить.

  3. Нажмите Удалить отмеченные.

  4. Для подтверждения удаления нажмите Удалить.
