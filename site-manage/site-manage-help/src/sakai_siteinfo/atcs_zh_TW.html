概要

群組是特定工作網站內參與者的子集。群組不會預先載入任何正式的課程資料，而且可由講師建立。群組是組織學習群組、專案團隊和其他工作網站參與者的非正式子集的有效方法。

註釋：將多個班別合併到一個母網站時，源班別的群組不會繼續保留。然而原先的每個班別在母網站中的功能將與群組相同，即群組識別工具將每個班別識別為一個群組。

您可以將群組與下列工具結合使用：

  • 通告： 發佈群組通告。

  • 作業：發佈群組作業。但是，您無法將群組作業新增到成績冊中。

  • 訊息： 向群組傳送私人郵件。

  • 資源： 允許存取群組中的檔案和資料夾。

  • 日程表：僅特定群組的日程表事件。

  • 網站資訊：管理群組。

存取群組

您可以使用網站資訊和工作網站設定工具存取群組。

如需使用網站資訊工具存取群組，請遵循下列說明進行操作：

  1. 在網站功能表欄中，按一下網站資訊。

  2. 按一下管理群組。

如需使用工作網站設定工具存取群組，請遵循下列說明進行操作：

  1. 按一下我的工作區。

  2. 從功能表欄按一下工作網站設定。

  3. 按一下您想要管理其中群組的課程旁的核取方塊，然後從頂部按一下編輯。

  4. 按一下管理群組。

建立群組

如需新建群組，請遵循下列步驟進行操作：

  1. 存取網站的群組區（請參考上文）。

  2. 按一下新增。

  3. 在「標題」欄位中輸入群組標題（例如 Project team 1）。您還可以新增文字描述。

  4. 從左側的視窗中選擇您想要新增至群組中的網站參與者，然後按一下新增至群組。重複此步驟，直到全部成員新增完畢。如需一次選擇多個成員，按住 Ctrl 鍵(Windows)或 Cmd 鍵(Mac OS X)，同時選擇您想要新增的成員。

  5. 操作完成後，按一下更新。

註釋：無法在現有群組的基礎上建立新的群組（即複製現有群組）。

編輯群組

如需編輯群組，請遵循下列步驟進行操作：

  1. 存取網站的群組區（請參考上文）。

  2. 按一下群組標題旁的編輯。

  3. 您可以修改「標題」和「描述」欄位。也可以按一下新增至群組或刪除新增或刪除群組成員。

  4. 操作完成後，按一下更新。
返回頂部

刪除群組

如需刪除群組，請遵循下列步驟進行操作：

  1. 存取網站的群組區（請參考上文）。

  2. 選中您想要刪除的群組標題旁的刪除核取方塊。

  3. 按一下刪除選中。

  4. 如需確認刪除操作，按一下刪除。
