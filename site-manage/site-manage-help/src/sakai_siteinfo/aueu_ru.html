Инструменты для управления электронной почтой сайта

Для управления коммуникациями между участниками сайта существует два отдельных инструмента: Архив электронной почты и Сообщения. Более того, при добавлении новой информации можно также извещать об этом участников по электронной почте с помощью других инструментов сайта.

Архив электронной почты

Архив электронной почты присваивает Вашему сайту собственный электронный адрес для общения со всеми участниками. Дополнительные сведения смотрите в Архив электронной почты: обзор.

Сообщения

Инструмент Сообщения позволяет участникам оставлять друг другу на сайте личные сообщения. С помощью этого инструмента также можно посылать сообщения нескольким участникам и заранее определенным блокам пользователей. Дополнительные сведения смотрите в Сообщения: обзор.

Извещение по электронной почте

С помощью некоторых инструментов (например Объявления, Ресурсы и Программа курса) можно посылать извещения по электронной почте участникам сайта (на их электронные адреса) при добавлении новой информации. Можно установить низкий приоритет извещения, получение которых каждым участником будет зависеть от параметров, заданных им для извещений; или высокий приоритет, который будет доминировать над индивидуально заданными параметрами и позволит доставить извещения всем участникам сайта.
