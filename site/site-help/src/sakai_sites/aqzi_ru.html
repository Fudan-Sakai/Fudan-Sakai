Добавление, изменение или удаление страниц сайта

Внимание: Чтобы использовать этот инстурмент, вы должны обладать правами администратора.

Чтобы добавлять, изменять или удалять страницы сайта:

  1. На линейке инстурментов раздела Мой сайт , нажмите Сайты. Появится список всех сайтов.

  2. В списке нажмите ID сайта, который требуется изменить.

  3. В появившейся форме нажмите кнопку Страница внизу формы.

  4. Вы увидите список страниц сайта. Доступны следующие изменения:

Добавление страницы

Чтобы добавить страницу, вверху формы нажмите Добавить страницу. Заполните поля в появившейся форме. Когда вы закончили, нажмите Сохранить.

Изменение страницы

Чтобы изменить информацию о странице в списке страниц нажмите ID страницы, которую нужно изменить. Произведите необходимые изменения в появившейся форме. Когда вы закончили, нажмите Сохранить.

Удаление страницы

Чтобы удалить страницу, нажмите ID страницы, которую нужно удалить. Вверху появившейся формы нажмите Удалить страницу.
