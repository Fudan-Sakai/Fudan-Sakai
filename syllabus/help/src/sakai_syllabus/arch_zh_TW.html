課程大綱：概述

功能

課程大綱是指課程的正式大綱。作為講師，如果您或您的學系已經編寫了線上課程大綱，那麼可將課程大綱工具設定為跳轉至已完成的提綱。否則，您可以輸入資料內容，以便直接發佈至課程大綱中。您可以將課程大綱設定為向全部一般公眾公開，或僅向課程成員公開。

傳統課程大綱的一些功能分別分佈在課程大綱和日程表區域中。一些講師使用課程大綱區顯示所在學系的正式線上課程大綱，而將日程表作為學生專用的詳細資源區。

主要概念

內容格式：您可以使用編輯視窗頂部的按鈕編排內容的格式。按鈕提供任何所需的 HTML 標記。您可以選擇在內容中顯示這些 HTML 標記或不顯示標記之間切換，也可以不使用按鈕，輸入自己的 HTML 標記。

附件：可以從資源檔案夾附加檔。如需瞭解更多資訊，請參考新建/新增、編輯或刪除課程大綱專案。

注意事項

  • 您可以透過多種方法存取網路資源。連結至網路資源前，先考慮是否將該資源作為日程表項目更為適當（如每週或定期作業的主題），或將其作為資源部分中的一項資源。

  • 如果從您所擁有的其他網站附加檔案（包括我的工作區），那麼除非您在該檔案屬性中選擇向非成員顯示（可公開瀏覽），否則網站參與者將無法看到這些檔案。如需瞭解更多資訊，請參考公開資源中的項目。
