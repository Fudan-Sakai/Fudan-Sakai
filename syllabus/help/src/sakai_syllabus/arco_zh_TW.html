顯示課程大綱的網頁、PDF 或 Word 檔案

註釋：如果要完成本頁中的操作，您必需擁有具有必要權限的角色。如需決定自己的角色，請遵循參與者角色中的說明進行操作。如需瞭解關於角色和權限的基礎知識，請參考權限和角色：概述。

您可以將課程網站或專案網站的課程大綱指向您已經上載至資源部分的 HTML、PDF 或 Microsoft Word 檔案，或指向外部 URL 地址。如需更改課程大綱指向：

  1. 按一下專案網站或課程網站的標題進入網站。

  2. 如果您無法在功能表欄中看到課程大綱工具，那麼您必需將該工具新增到網站中。如需將課程大綱工具新增至網站中，請參考新增、編輯或刪除工具。

  3. 還可以為課程大綱顯示外部網頁。或者也可以顯示您已經上載到資源部分的 HTML、PDF 或 Word 檔案。遵循下列適當步驟進行操作：

顯示外部 URL

如需顯示外部 URL，如學系伺服器上的網頁：

  1. 從功能表欄按一下課程大綱。

  2. 按一下建立/編輯，然後按一下重新導向。

  3. 在「URL」欄位中輸入您想要作為課程大綱顯示的網頁。

  4. 按一下儲存。

顯示 HTML、PDF 或 Word 檔案

如需可以顯示您已經上載到資源部分的 HTML、PDF 或 Microsoft Word 檔案：

  1. 將檔案上載到網站的資源部分。如需瞭解更多說明，請參考操作資源中的檔案。

  2. 將檔案發佈到網站中後，如果您無法看到資源清單，那麼從功能表欄按一下資源。

  3. 如需獲取檔案的 URL 地址，按右鍵（對於 Mac OS 或Mac OS X，Ctrl-按一下) 檔案名，然後選擇瀏覽器複製命令（即，對於 Internet Explorer，選擇複製捷徑；對於 Netscape 或 Mozilla Firefox，選擇複製連結網址）。

  4. 從功能表欄按一下課程大綱。

  5. 按一下建立/編輯，然後按一下重新導向。

  6. 將 URL 貼上至「URL」欄位中（對於 Windows，Ctrl-v；對於 Mac OS 或 Mac OS X，Cmd-v）。

  7. 按一下儲存。

註釋：對於使用 Internet Explorer 或 Mozilla Firefox 的 Mac 使用者，如果將 PDF 檔案作為課程大綱，那麼將開啟一個新視窗。
