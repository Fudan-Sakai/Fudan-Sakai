Членство: Как добавить инструмент Членство

Внимание: Для того чтобы выполнять действия,перечисленные ниже, Вы должны обладать правами Администратора.

Добавление инструмента

Инструмент Членство может быть добавлен на сайт только при помощи администрирующего инструмента "Сайты". Чтобы добавить инструмент, войдите в систему как Администратор и выполняйте следующие действия:

  1. Войдите в меню инструмента "Сайты" и найдите сайт, на который Вы хотите добавить инструментs.
  2. Нажмите на профиль сайта и отредактируйте его.
  3. Перейдите вниз и нажмите на кнопку "Страницы", чтобы добавить новую страницу.
  4. В следующем окне нажмите на кнопку "Новая страница" и выберите имя для страницы (стандартно - "Членство"), а затем нажмите "Инструменты".
  5. Нажмите на кнопку "Новый инструмент", выберите "Членство" (sakai.usermembership) и нажмите "Сохранить" для сохранения изменений.
  6. Теперь инструмент присутствует на отредактированном сайте.

Предоставление доступа

Если у пользователя нет прав администратора, ему можно предоставить доступ, выполнив действия, описанные здесь: Как сделать инструмент доступным для других пользователей.
