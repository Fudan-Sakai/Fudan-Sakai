使用者成員資格：新增使用者成員資格工具

註釋：下文中的說明需要管理員權限才能執行。

新增工具

只能使用管理員「網站」工具新增使用者成員資格工具。如需新增工具，以管理員身份登入，並遵循下列說明進行操作：

  1. 進入「網站」工具，並搜尋想要新增工具的網站。
  2. 按一下網站 ID 以編輯網站。
  3. 向下捲動頁面，按一下「頁面」新增新頁面。
  4. 在下一個螢幕上按一下「新建頁面」，選擇頁面名稱（一般為「使用者成員資格」），然後按一下「工具」。
  5. 按一下「新建工具」，然後選擇「使用者成員資格(sakai.usermembership)」，並按一下「儲存」儲存更改。
  6. 現在工具將出現在網站中。

准予存取

如果使用者沒有管理員權限，那麼必需遵循准予其他使用者存取工具中的說明准予存取。
