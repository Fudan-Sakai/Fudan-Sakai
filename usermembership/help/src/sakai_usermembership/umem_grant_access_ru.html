Членство: Как сделать инструмент доступным для других пользователей

Внимание: Для того чтобы выполнять описанные операции, Вам необходимо обладать правами Администратора.

Открыть доступ к инструменту

С того момента, как инструмент добавлен на сайт, для его использования любому пользователю, кроме Администратора, требуется предоставление доступа. Чтобы предоставить доступ к инструменту, войдите на сайт как Администратор и выполните следующие операции:

  1. Откройте инструмент 'Окружение' и найдите сайт, в котором присутствует инструмент "Членство". Вам может понадобиться инструмент "Сайты" для поиска ID соответствующего сайта.
  2. Нажмите на образ сайта, чтбы отредактировать его (он должен выглядеть как: /site/xxxxx).
  3. Перейдите вниз и нажмите на соответствующую роль, чтобы отредактировать ее (стандартно - роль "поддержка"). Она должна совпасть с ролью пользователя на сайте.
  4. Нажмите на кнопку 'usermembership.view' и выберите "Сохранить", чтобы сохранить изменения.
  5. Теперь у Вас есть возможность пользоваться инструментом.
