User Membership: Overview

What it does

The User Membership tool allows to find site and group membership information about Sakai users.

Key concepts

Membership: User enrollment in sites and groups.

Things to consider

  • The User Membership tool must be manually added to a site using the Admin Sites tool. For instructions, see Adding User Membership tool.
  • By default, only Administrators will be able to use the tool. To grant tool access to other users, see Granting tool access to other users.
