Поиск пользователей

Первая страница инструментов включает в себя параметры поиска. Поиск может быть ограничен по следующим параметрам:

  • Роль пользователя: осуществляет фильтр пользователей исходя из их роли. Предлагается список имеющихся ролей.
  • Тип пользователя: позволяет определить статус пользователя. "Внутренние" пользователи зарегистрированы во внутренней базе данных. Сторонние пользователи обслуживаются внешними механизмами идентификации (с использованием интерфейсаСписок пользователей).
  • Окно поиска: поиск по ключевому слову в имени, профиле или e-mail пользователя.
Нажав "Поиск",Вы увидите результаты поиска по выбранным параметрам. О каждом из результатов поиска Вы можете получить следующую информацию:
  • ID пользователя
  • Имя пользователя
  • Email
  • Тип (роль)
  • Тип пользователя (внутренний или сторонний)

Нажмите 'Экспорт в CSV', чтобы загрузить результаты поиска в файл с расширением .csv (доступен импорт в MS Excel).

Внимание: Сторонние пользователи попадают в список результатов поиска, только если они уже зарегистрированы на существующем сайте.

Как найти регистрационные списки пользователей

В результатах поиска нажмите на id пользователя - по этой ссылке отобразится новая страница с учетными записями пользователя. Для каждой учетной записи доступна следующая информация:

  • Имя сайта (ссылки на сайт)
  • Группы, зарегистрированные на сайте
  • Тип сайта (курс, проект и т.д.)
  • Сроки работы сайта (н-р, ОСЕНЬ 2007)
  • Роль (роль пользователя на сайте)
  • Статус сайта (опубликован, не опубликован)
  • Статус пользователя (активный, неактивный)
Нажмите 'Экспорт в CSV', чтобы загрузить результаты поиска в файл с расширением .csv (доступен импорт в MS Excel).
