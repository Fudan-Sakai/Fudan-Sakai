使用者成員資格：使用工具

在本頁中，您將瞭解：

  • 搜尋使用者
  • 尋找使用者註冊情況

搜尋使用者

第一個工具頁面顯示一些搜尋參數。可以透過指定下列內容縮小搜尋範圍：

  • 使用者類型：根據使用者角色過濾搜尋結果。顯示全部現有角色的清單。
  • 使用者授權：可選擇使用者來源。內部使用者存儲在內部 DB 中。為外部使用者提供外部授權機制（執行 UserDirectoryProvider 介面）。
  • 搜尋方塊：可指定使用者名稱、使用者 ID 和電子郵件關鍵字進行搜尋。
按一下「搜尋」將根據指定的參數顯示搜尋結果。對於每個搜尋結果，將顯示下列資訊：
  • 使用者 ID
  • 完整使用者姓名
  • 電子郵件
  • 類型（角色）
  • 授權（使用者來源）

按一下「匯出 CSV」以 .csv 檔案格式下載搜尋結果（可匯入 MS Excel）。

註釋：只有目前已經註冊到現有網站中的外部使用者才會顯示。

尋找使用者註冊情況

在使用者搜尋結果中，按一下使用者 ID 連結將在新頁面中顯示使用者的成員資格情況。對於每個網站註冊條目，將顯示下列資訊：

  • 網站名稱（網站連結）
  • 註冊到該網站中的群組
  • 網站類型（課程、專案等）
  • 網站學期（例如，2007 秋季）
  • 角色（網站中的使用者角色）
  • 網站狀態（已公開、未公開）
  • 使用者狀態（使用中、非使用中）
按一下「匯出 CSV」以 .csv 檔案格式下載搜尋結果（可匯入 MS Excel）。
