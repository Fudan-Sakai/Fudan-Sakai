Добавление, изменение и удаление страницы веб-ресурсов

Вы можете добавить ссылку на внешний сайт или страницу "Веб-ресурсов" на линейку инструментов для сайта Вашей группы или проекта, если Вы обладаете соответствующими полномочиями.

Любой пользователь может добавить страницу "Веб-ресурсов" на Мой сайт.

Дополнительные сведения по изменению или удалению страниц "Веб-ресурсов" с Вашего сайта смотрите в Добавить, изменить или удалить инструменты на линейке инструментов с помощью инструмента "Мои Сайты"

На этой странице:

  • Добавить страницу "Веб-ресурсов" на линейку инструментов сайта курса или проекта
  • Добавить страницу "Веб-ресурсов" на Мой сайт
  • Изменить страницу "Веб-ресурсов"
  • Удалить страницу "Веб-ресурсов"

    Добавить страницу "Веб-ресурсов" на линейку инструментов сайта курса или проекта

    Внимание: Вы должны обладать соответствующими полномочиями, чтобы добавить страницу "Веб-ресурсов" на сайт курса или сайт проекта.

    1. На сайте курса или проекта нажмите Настройки.

    2. На странице "Настройки" нажмите Изменить инструменты.

    3. Поставьте флажок рядом с Веб-ресурсы. (Если Ваш сайт уже содержит страницы "Веб-ресурсов", они будут перечислены под заглавиями, которые Вы им дали, или как "Веб-ресурсы", если Вы не меняли заглавие. Тем не менее, в списке инструментов должно быть еще одно пустое окно Веб-ресурсы; отметьте его флажком). Нажмите Продолжить.

    4. На странице "Настроить инструменты" введите следующую информацию:

      • В поле "Название:" введите название.
      • В поле "URL:" введите URL после "http://", который уже значится в этом поле.
      • Если Вы хотите добавить несколько страниц "Веб-ресурсов", выберите соответствующее число страниц из выпадающего меню Добавить еще одну страницу Веб-ресурсов?.

    5. Нажмите Продолжить. Вы увидете информацию о внесенных Вами на сайт изменениях. Нажмите Завершить. Если Вы хотите внести дополнительные изменения, нажмите Назад. Нажмите Отмена для отмены всех внесенных изменений.

    Добавить страницу "Веб-ресурсов" на Мой сайт

    1. На странице Мой сайт, нажмите Мои сайты.

    2. На странице Мои сайты Вы увидите список всех Ваших сайтов. Чтобы добавить страницу "Веб-ресурсов" на страницу "Мой сайт", отметьте флажком Мой сайт. Чтобы добавить страницу "Веб-ресурсов" на любой из управляемых Вами сайтов, поставьте флажок рядом с названием этого сайта.

    3. Нажмите Изменить. На следующей странице нажмите Изменить инструменты.

    4. Отметьте флажком Веб-ресурсы. (Если Ваш сайт уже содержит страницы "Веб-ресурсов", они будут перечислены под заглавиями, которые Вы им дали, или как "Веб-ресурсы", если Вы не меняли заглавие. Тем не менее, в списке инструментов должно быть еще одно пустое окно Веб-ресурсы; отметьте его флажком). Нажмите Продолжить.

    5. на странице "Настроить инструменты" введите следующую информацию:

      • В поле "Название:" введите название.
      • В поле "URL:" введите URL после "http://", который уже значится в этом поле.
      • Если Вы хотите добавить несколько страниц "Веб-ресурсов", выберите соответствующее число страниц из выпадающего меню Добавить еще одну страницу Веб-ресурсов?.

    6. Нажмите Продолжить. Вы увидете информацию о внесенных Вами на сайт изменениях. Нажмите Завершить. Если Вы хотите внести дополнительные изменения, нажмите Назад. Нажмите Отмена для отмены всех внесенных изменений.

    Теперь на линейке инструментов должна отобразиться ссылка на страницу "Веб-ресурсов" под заглавием, которое Вы ей дали.

    Изменить страницу "Веб-ресурсов"

    Чтобы изменить существующую страницу "Веб-ресурсов":

    1. На линейке инструментов сайта нажмите на название страницы "Веб-ресурсов".

    2. Наверху фрейма Веб-ресурса нажмите Настройки.

    3. На странице "Редактировать веб-ресурсы" Вы можете внести следующие изменения:

      • Назвние инструмента: Название, которое появляется наверху фрейма Веб-ресурсов
      • Заглавие страницы: Заглавие, которое появляется на линейке инструментов сайта
      • Высота фрейма: Высота (в пикселях) фрейма Веб-ресурсов
      • URL: URL для страницы "Веб-ресурсов"

    4. Для того, чтобы страница "Веб-ресурсов" открывалась в новом окне, поставьте флажок рядом с Открыть в новом окне?. Кроме того, Вы можете выбрать функцию отображения линейки браузера, поставив флажок рядом с Если открыт в новом окне,показывать линейку браузера?.

    5. Когда Вы закончите редактировать Веб-ресурсы, нажмите Обновить настройки для сохранения внесенных изменений или Отмена для отмены внесенных изменений.

    Удалить страницу "Веб-ресурсов"

    1. Чтобы удалить страницу "Веб-ресурсов", нажмите Мой сайт, затем нажмите Мои сайты.

    2. Если Вы хотите удалить страницу "Веб-ресурсов" с сайта курса или проекта поставьте флажок рядом с названием этого сайта. Если Вы хотите удадить страницу "Веб-ресурсов" со страницы "Мой сайт", поставьте флажок рядом с Мой сайт.

    3. Нажмите Изменить. Затем нажмите Изменить инструменты.

    4. В списке инструментов уберите флажок рядом со страницей "Веб-ресурсов", которую Вы хотите удалить, затем нажмите Продолжить. Проверьте список инструментов, который появится на следующей странице, затем нажмите Завершить для удаления страницы "Веб-ресурсов".

    Внимание: Если страница "Веб-ресурсов" по-прежнему отображается на линейке инструментов сайта, нажмите Главная для обновления линейки инструментов.
