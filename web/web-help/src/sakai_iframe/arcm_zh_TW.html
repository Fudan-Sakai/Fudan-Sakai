新增、編輯或刪除網路內容項目

如果您擁有適當的權限，那麼您可以將外部網站或網路內容項目連結新增到班級或專案工作網站的功能表欄中。

任何人都可以將網路內容項目新增到我的工作區中。

如需瞭解關於編輯項目或從工作網站刪除項目的更多說明，請參考基礎知識文件從工作網站設定新增、編輯或刪除功能表欄中的工具，

在本頁中，您將瞭解：

  • 向課程或專案網站的功能表欄新增網路內容項目
  • 新增網路內容項目至我的工作區
  • 編輯網路內容項目
  • 刪除網路內容項目
  • 功能示範

向課程或專案網站的功能表欄新增網路內容項目

註釋：您必須有適當的權限才可向課程或專案網站新增網路內容項目。

  1. 在相關課程或專案網站的功能表欄中，按一下網站資訊。

  2. 在網站資訊頁面中，按一下編輯工具。

  3. 選中 網路內容旁的核取方塊。按一下繼續。

  4. 在「自訂工具」螢幕中，輸入以下資訊：

    • 在「標題：」欄位中新增標題。
    • 在「URL:」欄位中已顯示的 "http://" 字樣後，輸入 URL 地址。
    • 如果您想要新增多個項目，那麼從更多網路內容工具？下拉式功能表選擇適當的項目數量。

  5. 按一下繼續。您將看到之前您對於工作網站所作更改的資訊。按一下完成，或者如果您想要進一步更改，按一下後退。按一下取消取消更改。

向我的工作區新增網路內容項目

  1. 從我的工作區按一下工作網站設定。

  2. 在工作網站設定頁面中，您將看到您的全部工作網站。如需向我的工作區新增網路內容項目，選中我的工作區旁的核取方塊。如需向您所管理的任何網站新增網路內容項目，選中該網站名稱旁的核取方塊。

  3. 按一下編輯。在下一個頁面中，按一下編輯工具。

  4. 選中 網路內容旁的核取方塊。按一下繼續。

  5. 在「自訂工具」螢幕中，輸入以下資訊：

    • 在「標題：」欄位中新增標題。
    • 在「URL:」欄位中已顯示的 "http://" 字樣後，輸入 URL 地址。
    • 如果您想要新增多個專案，那麼從更多網路內容工具？功能表選擇適當的項目數量。

  6. 按一下繼續。您將看到之前您對於工作網站所作更改的資訊。按一下完成，或者如果您想要進一步更改，按一下後退。按一下取消取消更改。

現在您應該看到功能表欄中顯示網路內容項目鏈結，以及您所提供的標題。

編輯網路內容項目

如需編輯現有網路內容項目，請遵循下列步驟進行操作：

  1. 在工作站點功能表欄中，按一下適當的網路內容項目標題。

  2. 在網路內容框架的頂部，按一下選項。

  3. 在「自訂網路內容」表單中，您可以更改下列設定：

    • 工具標題：顯示在網路內容框架頂部的標題。
    • 頁面標題：顯示在站點功能表欄中的標題。
    • 框架高度：網路內容框架的高度（以像素為單位）
    • URL：網路內容項目的 URL 地址

  4. 如需在新的瀏覽器視窗中開啟網路內容項目，按一下在新視窗中開啟？旁的核取方塊。此外，您還可以按一下如果在新視窗中開啟，顯示瀏覽器工具欄？旁的核取方塊，選擇顯示瀏覽器工具欄。

  5. 完成操作後，按一下更新選項儲存更改，或按一下取消取消更改。

刪除網路內容項目

  1. 如需刪除網路內容項目，按一下我的工作區，然後按一下工作網站設定。

  2. 如果您想要從課程或專案站點刪除網路內容，選中工作網站名稱旁的核取方塊。如果您想要從我的工作區刪除網路內容項目，選中我的工作區旁的核取方塊。

  3. 按一下編輯。然後按一下編輯工具。

  4. 從工具清單中，取消選中您想要刪除的項目，然後按一下繼續。檢查相關資訊，然後按一下完成刪除項目。

註釋：如果您的網路內容項目仍然顯示在網站工具欄中，那麼按一下首頁重新整理功能表欄。
