首頁／工作網站資訊：概述

功能

在每個課程網站或專案網站的首頁中，您將看到一般資訊區域（例如，工作網站資訊）和網站中所使用的交流工具的概要檢視（例如，最近通告、訊息和論壇通知，以及最近聊天訊息）。此外，您還可以檢視和更改網站的隱私狀態。

主要概念

內容選項：您可以選擇在工作網站資訊方塊中顯示網頁或文字。如需瞭解更多說明，請參考新增內容至工作網站資訊方塊。

隱私狀態：您可以選擇隱藏自己的身份，從而避免其他參與者透過網站與您聯絡。如需瞭解更多說明，請參考更改隱私狀態。

訊息和論壇通知：網站中全部新訊息和未讀訊息的概要資訊。如需瞭解更多資訊，請參考訊息和論壇通知。

注意事項

  • 擁有網站存取權限的每個人都可以看到工作網站資訊內所顯示的任何資訊。
  • 預設設定下，課程網站內的講師處於可見狀態。
