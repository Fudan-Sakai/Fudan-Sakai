更改網站的工作網站資訊方塊的大小

註釋：如果要完成本頁中的操作，您必需擁有具有必要權限的角色。如需決定自己的角色，請遵循參與者角色中的說明進行操作。如需瞭解關於角色和權限的基礎知識，請參考權限和角色：概述。

如需更改網站的工作網站資訊方塊的大小：

  1. 選擇一個課程網站或專案網站。

  2. 如果網站首頁尚未開啟，從功能表欄按一下首頁。

  3. 在工作網站資訊方塊中，按一下選項。

    註釋：由於工作網站資訊方塊的標題是可以自訂的，因此您所在網站中資訊方塊的標題可能不是「工作網站資訊」。但是，在網站首頁中，資訊方塊將始終位於功能表欄旁邊。

  4. 如需更改方塊的高度，從「框架高度」下拉式功能表選擇高度。無法更改方塊的寬度。

    註釋：大多數網頁在高度較大的方塊中顯示效果更佳（600 或更高像素）。

  5. 如需儲存更改，按一下更新選項。
