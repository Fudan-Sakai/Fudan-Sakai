新增內容至工作網站資訊方塊

註釋：如果要完成本頁中的操作，您必需擁有具有必要權限的角色。如需決定自己的角色，請遵循參與者角色中的說明進行操作。如需瞭解關於角色和權限的基礎知識，請參考權限和角色：概述。

工作網站資訊方塊位於網站首頁中功能表欄的旁邊。您可以選擇在方塊中顯示網頁或文字。根據您想要提供的內容的種類，遵循下列適當的步驟進行操作。

新增網頁

如需在工作網站資訊方塊中顯示網頁：

  1. 選擇一個課程網站或專案網站。

  2. 如果網站首頁尚未開啟，從功能表欄按一下首頁。

  3. 在工作網站資訊方塊中，按一下選項。

    註釋：由於工作網站資訊方塊的標題是可以自訂的，因此您所在網站中資訊方塊的標題可能不是「工作網站資訊」。但是，在網站首頁中，資訊方塊將始終位於功能表欄旁邊。

  4. 在「標題」欄位中輸入網站資訊方塊的標題，或保留預設或當前標題。

  5. 使用「框架高度」下拉式清單選擇工作網站資訊方塊的高度。

    註釋：大多數網頁在高度較大的框中顯示效果更佳（600 或更高像素）。

  6. 在「網站資訊 URL」欄位下輸入 URL（可省略 http:// 字首）。

  7. 按一下更新選項。

    註釋：您可能需要重新整理瀏覽器才能看到新的工作網站資訊。

新增文字描述

如需新增文字描述：

  1. 選擇課程或專案網站：

  2. 如果網站首頁尚未開啟，從功能表欄按一下首頁。

  3. 在工作網站資訊方塊中，按一下選項。

    註釋：由於工作網站資訊方塊的標題是可以自訂的，因此您所在網站中資訊框的標題可能不是「工作網站資訊」。但是，在網站首頁中，資訊框將始終位於功能表欄旁邊。

  4. 在「標題」欄位中輸入網站資訊方塊的標題，或保留預設或當前標題。

  5. 使用「框架高度」下拉式清單選擇工作網站資訊方塊的高度。

  6. 在「描述」欄位下輸入您想要顯示的文字。

  7. 按一下更新選項。

    註釋：您可能需要重新整理瀏覽器才能看到新的工作網站資訊。

新增自己的 HTML 檔案

如需顯示您已經建立並上載至資源工具中的 HTML 檔案：

  1. 根據您上載 HTML 檔案的位置，選擇課程或專案網站，或我的工作區。

  2. 按一下資源檢視資源清單。

  3. 按右鍵 HTML 檔案名，然後選擇瀏覽器的複製捷徑、複製連結網址或將連結複製至剪貼板以複製 HTML 檔案的 URL 地址。（或者，您也可以按一下檔案名，在新視窗中開啟檔案，然後從瀏覽器地址方塊複製 URL 地址。）

  4. 如果尚未開啟您想要顯示 HTML 檔案的目標課程或專案網站，請選擇該網站。

  5. 如果網站首頁尚未開啟，從功能表欄按一下首頁。

  6. 在工作網站資訊方塊中，按一下選項。

    註釋：由於工作網站資訊方塊的標題是可以自訂的，因此您所在網站中資訊框的標題可能不是「工作網站資訊」。但是，在網站首頁中，資訊框將始終位於功能表欄旁邊。

  7. 將 URL 貼上至「網站資訊 URL」欄位中。

  8. 按一下更新選項。

    註釋：您可能需要重新整理瀏覽器才能看到新的工作網站資訊。
