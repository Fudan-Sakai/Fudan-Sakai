檢視新聞摘要

如果您的網站中有一個新聞摘要連結，您將在網站功能表欄找到該連結。 如需檢視新聞，只需從功能表欄按一下新聞摘要標題。 根據網站擁有人為網站新聞摘要選擇的設定，標題會有所不同。 您可以在功能表欄中看到新聞，或其他更多描述性的標題，例如紐約時報、CNN 或 Yahoo! 新聞。

課程或專案網站中的新聞工具是一個 RSS「新聞聚合器」（或是新聞閱讀器），可以從 RSS 新聞摘要不斷讀取並顯示最新的線上內容。 RSS 是一種 XML 應用程式，即用於建立網路文件的標記語言。

除了新聞以外，一些網站還將 RSS 格式用於經常更新的資訊，如部落格（可公開存取的個人網頁日記）、事件清單或書本的修改歷史記錄。 通常您可以透過 URL 地址中的 .xml 或 .rss 來識別 RSS 摘要（例如，http://www.nytimes.com/services/xml/rss/nyt/Movies.xml 或 http://rss.cnn.com/rss/cnn_topstories.rss）。

如果您擁有必要的權限，那麼您可以透過新增任何 RSS 新聞摘要的網路地址來自訂課程或專案網站的新聞功能。 您可以透過使用 Google 來搜尋您感興趣的 RSS 摘要，或特殊的線上出版物或新聞來源。
