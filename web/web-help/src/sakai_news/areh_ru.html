Добавление, изменение и удаление страницы новостей

Внимание: Для выполнения этих действий Вам должна быть назначена роль с соответствующими полномочиями. Чтобы определить, какая у Вас роль, перейдите на страницу Роли участников. Основные сведения о ролях и правах доступа изложены также в Права доступа и роли: обзор.

Вы можете настроить сайт для отображения новостей, введя URL RSS-канала новостей (особым образом отформатированного списка новостных статей). В любое время можно добавить несколько инструментов "Новости".

Добавить страницу "Новости"

Чтобы добавить страницу "Новости":

  1. На линейке инструментов нажмите Настройки, затем нажмите Изменить инструменты.

  2. Отметьте флажком Новости, затем нажмите Продолжить.

  3. В поле "Название" field введите название страницы "Новости", которая впоследствии появится на линейке инструментов.

  4. В поле "URL" введите URL для канала новостей.

    Если сайт уже содержит страницы новостей или веб-ресурсы, информация о них отобразится здесь. Если все поля внизу списка заполнены, в выпадающем меню Добавить еще одну страницу новостей? выберите, сколько страниц новостей вы хотите добавить.

  5. Нажмите Продолжить. Вы увидите список изменений, внесеннных Вами в инструменты сайта.

  6. Нажмите Завершить.

Теперь Вы должны видеть ссылку на страницу новостей под названием, которое Вы ей присвоили, на линейке инструментов.

Изменить страницу "Новости"

Чтобы изменить страницу "Новости", на линейке инструментов нажмите на название страницы, которую необходимо изменить. Нажмите Настройки, затем внесите изменения в поля "Название" и "URL". Для завершения нажмите Обновить настройки.

Удалить страницу "Новости"

Чтобы удалить страницу "Новости", на линейке инструментов нажмите Настройки, затем Изменить инструменты. В списке инструментов снимите флажок с тех страниц новостей, которые Вы хотите удалить, затем нажмите Продолжить. Нажмите Завершить для удаления страницы.
